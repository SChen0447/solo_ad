## 1. 产品概述

面向电子音乐制作人的在线多轨混音协作平台，支持最多8轨音频同时编辑、实时混音预览与效果链调整，通过WebSocket实现多用户参数实时同步，并提供WAV格式混音导出功能。解决现有方案延迟高、协作功能缺失的问题。

- 目标用户：电子音乐制作人、音频工程师、远程协作音乐团队
- 核心价值：零延迟本地音频预览 + 实时协作参数同步 + 专业级效果链编辑

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 音乐制作人 | 直接访问 | 加载音频轨道、调整混音参数、导出混音 |
| 协作者 | 同房间连接 | 实时同步调整混音参数、预览混音结果 |

### 2.2 功能模块

1. **混音工作台页面**：轨道管理、波形可视化、音量/均衡器/效果链控件、全局播放控制、导出功能、协作状态指示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 混音工作台 | 全局控制栏 | 播放/停止按钮、导出WAV按钮（含进度条）、连接状态指示器 |
| 混音工作台 | 轨道列表区 | 可垂直滚动的轨道列表（最多8轨），每轨含波形缩略图和控件 |
| 混音工作台 | 轨道条 | 左侧波形缩略图（Canvas绘制，含播放进度光标）、右侧音量滑块、三段均衡器旋钮、效果链开关与参数 |
| 混音工作台 | 效果链面板 | 延迟效果器（延迟时间+反馈量）和混响效果器（衰减时间）的启用/禁用及参数微调 |
| 混音工作台 | 添加轨道按钮 | 从本地选择MP3/WAV文件（最大30MB）添加新轨道 |

## 3. 核心流程

### 3.1 轨道加载与播放流程

用户点击"添加轨道" → 选择本地音频文件 → 文件解码为AudioBuffer → 创建音频节点链（音源→均衡器→效果器→增益→主输出）→ 绘制波形缩略图 → 轨道出现在列表中 → 点击全局播放 → 所有轨道同步播放 → 波形显示进度光标 → 点击停止 → 光标归零

### 3.2 参数调整与协作同步流程

用户调节参数 → Zustand store更新 → audioEngine响应变化重新配置音频节点 → stateSync监听变化通过WebSocket广播 → 远程用户接收更新 → 回写Zustand store → 触发audioEngine重新配置

### 3.3 混音导出流程

用户点击"导出WAV" → 创建OfflineAudioContext → 按当前参数配置所有轨道节点链 → 执行离线渲染 → 进度条显示渲染进度 → 渲染完成 → 编码为WAV格式 → 生成Blob URL → 触发下载

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：#1a1a2e（深蓝紫）
- 副面板色：#16213e（暗蓝）
- 控件底色：#0f3460（中蓝）
- 高亮色：#e94560（品红红）
- 文字色：#e0e0e0（浅灰白）
- 按钮风格：圆形播放按钮（36px直径）、矩形导出按钮、旋钮为圆形带角度指示线
- 字体：JetBrains Mono（数据显示）+ Space Grotesk（UI文本）
- 布局：顶部固定全局控制栏 + 中部轨道列表（固定500px高度、可滚动）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 混音工作台 | 全局控制栏 | 圆形播放/停止按钮（播放时变绿）、矩形导出按钮（含内嵌进度条）、连接状态圆点（绿/红/灰） |
| 混音工作台 | 轨道条 | 80px高、左40%波形Canvas（含白色2px进度光标）、右侧音量滑块（渐变填充矩形）、三个均衡器旋钮（圆形+角度线）、效果开关 |
| 混音工作台 | 旋钮控件 | 圆形旋钮带角度指示线，hover亮度+10%、0.2s透明度过渡 |
| 混音工作台 | 滑块控件 | 带渐变填充的细长矩形，hover亮度+10%、0.2s过渡 |

### 4.3 响应式设计

- 桌面优先设计（≥768px）：波形缩略图左侧40%，控件右侧60%水平排列
- 移动端适配（<768px）：控件区改为垂直排列，波形缩略图下移，轨道高度自适应

### 4.4 动效设计

- 控件hover：亮度提升10% + 0.2秒透明度过渡
- 播放进度光标：2px白色竖线，平滑随currentTime移动
- 播放按钮状态切换：颜色从高亮红变为绿色
- 导出进度条：从0%到100%平滑填充
