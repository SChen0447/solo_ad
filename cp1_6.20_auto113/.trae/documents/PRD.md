# 动态文字海报编辑器 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
动态文字海报编辑器是一款面向设计师和社交媒体运营人员的在线工具，专注于快速制作带有动画特效的文字海报，支持直观调整每个字的位置与样式。

### 1.2 目标用户
- 设计师：需要快速生成文字创意海报
- 社交媒体运营人员：制作社交媒体内容配图
- 内容创作者：制作个人作品展示

### 1.3 核心价值
- 零设计基础也能制作精美动态文字海报
- 每个文字独立控制，精确定位和样式调整
- 丰富的动画预设，一键生成专业动效
- 支持高清导出，直接用于发布

---

## 2. 功能需求

### 2.1 海报画布模块
| 需求项 | 详细说明 |
|--------|----------|
| 画布尺寸 | 800px × 600px |
| 背景色 | 默认 #f5f0eb |
| 画布边框 | 2px 白色实线边框 #ffffff |
| 阴影效果 | 0 4px 20px rgba(0,0,0,0.1) |
| 渐变背景 | 10种预设渐变色模板，支持水平和对角线两种方向 |

### 2.2 文字添加模块
| 需求项 | 详细说明 |
|--------|----------|
| 文本输入 | 最多30个字符 |
| 添加方式 | 输入文字后点击添加按钮 |
| 初始位置 | 画布中央 |
| 独立元素 | 每个文字作为独立可拖拽元素 |
| 初始样式 | 字号36px，颜色#2d3436，字体'Georgia', serif |
| 拖拽效果 | 0.3秒弹性缓动，cubic-bezier(0.68, -0.55, 0.27, 1.55) |

### 2.3 样式编辑器模块
| 需求项 | 详细说明 |
|--------|----------|
| 面板尺寸 | 宽度280px |
| 面板样式 | 背景#ffffff，圆角16px，内边距24px |
| 字号调节 | 12-120px滑动条，步长1px，滑块颜色#6c5ce7 |
| 旋转角度 | -180到180度，圆形旋钮，步长1度，颜色#00b894 |
| 字体颜色 | 原生color picker，默认#2d3436 |
| 阴影效果 | 开关控制，开启后：水平偏移2px、垂直偏移2px、模糊4px、颜色rgba(0,0,0,0.3) |

### 2.4 动画预设模块
| 需求项 | 详细说明 |
|--------|----------|
| 动画选项 | 淡入、从上滑入、从左滑入、缩放出现、旋转出现，共5种 |
| 动画时序 | 每个字依次应用，延迟0.1秒，总持续0.6秒 |
| 缓动函数 | cubic-bezier(0.25, 0.46, 0.45, 0.94) |
| 播放控制条 | 底部固定，高48px，背景rgba(0,0,0,0.8)，圆角24px |
| 播放/暂停按钮 | 圆形，直径32px，颜色#dfe6e9，激活时#74b9ff |
| 进度条 | 宽200px，高6px，圆角3px，轨道色#636e72，填充色#74b9ff |
| 时间标签 | 白色12px字体 |

### 2.5 导出模块
| 需求项 | 详细说明 |
|--------|----------|
| 导出按钮 | 右上角圆形，直径36px，背景#2d3436，悬停#0984e3 |
| 导出格式 | PNG图片，2x分辨率，白色背景 |
| 确认对话框 | 半透明遮罩，居中卡片宽320px，圆角12px，背景#ffffff |
| 操作按钮 | '取消'和'导出'，导出按钮背景#0984e3，颜色#ffffff |

## 3. 界面设计规范

### 3.1 整体布局
- 左右两栏布局
- 左侧画布区域占比70%
- 右侧属性编辑面板固定280px
- 顶栏高56px，背景#2d3436

### 3.2 顶栏设计
- 左端：应用名称，白色18px字体，间距1px
- 中间：添加文字输入框（宽240px，高36px，背景#636e72，圆角6px，占位符#b2bec3）
- 添加按钮：高36px，背景#6c5ce7，圆角6px，悬停背景#5f27cd

### 3.3 右侧面板
- 背景色#ffffff，1px浅灰边框#dfe6e9
- 顶部4个标签页：文本、样式、动画、导出
- 无边框设计，选中状态底部显示2px #6c5ce7色条
- 标签切换0.2秒淡入淡出

### 3.4 通用设计规范
- 圆角：6px或12px
- 过渡动画：0.2-0.3秒
- 字体：'Georgia', serif

## 4. 性能要求

| 指标 | 要求 |
|------|------|
| 拖拽操作帧率 | ≥ 55fps |
| 旋转操作帧率 | ≥ 55fps |
| 动画播放帧率 | ≥ 50fps |

## 5. 技术栈

- **框架**: React 18+
- **语言**: TypeScript
- **构建工具**: Vite
- **动画库**: framer-motion
- **导出方案**: html2canvas 或原生 Canvas API
