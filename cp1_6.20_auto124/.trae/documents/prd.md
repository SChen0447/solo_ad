## 1. 产品概述

DataViz仪表盘是一款面向个人和小型团队的数据可视化工具，旨在解决现有图表工具无法便捷自定义仪表盘布局、以及无法从多个数据源动态拉取并整合展示的痛点。用户可以通过拖拽方式自由排列图表卡片，绑定多种数据源（静态JSON、REST API、CSV文件），实现实时数据联动与自动刷新。

- 目标用户：数据分析师、产品经理、小型创业团队
- 核心价值：零代码创建自定义仪表盘，多数据源实时联动展示

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 创建/管理仪表盘、配置数据源、添加图表卡片 |

### 2.2 功能模块

1. **仪表盘编辑页**：左侧边栏管理仪表盘列表，中央网格区域展示图表卡片，右侧抽屉配置数据源
2. **数据源配置**：支持静态JSON、REST API轮询、CSV上传三种数据源

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘编辑页 | 左侧边栏 | 仪表盘列表展示、创建/重命名/删除仪表盘、选中切换，紫色高亮条标识当前项 |
| 仪表盘编辑页 | 图表网格区域 | CSS Grid自适应布局，支持添加/删除图表卡片，拖拽重排，卡片尺寸三档预设 |
| 仪表盘编辑页 | 图表卡片 | Canvas渲染四种图表（折线/柱状/饼图/面积），显示数据源绑定和字段映射，全屏/删除按钮 |
| 仪表盘编辑页 | 右侧数据源抽屉 | 滑入式面板，配置JSON/API/CSV数据源，显示类型图标和相对更新时间 |
| 仪表盘编辑页 | 全局刷新按钮 | 紫色圆形按钮，悬停旋转360度，强制刷新所有数据源 |
| 仪表盘编辑页 | 数据联动 | 数据源更新时自动重绘图表，蓝色波纹提示条，淡入动画 |

## 3. 核心流程

用户打开应用后，在左侧边栏创建或选择一个仪表盘。在中央网格区域通过工具栏选择图表类型添加卡片，拖拽调整卡片位置和尺寸。点击右侧抽屉按钮打开数据源面板，配置JSON/API/CSV数据源。将数据源绑定到图表卡片并设置字段映射。当API轮询或CSV重新上传时，绑定该数据源的图表自动刷新并显示蓝色波纹提示。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，主背景#1a1a2e，次级背景#2d2d44，侧边栏#1e1e2e，抽屉#22223b
- **强调色**：紫色#7c3aed（主交互元素），蓝色#3b82f6（图表/链接），绿色#10b981，橙色#f59e0b，红色#ff4757
- **文字**：主色#e0e0e0，辅助色#9ca3af，标题16px
- **按钮**：圆角8px，悬停背景变亮10%；圆形按钮（删除#ff4757、全屏#00d2d3）悬停扩大
- **卡片**：背景#2d2d44，圆角12px，1px #3d3d56边框，0 4px 16px rgba(0,0,0,0.2)阴影
- **动画**：过渡0.2-0.3s，拖拽opacity:0.6，弹入cubic-bezier，加载占位旋转虚线圆环2s

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘编辑页 | 左侧边栏 | 宽260px，背景#1e1e2e，每项高40px圆角8px，悬停#2a2a3c 0.2s过渡，选中项左侧3px #7c3aed高亮条 |
| 仪表盘编辑页 | 网格区域 | CSS Grid自适应列，间隙16px，桌面三列/平板两列/手机一列 |
| 仪表盘编辑页 | 图表卡片 | 三种尺寸（250x200/400x300/600x400），标题栏36px，右上角24px圆形按钮 |
| 仪表盘编辑页 | 右侧抽屉 | 宽320px，背景#22223b，0.3s缓动滑入，数据源类型图标+相对时间 |
| 仪表盘编辑页 | 全局刷新按钮 | 直径36px圆形，背景#7c3aed，悬停旋转360度0.5s |
| 仪表盘编辑页 | 数据联动提示 | 蓝色波纹条100%宽3px #3b82f6，从左到右1s，图表0.5s淡入重绘 |

### 4.3 响应式适配

- **桌面（≥1024px）**：三列网格 + 左侧边栏 + 右侧抽屉
- **平板（768-1023px）**：两列网格 + 左侧边栏 + 右侧抽屉
- **手机（<768px）**：一列网格 + 顶栏（高56px，仪表盘名称水平滚动标签组）+ 右侧抽屉
