## 1. 产品概述

代码审查可视化批注工具，帮助团队在远程协作中高效分享和讨论代码片段。解决传统依赖截图和外部文档的代码审查方式，提供代码高亮、行级批注、短链接分享等功能，提升团队代码审查效率。

- **核心价值**：让代码片段的分享和讨论更直观、高效、可交互
- **目标用户**：开发团队、代码审查者、技术面试官
- **解决问题**：远程协作时代码上下文传递困难，缺乏代码高亮和交互反馈

## 2. 核心功能

### 2.1 功能模块

1. **代码编辑器**：支持 5 种语言语法高亮的代码输入
2. **预览与批注**：代码卡片展示、行选择、添加批注
3. **短链接分享**：生成唯一短链接，localStorage 持久化
4. **分享视图**：双栏布局，代码 + 批注列表联动
5. **主题切换**：亮色/暗色主题
6. **响应式布局**：桌面端双栏，移动端上下结构

## 3. 核心流程

用户创建代码片段并添加批注，生成短链接分享，其他用户通过短链接查看和讨论。

```mermaid
flowchart LR
    A["输入代码<br/>选择语言"] --> B["点击预览按钮"]
    B --> C["预览模式<br/>拖拽选中代码行"]
    C --> D["点击添加批注<br/>输入批注内容"]
    D --> E["保存生成短链接"]
    E --> F["分享短链接"]
    F --> G["输入短链接加载"]
    G --> H["双栏视图查看<br/>代码+批注列表"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：浅色主题 #fafafa 背景，#333 文字
- **强调色**：蓝色系（批注圆点、选中高亮）
- **按钮风格**：hover 上移 1px + 阴影过渡，200ms 动画
- **字体**：现代无衬线字体，代码使用等宽字体
- **布局风格**：卡片式设计，顶部导航栏 48px
- **卡片**：圆角 8px，浅灰背景

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 编辑器页 | 顶部导航 | 48px 高，白色背景，底部 1px 分隔线 |
| 编辑器页 | 短链接输入 | 圆角 20px，左侧搜索图标 |
| 编辑器页 | 语言选择器 | 左上角下拉框 |
| 编辑器页 | 水晶按钮 | 金属渐变 #e0e0e0 到 #c0c0c0，右上角 |
| 编辑器页 | 代码输入区 | textarea + 高亮覆盖层 |
| 预览页 | 代码卡片 | 圆角 8px，背景 #f5f5f5，可滚动 |
| 预览页 | 选中行 | 浅蓝色 #add8e6 半透明 |
| 预览页 | 批注圆点 | 直径 12px，蓝色，行右侧边缘 |
| 预览页 | Tooltip | 深灰 #333 背景，白色文字，圆角 4px，延迟 300ms |
| 分享视图 | 双栏布局 | 左 60% 右 40% |
| 分享视图 | 批注项 | 时间戳 yyyy-mm-dd HH:MM，预览 40 字 |
| 全局 | 暗色模式 | 背景 #1e1e1e，文字 #ddd，卡片 #252526 |
| 全局 | 月亮图标按钮 | 右上角主题切换 |

### 4.3 响应式

- 桌面端优先设计，<768px 自适应
- 移动端双栏变上下结构（左栏在上，右栏在下）
- 所有输入框宽度 100%
- 触摸交互优化

### 4.4 性能指标

- 500 行以内代码高亮渲染 ≤ 50ms
- 100 个批注数据加载恢复 ≤ 80ms
- 滚动画布帧率保持 60fps
