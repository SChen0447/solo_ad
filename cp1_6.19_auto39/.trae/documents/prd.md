## 1. 产品概述

文字排版沙盒是一个面向前端开发者和设计师的轻量级所见即所得工具，帮助用户通过侧面板实时调整文本的字体、字号、字间距、行高和对齐方式，并快速预览排版效果，同时支持将当前排版参数导出为CSS代码片段。

- 核心价值：消除手动调CSS的繁琐流程，提供即时视觉反馈，加速排版决策
- 目标用户：前端开发者、UI设计师、内容编辑人员

## 2. 核心功能

### 2.1 功能模块

1. **编辑面板**：字体选择器、字号滑块、字间距滑块、行高滑块、对齐按钮组、预设排版方案卡片、CSS导出按钮
2. **预览画布**：实时渲染示例文本，应用当前排版参数

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 字体选择器 | 下拉选择器，包含10种Web安全字体和Google Fonts，选择后画布即时渲染 |
| 主页面 | 字号滑块 | 范围12px-72px，步长1px，拖动实时更新 |
| 主页面 | 字间距滑块 | 范围-2px到8px，步长0.5px，拖动实时更新 |
| 主页面 | 行高滑块 | 范围1.0-3.0，步长0.1，拖动实时更新 |
| 主页面 | 对齐按钮组 | 左对齐、居中、右对齐、两端对齐四个按钮 |
| 主页面 | 预设排版方案 | 至少4种预设方案（新闻正文、博客标题、展览说明、诗歌排版），一键应用 |
| 主页面 | CSS导出 | 导出当前排版为CSS代码片段，支持复制到剪贴板和下载为.css文件 |
| 主页面 | 预览画布 | 居中显示示例文本，应用当前排版参数实时渲染 |

## 3. 核心流程

用户通过侧面板调整排版参数 → zustand store 更新全局状态 → PreviewPanel 订阅 store 变化 → 重新计算 CSS 样式 → 画布实时渲染更新。预设方案点击后一次性更新所有排版参数。导出时读取当前 store 状态生成 CSS 代码。

```mermaid
    "用户操作" --> "EditorPanel"
    "EditorPanel" --> "zustand store"
    "zustand store" --> "PreviewPanel"
    "zustand store" --> "CSS导出"
    "预设方案" --> "zustand store"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：#4a90d9（蓝色强调色）
- 侧面板背景：#f4f4f9
- 画布背景：#ffffff
- 分隔线颜色：#e0e0e0
- 按钮风格：圆角矩形，选中高亮
- 字体：侧面板使用系统字体，画布预览根据用户选择渲染
- 布局风格：左右两栏布局，侧面板固定320px宽度

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 侧面板 | 固定宽度320px，背景#f4f4f9，控制项间浅色分隔线 |
| 主页面 | 字体选择器 | 圆角8px下拉框，选中项高亮#4a90d9 |
| 主页面 | 滑块控件 | 轨道色#d0d0d0，滑块按钮色#4a90d9 |
| 主页面 | 对齐按钮组 | 28px×28px圆角矩形，选中蓝底白图标，未选中灰底 |
| 主页面 | 预设卡片 | 140px×60px卡片，悬停#eef2f7，选中边框#4a90d9，0.2s过渡 |
| 主页面 | 导出按钮 | 蓝底白字#4a90d9，圆角6px，点击缩放0.95动画0.15s |
| 主页面 | 画布区域 | 纯白背景，20px内边距，文本最大宽度800px，居中显示 |

### 4.3 响应式设计

- 桌面端：左右两栏布局（侧面板320px + 画布自适应）
- 移动端（窗口宽度 < 960px）：切换为上下布局，侧面板在上，画布在下

### 4.4 性能要求

- 滑块拖动和字体选择变化在50毫秒内更新画布
- 预设方案切换在100毫秒内完成画布更新
