## 1. 产品概述
创意灵感随机生成与灵感收藏簿应用——帮助创意工作者一键获取随机组合生成的灵感，收藏喜欢的灵感并添加备注和标签，以卡片网格形式管理和筛选收藏内容。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 生成灵感、收藏灵感、管理收藏夹 |

### 2.2 功能模块
1. **首页**: 灵感生成按钮、灵感卡片展示、收藏/重新生成操作
2. **收藏夹**: 卡片网格展示、标签筛选、时间排序、备注编辑

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 灵感生成按钮 | 圆角渐变紫按钮，悬停上移+阴影，点击缩放弹回+脉冲光效 |
| 首页 | 灵感卡片 | 展示生成的灵感文本，深色背景深紫边框，最多3行 |
| 首页 | 收藏按钮 | 星形图标，灰色未收藏/金黄已收藏，旋转动画 |
| 首页 | 重新生成按钮 | 点击重新请求灵感 |
| 收藏夹 | 筛选区域 | 标签下拉筛选（全部/已有标签）、排序下拉（时间升序/降序） |
| 收藏夹 | 卡片网格 | CSS Grid布局，每列最小280px，卡片高200px，含灵感文本/备注/标签 |
| 收藏夹 | 备注编辑 | 点击展开编辑框，深色背景输入框，聚焦时紫色边框 |
| 收藏夹 | 标签展示 | Pill形状标签，hover放大1.05倍 |

## 3. 核心流程

用户打开应用 → 首页点击"生成灵感" → 后端随机组合生成灵感 → 灵感卡片展示 → 点击收藏存入后端 → 可编辑备注和标签 → 切换到收藏夹查看 → 筛选/排序浏览收藏

```mermaid
flowchart TD

    "G" --> "H[点击收藏按钮]"
    "H" --> "I[POST /api/inspiration/collection]"

    "G" --> "K[点击重新生成]"
    "K" --> "C"
    "J" --> "L[切换到收藏夹]"
    "L" --> "M[GET /api/inspiration/collection]"
    "M" --> "N[卡片网格展示]"
    "N" --> "O[筛选/排序操作]"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：紫色系（#6b46c1, #a855f7, #8b5cf6）
- 辅助色：暖黄（#fbbf24）用于收藏状态
- 背景：深色主题（#0f0f23）
- 按钮风格：圆角（12-20px）、渐变色、平滑过渡动画
- 字体：系统无衬线字体
- 布局：卡片式布局，顶部导航

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 生成按钮 | 渐变紫背景#6b46c1→#a855f7，白字，圆角20px，200×56px，hover上移3px+阴影，click缩放0.95+脉冲光效0.5s |
| 首页 | 灵感卡片 | 背景#1e1e2e，圆角16px，内边距24px，宽500px，白字，2px紫色边框#7c3aed |
| 首页 | 收藏按钮 | 星形，灰#6b7280/金黄#fbbf24，旋转180° 0.4s ease |
| 收藏夹 | 卡片网格 | CSS Grid，列最小280px，gap 24px，卡片高200px，背景#2a2a3e，圆角12px，内边距16px |
| 收藏夹 | 备注输入框 | 背景#374151，边框#4b5563，聚焦边框#8b5cf6 |
| 收藏夹 | 标签 | Pill形状，背景#374151，灰白字，hover放大1.05倍 0.3s |
| 全局 | 导航栏 | 首页/收藏夹链接，切换0.3s淡入淡出 |

### 4.3 响应式适配
- 桌面优先设计
- 宽度<768px时：灵感卡片宽度改为90%，收藏夹网格变单列，导航链接堆叠

### 4.4 动画效果
- 页面切换：0.3s淡入淡出
- 收藏夹卡片加载：0.5s交错渐入
- 按钮：hover上移3px，click缩放0.95弹回，脉冲光效0.5s
- 收藏星标：旋转180° 0.4s ease
- 标签：hover放大1.05倍 0.3s
