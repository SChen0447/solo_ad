## 1. 产品概述

团队成员烹饪菜谱共享与协作评分平台，让团队内部分享各自的拿手菜谱，其他成员可以试做、评分和评论，系统根据口味偏好智能推荐菜谱。
- 解决团队内部菜谱分享缺乏集中平台、评分反馈不便的问题
- 面向喜欢烹饪的团队成员，提升团队互动和美食文化交流

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 本地模拟登录 | 浏览菜谱、发布菜谱、评分评论、查看个人中心 |

### 2.2 功能模块
1. **主页面**：瀑布流菜谱卡片展示区、导航栏、搜索入口
2. **菜谱详情页**：完整菜谱信息、评分条、评论区
3. **个人中心页**：发布统计、趋势图表、智能推荐

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 瀑布流卡片展示区 | 展示所有菜谱卡片，每张卡片含菜名、作者头像、评分星星（半透明金色渐变，悬停放大1.1倍并显示评分数字）、缩略图、底部点赞按钮（0.3秒弹性动画，心形图标从透明变实心+1.2倍缩放脉冲） |
| 主页面 | 导航栏 | 页面顶部导航，包含首页、个人中心切换 |
| 菜谱详情页 | 左侧菜谱信息 | 完整菜谱内容：食材清单（圆角标签排列）、步骤（1.5行高大号灰色数字编号）、段落间0.6秒淡入分隔线动画 |
| 菜谱详情页 | 右侧评分与评论 | 拖拽评分条（1-5分）、提交文字评论、评论列表（用户头像、时间、编辑删除按钮）、新评论从底部滑入带模糊入场效果 |
| 个人中心页 | 统计信息 | 发布数、总获赞数、平均评分 |
| 个人中心页 | 趋势图表 | 渐变柱状图展示近30天新增菜谱趋势 |
| 个人中心页 | 智能推荐栏 | 基于历史评分和收藏推荐最多6个菜谱，卡片带淡入和微上浮动画 |

## 3. 核心流程

用户进入主页浏览瀑布流菜谱卡片 → 点击卡片进入详情页查看完整菜谱 → 拖拽评分条评分并提交评论 → 返回主页继续浏览 → 进入个人中心查看统计和推荐

## 4. 用户界面设计

### 4.1 设计风格
- 主背景色：浅米色 #F5F0E8，营造温暖厨房氛围
- 强调色：深橙色 #D2691E，呼应美食主题
- 文字主色：深灰 #333333
- 按钮和链接悬停：0.2秒颜色渐变动画
- 字体：选用 Playfair Display 作为标题字体（优雅有质感），Noto Sans SC 作为正文字体
- 卡片布局：瀑布流，圆角8px，柔和阴影
- 图标风格：线条风格，与暖色调搭配

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 瀑布流卡片 | 多列不等高卡片布局，卡片圆角8px，柔和投影，悬停微上浮+阴影加深 |
| 主页面 | 评分星星 | 半透明金色渐变填充，悬停放大1.1倍并显示评分数字 |
| 主页面 | 点赞按钮 | 底部居中，心形图标，0.3秒弹性动画，透明↔实心切换+1.2倍缩放脉冲 |
| 菜谱详情页 | 食材标签 | 圆角标签排列，浅橙色背景 |
| 菜谱详情页 | 步骤编号 | 1.5行高大号灰色数字编号 |
| 菜谱详情页 | 分隔线 | 段落间0.6秒淡入动画 |
| 菜谱详情页 | 评分条 | 可拖拽，1-5分，橙色渐变填充 |
| 菜谱详情页 | 评论项 | 头像+昵称+时间+内容+编辑删除按钮，新评论底部滑入模糊入场 |
| 个人中心页 | 统计卡片 | 三个统计数字卡片，图标+数字+标签 |
| 个人中心页 | 柱状图 | 渐变柱状图，近30天趋势 |
| 个人中心页 | 推荐卡片 | 最多6个，淡入+微上浮动画 |

### 4.3 响应式设计
- 桌面优先设计，移动端自适应
- 移动端（宽度<768px）自动切换单列布局
- 卡片间距由32px缩减为16px
- 图片等比缩放
- 触摸优化：按钮点击区域增大，拖拽评分条适配触摸

### 4.4 性能要求
- 页面初次加载时间不超过2秒
- 使用图片懒加载
- 代码分割（路由级别）
- 评论提交后1秒内显示在列表中
