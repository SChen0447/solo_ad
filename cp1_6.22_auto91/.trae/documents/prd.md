## 1. 产品概述

在线协作食谱编辑与分享平台，支持用户创建、编辑和发布多步骤食谱，并允许他人对食谱进行评论和评分。目标用户为家庭烹饪爱好者、美食博主和食谱创作者，通过该平台实现食谱的数字化管理与社交互动。

- 解决传统食谱分散、难以结构化记录和分享的问题
- 为美食爱好者提供可视化、互动化的食谱创作与交流社区

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 免注册直接使用 | 浏览食谱、创建食谱、评论评分 |
| 访客 | 无需登录 | 浏览食谱 |

### 2.2 功能模块

1. **首页**：食谱网格列表展示、搜索筛选
2. **食谱详情页**：食谱完整信息、食材步骤、评论评分
3. **创建/编辑食谱页**：分步表单（基本信息→食材→步骤）、图片上传

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 食谱网格列表 | 展示所有食谱卡片（封面图、标题、烹饪时间、作者），鼠标悬停卡片上浮效果 |
| 首页 | 导航栏 | 平台名称、创建食谱入口按钮 |
| 食谱详情页 | 封面大图区 | 占满宽度300px高度封面图展示 |
| 食谱详情页 | 左侧食材步骤 | 食材清单列表、分步说明（含可选图片） |
| 食谱详情页 | 右侧信息栏 | 基本信息（准备时间、烹饪时间、食用人数）、星级评分组件 |
| 食谱详情页 | 评论区 | 评论列表（时间倒序）、评论输入框、评分提交 |
| 创建食谱页 | 第一步-基本信息 | 标题、封面图片上传、描述、准备时间、烹饪时间、食用人数 |
| 创建食谱页 | 第二步-食材清单 | 动态增删食材行（名称+用量） |
| 创建食谱页 | 第三步-分步说明 | 每步含文本和可选图片 |
| 创建食谱页 | 进度条 | 显示当前步骤、已完步骤高亮圆点 |

## 3. 核心流程

用户打开平台首页浏览食谱列表 → 点击食谱卡片进入详情页查看完整信息 → 可对食谱评分和评论 → 点击创建食谱进入分步表单 → 填写基本信息 → 添加食材 → 编写步骤 → 提交发布食谱 → 返回首页查看新食谱

## 4. 用户界面设计

### 4.1 设计风格

- 主色：橙黄 #ed8936，辅色：米白 #fefcbf，背景：浅奶油 #fffff0
- 按钮：圆角8px，提交按钮渐变色从#48bb78到#38a169，点击有ripple扩散效果（0.6秒消失）
- 标题字体：serif（Georgia），正文：sans-serif
- 卡片布局：宽300px，圆角16px，背景#fff，阴影0 4px 12px rgba(0,0,0,0.08)
- 评分星星：金色#f6ad55，空星灰色#e2e8f0，悬停缩放1.15倍（0.2秒）
- 列表项逐个淡入动画（0.3秒，从上偏移10px）
- 表单输入框聚焦时橙黄边框+外发光box-shadow 0 0 0 3px rgba(237,137,54,0.3)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 食谱网格 | 3列网格，卡片300px宽，封面图156px高，标题serif字体，烹饪时间图标标注，悬停上浮8px阴影加深 |
| 食谱详情页 | 封面区 | 全宽300px高，object-fit cover，标题叠加底部 |
| 食谱详情页 | 内容区 | 左右双栏，左侧食材步骤列表，右侧信息卡+评分组件 |
| 食谱详情页 | 评论区 | 评论框圆角8px背景#f7fafc，提交按钮渐变绿，评论按时间倒序，头像首字母圆形随机色 |
| 创建食谱页 | 分步表单 | 3步表单切换，底部进度条深青#319795，当前步骤高亮圆点，切换0.5秒内无白屏 |
| 创建食谱页 | 图片上传 | 支持本地选择，自动压缩至800px宽/300KB以下 |

### 4.3 响应式设计

- 桌面优先设计，移动端（<768px）自动适配
- 移动端：单列布局，卡片占满宽度，详情页图片高度降为200px，表单全宽
- 触摸优化：按钮和卡片点击区域足够大
