## 1. 产品概述

水印批量生成工具是一款面向独立摄影师和设计师的 Web 应用，旨在解决手动添加文字水印效率低、水印样式单一、图片色彩不一致的问题。用户可通过可视化控制面板快速设置个性化水印参数，实时预览效果，并批量处理多张图片并一键下载。

- 目标用户：独立摄影师、设计师、内容创作者
- 核心价值：提升图片版权保护效率，提供丰富的水印样式自定义能力

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传图片、配置水印、预览效果、批量导出 |

### 2.2 功能模块
1. **主工作区：图片上传区、缩略图网格、水印控制面板、实时预览区

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作区 | 图片上传区 | 虚线拖拽上传（支持 JPG/PNG，单图≤10MB）、拖拽时蓝色边框脉动光晕、多图批量上传 |
| 主工作区 | 缩略图网格 | 预览图展示、悬停放大10%+半透明遮罩显示文件名和尺寸、单张删除 |
| 主工作区 | 水印控制面板 | 文字输入（中英文）、字体选择（4种预置）、字号滑块（10-100px）、颜色选择器+透明度滑块、九宫格位置选择、旋转角度滑块（-90°~90°） |
| 主工作区 | 实时预览区 | 水印效果实时渲染（200ms缓动过渡）、主预览大图展示 |
| 主工作区 | 批量生成模块 | 蓝紫渐变进度条、单张下载、全部ZIP打包下载 |

## 3. 核心流程

用户上传图片 → 配置水印参数 → 实时预览效果 → 批量生成水印 → 单张/打包下载

```mermaid

    A["用户进入页面" --> B["拖拽/点击上传图片"]
    B --> C["图片加载到缩略图网格"]
    C --> D["配置水印参数"]
    D --> E["实时预览水印效果"]
    E --> F["点击批量生成"]
    F --> G["进度条展示生成进度"]
    G --> H["单张下载 / ZIP打包下载"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：浅灰背景 #F5F5F7，深蓝强调色 #1E3A5F
- 卡片圆角：12px
- 按钮样式：深蓝底色白色文字，禁用状态40%透明度
- 字体：预置4种字体供用户选择
- 布局风格：极简风格，大量留白，卡片式布局
- 微动效：所有交互元素平滑过渡（按钮悬停、卡片选中、滑块拖拽）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|-----------|
| 主工作区 | 图片上传区 | 虚线边框拖拽区、脉动光晕动画、拖拽时蓝色边框 |
| 主工作区 | 缩略图网格 | 悬停放大10%、半透明暗色遮罩、文件名+尺寸显示 |
| 主工作区 | 水印控制面板 | 滑动条、颜色选择器、九宫格按钮组、输入框 |
| 主工作区 | 实时预览区 | Canvas渲染预览图 |
| 主工作区 | 批量生成进度条 | 蓝色到紫色渐变色彩条 |

### 4.3 响应式设计
- 桌面端：左右两栏布局，左侧40%（上传+缩略图），右侧60%（预览+控制面板）
- 移动端（<768px）：上下垂直布局
- 触摸优化：触控区域充足的按钮尺寸

## 5. 性能指标
- 5张图片上传后，参数调整到预览更新响应时间 ≤ 300ms
- 批量生成5张图片总耗时 ≤ 5秒
