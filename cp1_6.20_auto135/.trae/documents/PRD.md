# 团队脑暴与创意投票排序应用 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
面向团队协作的在线脑暴白板工具，支持创意发散、实时协作编辑与投票排序，帮助团队高效筛选最优方案。

### 1.2 目标用户
- 产品团队：头脑风暴产品功能方向
- 设计团队：创意方案讨论
- 研发团队：技术方案选型
- 任何需要集体创意决策的团队

### 1.3 核心价值
- 无门槛加入：6位会话码即可快速加入
- 实时协作：多人同时编辑白板，毫秒级同步
- 视觉化创意：卡片式想法展示，直观清晰
- 民主决策：投票排序机制，快速达成共识

---

## 2. 功能需求

### 2.1 会话管理模块

#### 2.1.1 创建会话
- **输入**：标题（最长20字）、描述（最长60字）
- **输出**：唯一6位字母数字会话码
- **规则**：
  - 标题必填，描述可选
  - 创建者自动成为会话管理员
  - 会话码全局唯一，不可重复

#### 2.1.2 加入会话
- **输入**：6位会话码
- **校验**：会话码存在且会话未结束
- **输出**：进入白板页面

#### 2.1.3 用户身份
- 需要输入昵称加入
- 每个用户分配唯一ID
- 区分创建者（管理员）和普通成员

---

### 2.2 创意白板模块

#### 2.2.1 白板区域
- 宽100%，高600px
- 背景色：#f5f0eb（米白色）
- 圆角：16px
- 支持滚动（当卡片超出可视区域）

#### 2.2.2 创建卡片
- 交互：双击白板空白处
- 默认尺寸：260px × 180px
- 默认位置：双击坐标居中
- 初始内容：空白文本，无标签颜色

#### 2.2.3 卡片组成
| 元素 | 规格 |
|------|------|
| 顶部边框 | 4px高，与标签颜色联动 |
| 文本输入区 | 多行文本，最多120字 |
| 创建者信息 | 昵称，灰色小字#999 |
| 标签选择器 | 8种预设颜色 |

#### 2.2.4 卡片拖拽
- 拖动中：缩放1.05倍，阴影加深(0 12px 28px rgba(0,0,0,0.15))
- 释放后：恢复原状，动画0.2s
- 坐标实时同步到其他用户

#### 2.2.5 标签颜色
预设8种颜色：
- #ff6b6b（珊瑚红）
- #51cf66（薄荷绿）
- #339af0（天空蓝）
- #ffd43b（金色）
- #cc5de8（紫罗兰）
- #ff922b（橙色）
- #20c997（青绿）
- #868e96（岩石灰）

交互：悬停放大1.1倍并显示颜色名称

---

### 2.3 投票排序模块

#### 2.3.1 发起投票
- 仅创建者可发起
- 倒计时：5分钟（300秒）
- 倒计时显示：红色#e03131，每秒跳动一次
- 投票期间不可编辑卡片内容

#### 2.3.2 投票机制
- 每个用户最多投3票
- 投票方式：拖拽卡片到顶部热门区
- 热门区规格：
  - 背景：#ffd43b 透明度0.15
  - 边框：虚线
  - 圆角：12px
  - 拖入高亮：透明度变为0.3，弹性动画

#### 2.3.3 实时排序
- 热门区内卡片按得票数降序排列
- 得票变化实时更新排序
- 每票对应一个用户的拖拽动作

---

### 2.4 结果展示模块

#### 2.4.1 倒计时结束自动展示结果
#### 2.4.2 卡片排名样式
| 排名 | 背景样式 | 文字颜色 |
|------|---------|---------|
| 第1名 | 金色渐变(#ffd43b → #ffb005) | 白色 |
| 第2名 | 银灰色(#ced4da) | 默认 |
| 第3名 | 铜色(#cd9b5b) | 默认 |
| 其他 | 白色 | 默认 |

#### 2.4.3 入场动画
- 使用 framer-motion AnimatePresence
- 从底部逐个飞入
- 动画间隔：0.1s

---

### 2.5 实时协作模块

#### 2.5.1 同步内容
- 卡片创建/删除
- 卡片位置变化
- 卡片内容编辑
- 卡片标签颜色变更
- 投票状态变化
- 用户在线状态

#### 2.5.2 性能要求
- WebSocket消息延迟 < 200ms
- 50张卡片同时拖拽帧率 ≥ 45fps

## 3. 界面设计规范

### 3.1 色彩系统

| 用途 | 色值 |
|------|------|
| 主色（橙色） | #ff922b |
| 辅色（金色） | #ffd43b |
| 背景色（米白） | #f5f0eb |
| 卡片背景 | #ffffff |
| 倒计时红色 | #e03131 |
| 次要文字 | #999999 |
| 主按钮悬停 | #e8590c |

### 3.2 字体
- 使用系统字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`

### 3.3 头部
- Logo：圆形40px，渐变#ff922b→#ffd43b，中心白色"B"
- 会话码：加粗，字号1.2rem

### 3.4 按钮规范
- 圆角：8px
- 主按钮背景：#ff922b
- 主按钮悬停：#e8590c
- 点击缩放：0.95倍
- 过渡时长：0.2s

### 3.5 侧边栏（在线成员）
- 宽度：220px
- 背景：白色
- 右侧阴影
- 成员头像：圆形32px
- 成员状态："正在编辑" / "浏览中"

### 3.6 响应式设计
- 断点：768px
- < 768px：侧边栏折叠为悬浮按钮
  - 位置：右下角固定
  - 尺寸：圆形50px
  - 样式：背景#ff922b，白色加号图标

### 3.7 动画规范
- 过渡时长：0.2s ~ 0.3s
- GPU加速属性：transform, opacity
- 缓动函数：ease-out / spring（弹性）

---

## 4. 非功能需求

### 4.1 性能要求
1. 50张卡片同时拖拽帧率 ≥ 45fps
2. WebSocket消息端到端延迟 < 200ms
3. 首屏加载时间 < 2s

### 4.2 兼容性
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 4.3 数据一致性
- 操作顺序保证（基于时间戳）
- 冲突解决：最后写入优先（LWW）

---

## 5. 用户流程

### 5.1 创建者流程
1. 进入首页 → 填写标题/描述 → 创建会话
2. 获取6位会话码 → 分享给团队成员
3. 白板阶段：自己创建/编辑卡片，同步查看成员操作
4. 点击"发起投票"按钮 → 开始5分钟倒计时
5. 倒计时结束 → 自动展示投票结果

### 5.2 参与者流程
1. 进入首页 → 输入6位会话码 + 昵称 → 加入会话
2. 白板阶段：双击创建卡片，拖拽调整位置
3. 投票阶段：拖拽最多3张卡片到热门区
4. 倒计时结束 → 查看投票结果
