## 1. 产品概述

虚拟展览策展方案应用，为艺术爱好者和策展人提供线上展览快速规划工具，解决展品布局、信息标注和分享链接生成的痛点。

- 核心目标：让策展人通过拖拽式编辑器快速完成展品布局规划，管理展品详细信息，并生成可分享的沉浸式预览链接。
- 目标用户：艺术爱好者、独立策展人、画廊从业者、教育工作者。

## 2. 核心特性

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 策展用户 | 免注册（本地存储） | 创建编辑展览、管理展品、预览展览、生成分享链接 |

### 2.2 功能模块

1. **编辑器页面**：6x8网格画布、展品库侧边栏、撤销/重做操作、展品编号显示
2. **展品信息面板**：名称、艺术家、创作年份、描述文字、语音解说文本
3. **预览页面**：全屏沉浸模式、逐件浏览导航、进度条、分享链接生成

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 编辑器页面 | 网格画布 | 6x8格子布局，支持拖拽放置展品，自动吸附到网格交叉点，显示展品编号 |
| 编辑器页面 | 展品库侧边栏 | 可拖拽展品列表（色块+标题占位），支持40件展品容量 |
| 编辑器页面 | 编辑工具 | 撤销(Ctrl+Z)、重做(Ctrl+Y)按钮，历史栈管理 |
| 编辑器页面 | 信息编辑面板 | 从右侧滑入，带半透明遮罩，淡入动画，修改展品详细信息 |
| 预览页面 | 全屏画布 | 平滑缩放动画，画布放大至铺满屏幕 |
| 预览页面 | 展品导航 | 方向键/左右箭头逐件浏览，高亮当前展品 |
| 预览页面 | 进度条 | 底部显示当前展品序号/总数 |
| 预览页面 | 分享功能 | 生成8位随机码短链接，复制到剪贴板，Toast提示 |

## 3. 核心流程

策展用户打开应用 → 进入编辑器页面 → 从侧边栏拖拽展品到网格 → 点击展品打开信息面板 → 填写展品信息并保存 → 点击预览按钮进入全屏模式 → 浏览展品 → 点击分享按钮生成链接 → 复制链接分享。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰(#2c2c2c)背景 + 暖橙(#f5a623)强调色
- **圆角**：所有卡片和面板 12px 圆角
- **阴影**：微投影 0 4px 12px rgba(0,0,0,0.15)
- **动画**：0.3秒缓出过渡，按钮点击scale 0.95反馈
- **字体**：Google Fonts Nunito
- **布局**：1920x1080黄金比例侧边栏(382px) + 主画布区域；768px以下侧边栏折叠为底部抽屉

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 编辑器页面 | 顶部导航 | 深灰背景，暖橙品牌色，预览按钮，撤销/重做按钮 |
| 编辑器页面 | 左侧展品库 | 卡片式展品列表，可拖拽，色块+标题占位 |
| 编辑器页面 | 中央画布 | 6x8浅灰网格线，展品卡片吸附定位，编号显示 |
| 编辑器页面 | 右侧滑入面板 | 半透明黑色遮罩，白色面板从右滑入，输入框淡入动画 |
| 预览页面 | 全屏画布 | 缩放动画，当前展品高亮边框，暖橙色指示 |
| 预览页面 | 展品详情 | 右侧信息卡片，名称/艺术家/年份/描述分层次显示 |
| 预览页面 | 底部控制栏 | 进度条，左右箭头按钮，分享按钮 |

### 4.3 响应式设计

- 桌面端（≥1920px）：侧边栏382px黄金比例 + 主画布
- 平板端（768-1919px）：侧边栏缩小至300px，画布自适应
- 移动端（<768px）：侧边栏折叠为底部抽屉式，画布垂直排列

### 4.4 性能指标

- 画布交互帧率：≥45fps
- 40件展品首次加载：≤2秒
- 动画流畅度：GPU加速CSS transitions
