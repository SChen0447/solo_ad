# 社区花园农活任务协同与工具共享应用 - 产品需求文档 (PRD)

## 1. 产品概述
社区花园农活协同应用旨在解决社区花园成员之间农活任务协调困难、工具借还混乱的问题，提供数字化的任务分配、工具共享、产出记录和积分激励功能。

## 2. 目标用户
- 社区花园管理员：创建和管理任务、工具库存
- 社区花园成员：认领任务、预约工具、记录产出

## 3. 功能需求

### 3.1 任务看板模块
- 支持创建新任务（浇水、施肥、除草、采摘）
- 任务字段：标题、截止日期、紧急程度（普通/紧急/非常紧急）
- 紧急任务卡片显示红色闪烁边框动画（每2秒一次）
- 任务状态流转：待认领 → 进行中 → 已完成 → 历史记录
- 用户可点击认领、完成任务

### 3.2 工具预约模块
- 展示工具库存（水桶、铲子、修枝剪等）
- 显示当前借用人、预计归还时间
- 支持选择未来3天内日期和时段（上午/下午/全天）预约
- 已预约时段标记：灰色半透明遮挡 + 小锁图标

### 3.3 花园产出记录
- 成员可记录采摘产出（品类、重量/数量）
- 按周汇总统计近4周产量
- 柱状图展示产量变化趋势

### 3.4 积分排名系统
- 完成任务：+10分
- 工具按时归还：+5分
- 记录产出：每100g +1分
- 排行榜显示前5名，头像为圆形首字母（40px直径）
- 当前登录用户高亮显示（背景色#FEF3C7）

## 4. 界面设计规范

### 4.1 配色方案
- 主色：#2E7D32（深绿）
- 辅色：#66BB6A（浅绿）
- 背景：#F1F8E9（淡绿）
- 高亮：#FEF3C7（淡黄）
- 文字辅助：#9CA3AF（灰）

### 4.2 布局规范
- 头部导航栏高度：56px（移动端375px时48px）
- 主内容区：三列网格（任务40% / 工具35% / 排名25%）
- 面板标题栏：高度40px，圆角8px
- 面板内容区最大高度：calc(100vh - 120px)
- 卡片：白色背景、圆角12px、阴影 0 2px 8px rgba(0,0,0,0.08)

### 4.3 动画规范
- 内容切换：淡入+上移20px，0.3秒 ease-out
- 紧急任务：红色边框闪烁，每2秒一次

### 4.4 响应式设计
- 桌面端：三列网格布局
- ≤768px：堆叠单列布局，高度自动扩展
- ≤375px：导航栏高度48px，字体缩小

## 5. 性能约束
- 列表初次加载 ≤1.5秒（10任务/8工具/20成员）
- 任务认领/工具预约接口响应 ≤300ms

## 6. 技术栈
- 前端：React 18.2 + TypeScript 5.3 + Vite 5.0
- 后端：Express 4.18 + Node.js
- 数据存储：内存存储
- 跨域：cors 2.8.5
- ID生成：uuid 9.0.0
