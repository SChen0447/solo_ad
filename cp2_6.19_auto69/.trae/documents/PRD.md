## 1. 产品概述

节拍跑酷是一款基于节奏点击的街机风格无限跑酷游戏。玩家控制像素风格红色方块小人在无限跑道上奔跑，跑道上的障碍物和金币与背景电子鼓点节拍同步生成，玩家需要按节拍跳跃或滑铲来躲避障碍并收集金币，跑得越远分数越高。

- 目标用户：喜欢节奏游戏和街机风格休闲游戏的玩家
- 核心价值：将音乐节拍与跑酷操作深度结合，创造独特的视听同步游戏体验

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：全屏Canvas画布，深蓝到紫色径向渐变背景，白色虚线跑道，角色奔跑动画
2. **游戏结束界面**：最终分数、最高纪录（打字机效果）、重新开始按钮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 节拍系统 | Web Audio API生成0.5秒间隔电子鼓点，节拍脉冲光晕，警示圈动画 |
| 游戏主界面 | 角色控制 | 空格跳跃（360度旋转，0.6秒），向下箭头滑铲（压缩1/3高度，0.4秒白色粒子拖尾） |
| 游戏主界面 | 障碍物系统 | 尖刺（红色三角，触碰死亡）、矮墙（灰色矩形，必须跳跃）、横杆（蓝色横条，必须滑铲） |
| 游戏主界面 | 金币系统 | 普通金币（黄色圆片，+10分，旋转发光）、节拍金币（紫色星形，+50分，闪烁紫色粒子） |
| 游戏主界面 | 分数系统 | 顶部白色大字分数（金色阴影），连击计数，>5连击橙色脉冲动画，分数弹出动画 |
| 游戏主界面 | 音量控制 | 右下角半透明滑块，悬停显现放大1.1倍 |
| 游戏结束界面 | 结果展示 | 最终分数、最高纪录（localStorage，打字机效果逐字出现） |

## 3. 核心流程

玩家启动游戏后，角色自动在跑道上奔跑。背景音乐以0.5秒间隔播放电子鼓点节拍，每次节拍时：
1. 画面边缘产生脉冲光晕（0.2秒，透明度0.4→0）
2. 节拍前0.1秒出现红色警示圈（半径0→1.5，持续0.3秒）
3. 节拍瞬间障碍物或金币实体弹出

玩家根据障碍类型按空格跳跃或按向下箭头滑铲，成功躲避障碍或收集金币获得分数。连续按节拍收集金币累积连击数，连击>5时分数变橙色并脉冲。触碰尖刺或未正确躲避障碍则游戏结束。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色（#0a0a2e）到紫色（#4a0e8f）径向渐变背景
- **强调色**：红色（角色/尖刺）、黄色（金币）、紫色（节拍金币/光晕）
- **跑道风格**：白色虚线两条平行线，赛博朋克感
- **字体**：像素风/等宽字体，白色大字配金色阴影
- **布局**：全屏Canvas，无滚动，纯游戏画面
- **动画风格**：脉冲、闪烁、粒子拖尾等电子风格动效

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 背景 | 深蓝到紫色径向渐变，边缘脉冲光晕 |
| 游戏主界面 | 跑道 | 白色虚线两条平行线，随速度滚动 |
| 游戏主界面 | 角色 | 红色方块小人，奔跑/跳跃/滑铲动画 |
| 游戏主界面 | 障碍物 | 红色三角尖刺、灰色矩形矮墙、蓝色横条横杆 |
| 游戏主界面 | 金币 | 黄色旋转圆片、紫色闪烁星形 |
| 游戏主界面 | HUD | 顶部白色分数（金色阴影）、连击数、右下角音量滑块 |
| 游戏主界面 | 警示圈 | 红色圆环，半径0→1.5，淡入效果 |
| 游戏结束界面 | 结果 | 最终分数、最高纪录（打字机效果） |

### 4.3 响应式

- 桌面优先设计，Canvas自适应窗口大小
- 键盘操作（空格/下箭头）

### 4.4 性能要求

- 目标60FPS流畅运行
- 帧率<50时自动降级：粒子数50→25，光晕分辨率全→半
