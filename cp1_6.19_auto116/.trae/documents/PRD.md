## 1. 产品概述

代码片段分享与语法高亮展示平台，让开发者快速粘贴代码、选择语言、查看实时高亮效果，并生成永久分享链接。目标用户为前端/后端开发者，解决代码片段临时分享缺乏美观展示和持久化存储的痛点。

## 2. 核心功能

### 2.1 功能模块

1. **首页（瀑布流列表）**：代码片段卡片网格展示，按语言筛选，点击进入详情
2. **编辑器页面**：粘贴代码、选择语言、实时高亮预览、生成分享链接

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 瀑布流卡片列表 | 以网格排列展示所有代码片段，每张卡片显示前10行代码预览、语言标签和创建时间 |
| 首页 | 语言筛选栏 | 标签式点击切换，带弹性动画，点击高亮切换筛选语言 |
| 编辑器页 | 代码编辑区 | textarea带行号，实时highlight.js语法高亮，语言下拉选择（至少8种），切换时淡入动画 |
| 编辑器页 | 元信息卡片 | 语言标签（圆角彩色徽章）、创建时间、字符数、行数，卡片阴影+悬停上移2px+阴影加深，一键复制代码按钮 |
| 编辑器页 | 分享链接生成 | 代码片段pako压缩后存入localStorage，生成短哈希URL片段，复制到剪贴板+底部滑入toast（2秒淡出） |
| 详情页 | 完整代码展示 | 查看完整代码和元信息，共享编辑器页的展示组件 |

## 3. 核心流程

用户打开平台 → 首页看到瀑布流代码片段列表 → 点击语言标签筛选 → 点击卡片进入详情页查看完整代码
或：用户在编辑器页粘贴代码 → 选择语言 → 实时查看高亮 → 点击"生成分享链接" → 代码压缩存入localStorage → 生成短哈希URL → 复制到剪贴板 → toast提示成功

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#cba6f7（紫罗兰色），搭配深色背景#1e1e2e，卡片#2a2a3e
- **按钮风格**：圆角按钮，hover时颜色渐变过渡（0.3s ease）
- **字体**：代码区使用JetBrains Mono等宽字体，UI使用系统无衬线字体
- **布局风格**：卡片式布局，瀑布流网格，顶部筛选栏
- **动画风格**：逐项渐入（stagger 0.05s），淡入切换，悬停上移2px

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 筛选栏 | 水平标签列表，选中态带#cba6f7高亮+弹性缩放动画 |
| 首页 | 卡片网格 | 2-3列瀑布流，卡片圆角8px，阴影0 4px 12px rgba(0,0,0,0.3)，代码预览区深色#1e1e2e，悬停上移2px+阴影加深 |
| 编辑器页 | 编辑区 | 左侧行号区（颜色#585b70，分隔线），右侧代码区（JetBrains Mono），highlight.js One Dark Pro高亮 |
| 编辑器页 | 元信息卡片 | 圆角彩色徽章语言标签，阴影+悬停提升，复制按钮带图标 |
| 编辑器页 | Toast提示 | 底部滑入，#cba6f7背景，2秒后淡出 |

### 4.3 响应式

- 桌面优先设计，最小宽度1024px
- 卡片网格在大屏3列、中屏2列、小屏1列自适应

## 5. 性能指标

- 代码高亮实时渲染延迟 ≤ 100ms
- 首屏5个片段渲染时间 ≤ 800ms
- 所有按钮hover过渡0.3s ease
- 卡片加载逐项渐入stagger 0.05s
- 首屏容器从下往上淡入0.5s
