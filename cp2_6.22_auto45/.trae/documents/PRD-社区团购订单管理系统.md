## 1. 产品概述

社区团购订单管理与合并工具是一个轻量级的内部管理系统，旨在帮助社区团购组织者高效处理微信群内的商品订购信息。系统解决了手工统计订单、合并计算、通知取货等繁琐流程的痛点，大幅提升工作效率。

- **主要用途**：商品目录维护、多用户模拟下单、订单自动合并与总价计算、取货状态跟踪
- **目标用户**：社区团购组织者、管理员
- **市场价值**：减少人工统计错误，缩短订单处理时间，提升社区团购运营效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 系统预设 | 浏览商品、加入购物车、提交订单、查看个人订单 |
| 管理员 | 系统预设 | 商品管理、库存调整、查看合并订单、标记取货状态 |

### 2.2 功能模块

1. **商品列表区**：商品卡片展示、库存状态指示、数量选择浮层
2. **购物车面板**：已选商品列表、数量调整、商品移除、小计计算
3. **用户切换器**：预设用户下拉选择（张三、李四、王五）
4. **订单面板**：当前用户订单查看、订单详情、数量修改、订单取消
5. **合并视图**：按商品合并订单、总数量/金额统计、取货状态标记与过滤
6. **统计概览栏**：总订单数、商品种类数、总金额、取货完成进度
7. **商品管理（管理员）**：新增商品、调整库存

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主界面 | 用户切换器 | 顶部下拉框切换3个预设用户，数据同步切换 |
| 主界面 | 商品网格 | 商品卡片展示，点击弹出数量选择浮层，库存状态可视化 |
| 主界面 | 购物车面板 | 左侧300px面板，实时显示购物车条目，支持增减和移除 |

| 主界面 | 统计概览栏 | 底部统计栏，4项核心指标，进度条展示取货完成率 |
| 主界面 | 订单面板 | 展示当前用户历史订单，支持修改和取消 |
| 主界面 | 商品管理区 | 管理员功能，表单添加新商品，调整现有商品库存 |

## 3. 核心流程

### 3.1 用户下单流程

用户选择身份 → 浏览商品列表 → 点击商品选择数量 → 加入购物车 → 确认购物车 → 提交订单 → 后端自动合并所有订单 → 前端展示合并结果

```mermaid
flowchart LR
    A["选择用户身份"] --> B["浏览商品卡片"]
    B --> C["点击商品弹出数量选择"]
    C --> D["确认数量加入购物车"]
    D --> E["购物车内调整/移除"]
    E --> F["点击提交订单"]
    F --> G["后端存储订单并合并"]
    G --> H["前端更新合并视图"]
```

### 3.2 取货管理流程

管理员切换到合并视图 → 查看按商品合并的订单 → 按商品或按用户标记取货 → 记录取货时间戳 → 视图更新并可按状态过滤

```mermaid
flowchart LR
    A["进入合并视图"] --> B["查看合并订单表格"]
    B --> C["逐商品/逐用户标记取货"]
    C --> D["记录取货时间戳"]
    D --> E["视觉反馈（置灰+绿色对勾）"]
    E --> F["选择状态过滤（全部/未取/已取）"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主背景色**：暖白 #FFFBEB
- **重点强调色**：橙棕色 #D97706
- **成功/取货色**：绿色 #10B981
- **警告/低库存色**：红色 #EF4444
- **次背景色**：浅黄 #FEF3C7 / #FEF9C3
- **按钮样式**：圆角8px，悬停缩放1.05倍，背景色加深0.2s过渡
- **卡片样式**：圆角16px，白底，底部6px橙棕色色条，悬停抬起6px并加深阴影
- **字体**：优先选用 Noto Sans SC 或 PingFang SC，确保中文显示美观
- **布局风格**：左中右三栏布局，桌面端并排，移动端堆叠
- **图标风格**：简洁线条风格，使用 emoji 辅助（🛒📦✅❌等）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主界面 | 顶部导航 | 用户下拉选择器、管理员模式开关、页面标题 |
| 主界面 | 商品卡片网格 | 自适应网格，200×280px卡片，库存指示点（绿/红闪烁） |
| 主界面 | 购物车面板 | 300px宽，#FEF3C7背景，圆角12px带阴影，条目列表+小计+提交按钮 |
| 主界面 | 合并视图面板 | 400px宽，表格布局，行高48px，偶数行#FEF9C3，取货按钮 |
| 主界面 | 数量选择浮层 | 模态窗口，#FFFFFF背景，圆角16px，半透明遮罩，数字输入+确认取消 |
| 主界面 | 统计概览栏 | 底部固定栏，4个数据卡片，进度条#10B981填充，淡入动画0.3s |

### 4.3 响应式设计

- **桌面端（≥1024px）**：左（购物车）-中（商品网格）-右（合并视图）三栏并排布局
- **平板/移动端（<1024px）**：三栏转为上下堆叠，购物车在上、商品居中、合并视图在下
- **触摸优化**：所有可点击区域最小高度44px，增大触摸目标
- **断点**：1024px为主要断点

### 4.4 动效与性能

- **卡片悬停**：transform: translateY(-6px)，box-shadow 加深，0.3s ease
- **按钮悬停**：transform: scale(1.05)，背景色加深，0.2s 过渡
- **低库存卡片**：红色指示点轻微闪烁动画
- **统计数字更新**：0.3s 淡入动画
- **性能目标**：页面切换流畅≥50fps，50条合并视图渲染≤200ms，过滤操作≤50ms
