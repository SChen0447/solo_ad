# 摄影比赛在线作品提交与评委打分系统 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
面向摄影爱好者和专业评委的在线摄影比赛平台，提供作品上传、多维度评分、自动排名的一站式解决方案。

### 1.2 目标用户
- **参赛用户**：摄影爱好者，注册登录后上传作品参赛
- **评委用户**：受邀专业人士，登录后对作品进行多维度打分
- **管理员**：管理比赛进程，导出比赛结果

### 1.3 核心价值
- 简化比赛组织流程，降低管理成本
- 提供专业、客观的多维度评分体系
- 实时展示比赛结果，增加比赛透明度

---

## 2. 功能需求

### 2.1 用户模块

#### 2.1.1 用户注册
- 支持用户名、邮箱、密码注册
- 密码加密存储
- 注册成功自动登录

#### 2.1.2 用户登录
- 支持邮箱/用户名 + 密码登录
- 登录状态持久化（localStorage）
- 退出登录功能

#### 2.1.3 角色区分
- 普通用户：可上传作品、浏览作品
- 评委用户：可查看待评分作品、进行打分
- 角色在用户表中标记（is_judge字段）

### 2.2 作品上传模块

#### 2.2.1 上传功能
- 支持JPG、PNG格式
- 单张图片不超过5MB
- 拖拽上传或点击选择
- 上传进度显示

#### 2.2.2 作品信息
- 作品标题（必填）
- 创作说明（最多200字）
- 自动记录上传时间和上传用户

#### 2.2.3 图片处理
- 自动生成缩略图（16:9比例）
- 原图存储用于详情页展示

### 2.3 作品展示模块

#### 2.3.1 主页网格展示
- 响应式网格布局
- 桌面端：多列布局，卡片间距20px
- 移动端（<768px）：单列布局，卡片间距12px
- 卡片包含：缩略图（16:9，圆角12px，1px浅灰边框#e0e0e0）、标题、用户名
- 悬停效果：向上平移6px，添加柔和阴影（rgba(0,0,0,0.15)），0.3秒缓动

#### 2.3.2 图片懒加载
- 使用IntersectionObserver实现
- 加载占位图
- 首屏加载时间不超过2秒

#### 2.3.3 作品详情页
- 展示完整图片
- 显示作品标题、创作说明、作者、上传时间
- 评委可在此页进入打分界面

### 2.4 评委打分模块

#### 2.4.1 待评分列表
- 按上传时间倒序排列
- 显示作品缩略图、标题、上传时间
- 标记已打分/未打分状态

#### 2.4.2 四维打分
- **构图**：1-10分（整数）
- **色彩**：1-10分（整数）
- **创意**：1-10分（整数）
- **情感表达**：1-10分（整数）
- 使用滑块组件（轨道#e0e0e0，按钮#5a67d8）

#### 2.4.3 雷达图展示
- Canvas绘制五边形雷达图
- 配色：#667eea、#764ba2、#f093fb、#f5576c、#4facfe
- 背景浅灰半透明，网格线浅色虚线
- 实时展示各维度得分

#### 2.4.4 打分提交
- 提交后显示总分
- 记录打分时间和评委信息
- 每个评委对同一作品只能打分一次

### 2.5 排名榜单模块

#### 2.5.1 自动排名
- 比赛结束后自动计算排名
- 按平均分从高到低排列
- 显示：排名、作品缩略图、标题、作者

#### 2.5.2 统计信息
- 平均分（保留两位小数）
- 最高分
- 最低分
- 每个维度的平均分

#### 2.5.3 条形图展示
- 四个维度平均分横向条形图
- 颜色渐变：#3b82f6 → #8b5cf6

#### 2.5.4 CSV导出
- 一键导出完整榜单为CSV
- 包含：排名、作品标题、作者、各维度分数、平均分、最高分、最低分

### 2.6 导航栏模块

#### 2.6.1 布局
- 高度60px
- 纯白背景
- 底部浅灰分割线#e5e7eb

#### 2.6.2 Logo
- 左侧位置
- 粗体字体
- 颜色#1f2937

#### 2.6.3 菜单项
- 中间位置
- 字体大小14px
- 间距24px
- 激活项带下划线动画
- 激活颜色#3b82f6

#### 2.6.4 用户区域
- 右侧位置
- 登录状态显示用户名
- 未登录显示登录/注册按钮

## 3. 非功能需求

### 3.1 性能要求
- 列表页首次加载时间 ≤ 2秒
- 图片懒加载，减少首屏资源请求
- API响应时间 ≤ 500ms

### 3.2 兼容性要求
- 支持Chrome、Firefox、Safari、Edge最新版本
- 响应式设计，适配桌面端和移动端

### 3.3 安全要求
- 密码加密存储（bcrypt）
- 用户身份验证（JWT Token）
- 文件上传类型和大小校验
- 防止SQL注入和XSS攻击

### 3.4 体验要求
- 页面切换过渡动画
- 表单验证实时反馈
- 加载状态和错误提示
- 操作成功/失败Toast提示

## 4. 页面结构

| 页面路径 | 页面名称 | 访问权限 |
|---------|---------|---------|
| / | 首页（作品列表） | 公开 |
| /login | 登录页 | 公开 |

## 5. 数据模型

### 5.1 用户表（users）
- id: 主键
- username: 用户名（唯一）
- email: 邮箱（唯一）
- password_hash: 密码哈希
- is_judge: 是否为评委
- created_at: 创建时间

### 5.2 作品表（works）
- id: 主键
- title: 作品标题
- description: 创作说明
- image_path: 图片路径
- thumbnail_path: 缩略图路径
- user_id: 上传用户ID
- created_at: 上传时间

### 5.3 评分表（scores）
- id: 主键
- work_id: 作品ID
- judge_id: 评委ID
- composition: 构图分数
- color: 色彩分数
- creativity: 创意分数
- emotion: 情感表达分数
- total_score: 总分
- created_at: 打分时间

## 6. 视觉设计规范

### 6.1 色彩系统
| 用途 | 颜色值 |
|-----|-------|
| 主色调 | #3b82f6 |
| 强调色 | #5a67d8 |
| 背景色 | #f9fafb |
| 卡片背景 | #ffffff |
| 文字主色 | #1f2937 |
| 文字次色 | #6b7280 |
| 边框色 | #e5e7eb |
| 卡片边框 | #e0e0e0 |

### 6.2 间距系统
- 基础间距：4px
- 常用间距：8px、12px、16px、20px、24px、32px、48px

### 6.3 圆角系统
- 卡片圆角：12px
- 按钮圆角：8px
- 输入框圆角：6px

### 6.4 字体系统
- 标题字体：系统无衬线字体，粗体
- 正文字体：系统无衬线字体，常规
- 代码字体：等宽字体

### 6.5 动画规范
- 过渡时间：0.3s
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- 悬停位移：6px向上平移

## 7. 技术选型（见技术架构文档）

- 前端：React 18 + TypeScript + Vite
- 状态管理：React Hooks + Context
- 路由：React Router v6
- HTTP客户端：Axios
- 动画：Framer Motion
- 后端：Python Flask
- 数据库：SQLite（开发环境）
- 认证：JWT
