# 3D分子结构查看器 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
一款基于WebGL的交互式3D分子结构可视化工具，面向化学教育、科研人员和学生，提供直观的分子三维结构展示与交互操作。

### 1.2 目标用户
- 化学专业学生和教师
- 科研人员进行分子结构可视化
- 对分子结构感兴趣的科普爱好者

### 1.3 核心价值
通过沉浸式3D交互体验，帮助用户直观理解分子的空间结构、键角关系和原子排布，降低化学结构学习门槛。

---

## 2. 功能需求

### 2.1 分子模型库
- **预设分子**：内置水分子（H₂O）、甲烷（CH₄）、苯环（C₆H₆）三种经典分子模型
- **模型参数**：
  - 水分子：2个氢原子 + 1个氧原子，O-H键角104.5°
  - 甲烷：1个碳原子 + 4个氢原子，正四面体结构
  - 苯环：6个碳原子 + 6个氢原子，平面环状结构

### 2.2 3D交互操作
- **旋转查看**：鼠标拖拽自由旋转分子，支持阻尼效果
- **缩放操作**：鼠标滚轮缩放，范围0.5-5倍
- **平移拖拽**：右键拖拽平移视角
- **自动面向**：原子标签始终面向屏幕，键角标注清晰可读

### 2.3 显示控制
- **原子标签**：可开关显示元素符号（H、O、C），位于原子球体上方
- **键角标注**：可开关显示相邻键之间的角度数值，灰色虚线连接
- **重置视角**：一键恢复初始视角

### 2.4 视觉效果
- **球棍模型**：原子为球体，化学键为圆柱体
- **半透明效果**：使用浅蓝半透明球谐光照
- **泛光效果**：UnrealBloomPass实现微弱发光
- **悬停高亮**：鼠标悬停时原子外发光，光标变为手型
- **平滑过渡**：分子切换时0.5秒淡出淡入动画

## 3. 界面设计

### 3.1 整体风格
- **深色主题**：背景色 #1a1a2e，营造科技感
- **主题色**：浅蓝 #4fc3f7 用于交互控件
- **原子配色**：
  - 氧原子：红色 #ff0000，半径0.6
  - 氢原子：白色 #ffffff，半径0.3
  - 碳原子：需定义默认颜色

### 3.2 布局结构
- **渲染区域**：占满全屏（100vw × 100vh）
- **控制面板**：固定右下角，距离边缘20px
  - 半透明深色背景 rgba(16,16,32,0.8)
  - 圆角12px，内边距16px
  - 白色文字，浅蓝主题色控件

### 3.3 控制面板组件
1. **分子选择下拉框**：水 / 甲烷 / 苯环
2. **原子标签开关**：Checkbox
3. **键角显示开关**：Checkbox
4. **重置视角按钮**：Button

### 3.4 交互动效
- 面板弹出动画：切换分子时 scale 1→1.05，时长0.2s
- 模型切换：0.5s淡出旧模型 + 0.5s淡入新模型
- 悬停反馈：原子高亮，光标手型

---

## 4. 技术约束

### 4.1 技术栈
- **语言**：TypeScript
- **3D引擎**：Three.js
- **构建工具**：Vite
- **UI框架**：dat.gui（控制面板）
- **性能监控**：stats.js

### 4.2 性能要求
- 帧率：稳定60FPS
- 最大规模：支持100个原子 + 150个键
- 渲染优化：BufferGeometry + InstancedMesh
- 抗锯齿：sampleLevel=2

### 4.3 浏览器兼容
- 支持WebGL 2.0的现代浏览器
- Chrome 90+ / Firefox 88+ / Edge 90+

---

## 5. 非功能需求

### 5.1 可扩展性
- 分子数据与渲染逻辑分离，便于新增分子类型
- 模块化架构，各组件职责单一

### 5.2 可维护性
- 严格TypeScript类型检查
- 清晰的代码组织结构
- 遵循SOLID设计原则

### 5.3 用户体验
- 首屏加载时间 < 3s
- 操作响应延迟 < 100ms
- 直观的操作指引（可通过UI暗示）
