## 1. 产品概述

星辰棋局是一款双人策略棋盘游戏应用，两位玩家在8x8（或10x10）的宇宙主题网格棋盘上轮流放置不同颜色的星灵代币，通过占据行星节点和激活星座连线来获取分数，率先达到目标分数者获胜。

- 目标用户：喜欢策略对战的休闲玩家
- 核心价值：将经典棋盘博弈与宇宙星象主题融合，通过行星节点和星座连线机制增加策略深度

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 玩家1（红方） | 自动分配 | 放置红色星灵代币、使用星灵技能 |
| 玩家2（蓝方） | 自动分配 | 放置蓝色星灵代币、使用星灵技能 |

### 2.2 功能模块
1. **游戏主界面**：棋盘渲染、代币放置、分数面板、回合指示、技能操作
2. **难度选择界面**：普通/进阶/大师三种模式预设
3. **结算界面**：获胜庆祝动画、对局统计、得分曲线图

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 棋盘区域 | 渲染8x8/10x10网格，支持代币放置、行星节点标记、星座连线高亮 |
| 游戏主界面 | 分数面板 | 顶部左右两侧显示双方分数，激活状态发光效果 |
| 游戏主界面 | 回合指示器 | 中央上方，当前回合玩家代币图标脉动放大 |
| 游戏主界面 | 技能按钮 | 棋盘下方，能量槽六个圆点，满电闪烁 |
| 游戏主界面 | 难度选择 | 游戏开始前选择三种模式 |
| 结算界面 | 获胜庆祝 | 粒子爆炸动画、获胜玩家信息 |
| 结算界面 | 对局统计 | 总回合数、最长连线、技能使用次数、双方得分变化曲线图 |

## 3. 核心流程

玩家进入游戏 → 选择难度模式 → 初始化棋盘（随机生成行星节点）→ 玩家1放置代币 → 检测星座连线 → 计分 → 轮到玩家2 → 重复直到一方达到目标分数 → 显示获胜庆祝 → 展示对局统计

    "是否达到目标分数" --> "否" --> "玩家2回合"

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：宇宙深空主题，背景从#0a0a23到#1a1a3e的径向渐变
- 辅助色：半透明深蓝#16213e（格子背景）、金色（行星节点）、淡蓝色光晕（选中效果）
- 玩家1色：红#ff4757到#ff6b81渐变
- 玩家2色：蓝#3742fa到#5352ed渐变
- 字体：Google Fonts Orbitron（科幻风格显示字体）
- 按钮风格：圆角、深蓝底色、发光边框
- 布局：Flexbox居中，最小宽度900px，窄屏等比缩放

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 棋盘区域 | 8x8网格（每格65x65px）、深蓝边框#1a1a2e、行星节点金色星形、代币圆形渐变、选中格淡蓝光晕脉动、放置缩放动画0.2s |
| 游戏主界面 | 分数面板 | 顶部左红右蓝、大号数字、激活发光效果 |
| 游戏主界面 | 回合指示器 | 中央上方、当前玩家代币脉动放大 |
| 游戏主界面 | 技能按钮 | 棋盘下方、能量槽6圆点、满电闪烁、替换对方代币功能 |
| 游戏主界面 | 难度选择 | 三种模式按钮、普通/进阶/大师 |
| 结算界面 | 获胜庆祝 | 粒子爆炸动画、获胜信息 |
| 结算界面 | 对局统计 | 回合数、最长连线、技能使用次数、得分曲线图 |

### 4.3 响应式设计
- 桌面优先，最小宽度900px
- 窄屏下棋盘等比缩放但保持可点击
- 所有交互0.3秒ease-out过渡
- 错误操作触发摇晃动画

### 4.4 动画细节
- 代币放置：缩放动画0.2s
- 选中/可放置格子：淡蓝色光晕循环脉动
- 回合指示：代币脉动放大
- 技能能量满：圆点变亮闪烁
- 获胜庆祝：粒子爆炸动画
- 错误操作：摇晃动画
- 星座连线：高亮显示动画
