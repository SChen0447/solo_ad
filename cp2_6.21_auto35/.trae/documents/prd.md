## 1. 产品概述

DIY香氛调配与社区分享平台——一个面向香水爱好者的创意工坊，用户可以按比例混合前中后调香料、生成可视化香料轮盘、并在社区中分享和发现独特配方。

- 核心痛点：香水爱好者缺乏直观的调香工具来记录和可视化配方构成
- 目标用户：DIY调香爱好者、香水收藏家、芳香疗法实践者
- 产品价值：将抽象的香气构成转化为可视化轮盘，降低调香门槛，建立配方分享社区

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览社区、调香工作台 |
| 创作者 | 自动成为 | 发布配方、评论点赞、收藏配方 |

### 2.2 功能模块

1. **调香工作台页面**：香料选择面板、调香工作台、香料轮盘实时预览
2. **社区发现页面**：帖子卡片流、配方详情页、评论互动

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 调香工作台 | 香料选择面板 | 按香调分组展示香料卡片，点击添加至工作台 |
| 调香工作台 | 调香工作台 | 垂直条状工作区，显示已选香料标签及比例滑块 |
| 调香工作台 | 香料轮盘 | SVG圆环图实时可视化香料构成比例 |
| 社区发现 | 帖子卡片流 | 瀑布流展示配方卡片，含缩略轮盘、点赞和评论数 |
| 社区发现 | 配方详情页 | 全屏覆盖展示完整轮盘、比例列表、评论区 |

## 3. 核心流程

用户进入社区发现页浏览配方卡片 → 点击导航栏"调香台"按钮切换至工作台 → 在香料选择面板选择香料 → 调节各香料比例滑块 → 实时查看香料轮盘变化 → 保存配方并发布到社区 → 返回社区查看他人配方 → 点赞/收藏/评论互动

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：温暖奶油色系（主色#FFF8E1，辅色#FFCC80，强调色#FF8A65）
- **香调色彩系统**：前调浅黄#FFF3E0、中调淡粉#FCE4EC、后调浅棕#EFEBE9
- **按钮风格**：圆角8px，渐变色按钮（#FF8A65→#FF6F00），评论提交渐变绿（#4CAF50→#2E7D32）
- **字体**：中文使用系统默认圆体，英文使用 Playfair Display 作为展示字体搭配 Noto Sans SC 作为界面字体
- **布局风格**：固定顶部导航栏，左右分栏调香工作台，卡片流社区
- **图标风格**：Lucide线性图标，爱心/气泡等语义化图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 调香工作台 | 香料选择面板 | 三组香调卡片（圆角8px，宽度自适应≤120px），背景色按香调区分 |
| 调香工作台 | 工作台区域 | 280px宽垂直条，米白#FFF8F0背景，2px虚线边框#D7CCC8，香料标签+滑块 |
| 调香工作台 | 香料轮盘 | SVG圆环图，扇形按比例分配，悬停外扩5px+提示框，切换旋转动画0.4s |
| 社区发现 | 帖子卡片 | 320px宽，圆角12px，白底#FFFFFF，0.5px浅灰#E0E0E0边框，8px阴影 |
| 社区发现 | 详情页 | 全屏覆盖，从右滑入0.3s，60%黑遮罩，完整轮盘+比例列表+评论区 |
| 全局 | 导航栏 | 固定顶部56px，磨砂玻璃效果，右侧渐变调香台按钮 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度1024px
- 调香工作台左侧面板和右侧工作区按比例自适应
- 社区卡片流在小屏幕下单列排列
- 触摸优化：滑块控件增大触摸区域

### 4.4 动效设计

- 香料轮盘切换：0.4秒旋转展开动画
- 详情页进入：从右滑入0.3秒 + 60%黑遮罩淡入
- 卡片悬停：微上浮 + 阴影加深
- 轮盘扇形悬停：外扩5px + 提示框
- 导航栏：磨砂玻璃背景模糊效果
