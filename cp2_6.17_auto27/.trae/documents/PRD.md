## 1. 产品概述

室内空间光照模拟与光源编辑器是一个基于Web的3D交互应用，让用户能够在可视化的室内场景中添加、移动、旋转和调整点光源与聚光灯，实时查看光照效果（阴影、漫反射、光晕），并支持日/夜模式切换与导出渲染快照。
- 目标用户：室内设计师、照明工程师、家装爱好者
- 核心价值：零门槛的室内光照可视化预览，实时交互反馈

## 2. 核心功能

### 2.1 功能模块

1. **3D室内场景页面**：5m×4m×3m房间、隔断墙、家具轮廓、初始3个点光源、实时阴影渲染
2. **光源编辑控制面板**：光源类型选择、位置/颜色/强度调节、日/夜模式切换、快照导出

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D室内场景 | 房间渲染 | 渲染5m×4m×3m房间，含地面(浅木色#d2b48c)、墙壁(米白#f5f0e1)、天花板(浅灰#e0e0e0)、隔断墙(深米#dcd3c6, 2.5m高) |
| 3D室内场景 | 家具轮廓 | 矩形餐桌(2m×1m)、两个正方形沙发(0.8m×0.8m)，半透明2D多边形绘制于地面 |
| 3D室内场景 | 初始光源 | 天花板吸顶灯(4000K, 强度0.8)、书桌台灯(3000K, 强度0.5)、墙角落地灯(2700K, 强度0.4) |
| 3D室内场景 | 实时阴影 | 点光源PCF柔和阴影、聚光灯锐利锥形阴影，家具和隔断墙投影带模糊边缘 |
| 3D室内场景 | 场景拖拽 | OrbitControls旋转/缩放，光源gizmo拖拽调整位置 |
| 控制面板 | 光源类型 | 下拉选择点光源/聚光灯 |
| 控制面板 | 位置调节 | X/Y/Z滑块(0-5m, 步长0.1m) |
| 控制面板 | 颜色选择 | 颜色选择器(默认白色) |
| 控制面板 | 强度调节 | 强度滑块(0-2, 步长0.1) |
| 控制面板 | 日/夜切换 | 环境光从冷色(#aaccff)平滑过渡到暖色(#ffccaa)，1.5秒，天空盒从蓝天变星空 |
| 控制面板 | 快照导出 | 相机图标按钮导出1920×1080 PNG，带旋转进度动画 |

## 3. 核心流程

用户打开页面 → 加载默认房间与初始3个光源 → 通过控制面板选择/添加光源 → 调节位置/颜色/强度 → 实时查看光照与阴影变化 → 切换日/夜模式查看不同氛围 → 导出渲染快照

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：浅蓝(#4fc3f7)与暖橙(#ffa726)
- 背景色：深灰蓝(#1a1a2e)
- 按钮风格：圆角卡片、半透明毛玻璃效果
- 字体：系统默认无衬线字体
- 布局风格：顶部浮动控制栏 + 全屏3D场景
- 图标风格：扁平线条图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 3D室内场景 | 全屏Canvas | 占满视口，Three.js渲染3D房间 |
| 控制面板 | 顶部浮动栏 | 高度60px，半透明深色背景rgba(30,30,40,0.85)，左右居中，圆角12px |
| 控制面板 | 滑块控件 | 浅蓝色(#4fc3f7)轨道，圆形拖柄16px，拖拽时轨道变亮 |
| 控制面板 | 日/夜按钮 | 切换按钮，1.5秒CSS过渡动画 |
| 控制面板 | 快照按钮 | 32×32蓝灰色#3f51b5圆角8px，悬停#303f9f，下方旋转进度20px |

### 4.3 响应式设计

- 桌面优先，默认适配1920×1080
- 窗口缩放到1280×720时UI自动缩小80%但保持可用
- CSS过渡动画0.2-0.3秒

### 4.4 3D场景指引

- 环境：室内场景，无HDRI，使用程序化天空盒（蓝天/星空）
- 灯光：3个初始点光源 + 用户可添加的聚光灯，PCF软阴影
- 相机：透视相机，OrbitControls交互，初始俯瞰45度角
- 构图：房间居中，相机默认距离8m
- 交互：光源gizmo拖拽，OrbitControls旋转/缩放
- 性能预算：10个光源内30fps以上，加载3秒内

## 5. 性能要求

- 最多10个光源同时存在
- 普通笔记本(i5-8代，集成显卡)帧率≥30fps
- 场景加载时间≤3秒
- 阴影实时更新，光源调整无卡顿
