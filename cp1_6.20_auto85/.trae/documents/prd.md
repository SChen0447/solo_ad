## 1. 产品概述

AI驱动的在线模拟面试训练应用，为学员提供高频次、低压力的面试练习环境。用户可选择岗位方向，由AI生成面试问题，支持语音或文字作答，并获得多维度评分与反馈。

- 目标用户：求职学员、转行人员、面试准备者
- 核心价值：降低模拟面试成本，提供随时随地的练习机会，通过智能评分帮助学员定位薄弱环节

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学员 | 无需注册 | 选择岗位、参加面试、查看历史记录 |

### 2.2 功能模块

1. **首页**：岗位选择、面试官头像与语音播报、历史记录列表
2. **面试面板**：问题展示、语音录制/文字输入、评分仪表盘、维度反馈

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 岗位选择区 | 提供前端开发、后端开发、产品经理、数据分析师四个岗位卡片，点击选中后高亮 |
| 首页 | 面试官形象 | 圆形卡通头像（背景色#5b9bd5，带微笑表情），选择岗位后出现，中文语音播报欢迎语 |
| 首页 | 历史记录列表 | 展示最近10次面试记录，含日期、岗位、综合得分进度条，可展开查看详情 |
| 面试面板 | 问题展示区 | 聊天气泡样式展示面试问题，左侧头像右侧文字，0.5秒淡入动画 |
| 面试面板 | 回答输入区 | 语音录制按钮（Web Audio API，最长2分钟）和文字输入框（500字符限制，字数统计） |
| 面试面板 | 评分仪表盘 | 综合得分环形进度条（渐变#ff6b6b→#48dbfb），0.8秒缓动动画 |
| 面试面板 | 维度反馈 | 技术准确性(#ff6b6b)、表达逻辑性(#48dbfb)、完整性(#1dd1a1)三维度标签评语 |
| 面试面板 | 流程控制 | 暂停/继续按钮（半透明遮罩）、中途退出（保留已答记录） |

## 3. 核心流程

用户打开应用 → 选择岗位方向 → 面试官出现并语音播报欢迎语 → 点击"开始面试" → 系统随机抽取5道问题 → 逐题展示（聊天气泡样式，淡入动画）→ 用户选择语音或文字作答 → 提交答案 → 后端评分 → 展示评分仪表盘与维度反馈 → 下一题 → 5题完成 → 面试结束 → 保存记录到LocalStorage → 返回首页查看历史

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：#5b9bd5（专业蓝），辅色调：#ff6b6b（红）、#48dbfb（蓝）、#1dd1a1（绿）
- 按钮样式：圆角8px，主色#5b9bd5，悬停#4a8bb5，0.2秒过渡动画
- 字体：Poppins 600用于标题，系统字体用于正文
- 布局：卡片式布局，白色背景，圆角12px，阴影0 4px 12px rgba(0,0,0,0.06)
- 图标：对话气泡图标作为Logo，lucide-react图标库

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 顶部导航栏 | Logo图标+"模拟面试助手"标题，字体Poppins 600 #2c3e50 |
| 首页 | 岗位选择卡片 | 4张卡片flex布局，选中态边框高亮#5b9bd5 |
| 首页 | 面试官区域 | 圆形头像#5b9bd5背景+微笑表情，下方欢迎语文字 |
| 首页 | 历史记录列表 | 白色卡片，flex行布局（日期左、岗位中、得分右），可展开 |
| 面试面板 | 问题气泡 | 左侧圆形头像+右侧气泡(#e6f2ff背景，圆角16px)，0.5s淡入 |
| 面试面板 | 录音按钮 | 圆形按钮，录音时红色脉冲动画(0.6s呼吸效果) |
| 面试面板 | 文字输入 | 多行文本框，底部字数统计，500字符限制 |
| 面试面板 | 评分仪表盘 | 环形进度条，渐变#ff6b6b→#48dbfb，内圈灰色，0.8s缓动 |
| 面试面板 | 维度标签 | 技术(红#ff6b6b)、逻辑(蓝#48dbfb)、完整(绿#1dd1a1)彩色标签 |
| 面试面板 | 暂停遮罩 | 半透明遮罩覆盖，当前问题不可操作 |

### 4.3 响应式设计

- 桌面优先设计，主面板最大宽度800px，左右自适应边距
- 宽度<768px时：主面板100%宽度，左右边距16px，历史记录列表改为纵向排列
- 录音按钮和文字输入框在小屏下自适应宽度

### 4.4 背景与氛围

- 主背景：浅灰蓝渐变 #eef2f7 → #d9e2ec
- 卡片：白色背景，统一圆角12px，阴影0 4px 12px rgba(0,0,0,0.06)
- 卡片间距：20px
