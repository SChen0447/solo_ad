## 1. 产品概述

旅行手账生成器是一款在线照片上传与个性化旅行手账制作应用，解决旅行照片散落手机相册难以组织、手账排版费时费力的问题。目标用户为热爱旅行并希望将旅行记忆整理为美观手账的用户。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传照片、编辑手账、导出图片 |

### 2.2 功能模块
1. **手账编辑页面**：素材上传面板、手账编辑画布、主题切换、文字编辑、导出功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 手账编辑页面 | 素材面板 | 上传照片（最多9张/次，jpg/png），调用后端API生成缩略图，网格展示缩略图，支持拖拽源 |
| 手账编辑页面 | 手账编辑画布 | A4竖版画布（600×800px），网格吸附（20px），照片拖放、调整大小（80-300px）、旋转（0-360°，步进15°），文字标签编辑，纯文字段落添加 |
| 手账编辑页面 | 主题选择 | 5种预设主题（简约白、复古黄、旅行蓝、森林绿、日落橙），0.5秒渐变过渡 |
| 手账编辑页面 | 标题与日期 | 自定义旅行标题（最大20字），日期选择器（默认当天，支持日期范围） |
| 手账编辑页面 | 数据持久化 | localStorage实时保存，刷新自动恢复 |
| 手账编辑页面 | 导出功能 | 300dpi PNG导出，3秒倒计时进度条，自动下载 |
| 手账编辑页面 | 工具栏 | 上传按钮、主题选择下拉、导出按钮、清空按钮 |
| 手账编辑页面 | 分隔条 | 左右区域1:2比例，可拖拽分隔条，拖拽时变粗高亮 |

## 3. 核心流程

用户打开应用 → 上传旅行照片 → 照片出现在左侧素材区 → 拖拽照片到右侧手账画布 → 调整位置/大小/旋转 → 双击添加文字标签或纯文字段落 → 选择主题背景 → 编辑标题和日期 → 导出PNG图片

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：#2A9D8F（青绿色）
- 辅助色：#E9C46A（暖黄色）
- 按钮样式：圆角按钮，悬停上浮（translateY(-2px) + box-shadow增强），点击缩放脉冲（scale 0.95）
- 字体：标题使用 Playfair Display，正文使用 Noto Sans SC
- 布局风格：卡片式布局，左1:右2比例，顶部工具栏
- 图标风格：lucide-react 线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 手账编辑页面 | 工具栏 | 主色调背景，上传/主题/导出/清空按钮，悬停上浮效果 |
| 手账编辑页面 | 素材面板 | 白色卡片背景，3列网格缩略图，拖拽时半透明预览 |
| 手账编辑页面 | 编辑画布 | A4竖版，米白色默认背景，20px网格辅助线，照片带阴影和旋转手柄 |
| 手账编辑页面 | 文字编辑 | 气泡编辑框，缩放消失动画，半透明文字卡片 |

### 4.3 响应式
- 桌面优先设计，最小宽度1024px
- 1024-1280px区间：素材区等比缩小，编辑区保持600px
- 1280px以上：素材区按1:2比例展示

- 不适用
