## 1. 产品概述

3D交互式多城市气候数据对比可视化工具，允许用户从8个预设城市中选择2-4个，在3D场景中直观对比它们在12个月内的气象要素（温度、降水量、日照时长、风速）变化趋势。目标用户为气候数据探索者、地理信息爱好者和教育工作者，核心价值是提供零门槛的3D气候数据交互体验，无需GIS或数据分析软件。

## 2. 核心功能

### 2.1 功能模块

1. **主场景页**：3D可视化场景 + 侧边控制面板，支持曲线叠加与并排柱状图两种对比模式

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景页 | 3D气候场景 | 展示选中城市的气象要素变化曲线或柱状图，支持旋转缩放交互 |
| 主场景页 | 曲线渲染器 | Catmull-Rom平滑曲线，半透明管状体，可点击节点弹出详情卡片 |
| 主场景页 | 柱状图渲染器 | 并排柱状图，渐变填充，柱顶数值标签 |
| 主场景页 | 控制面板 | 城市多选、模式切换、要素切换、月份滑块、图例 |
| 主场景页 | 详情卡片 | 节点点击弹出，毛玻璃背景，含精确数值和极端天气事件 |
| 主场景页 | 坐标轴系统 | X轴(月份)、Y轴(数值)自动刻度，辅助网格线 |

## 3. 核心流程

用户打开页面 → 默认展示北京+上海全年温度曲线 → 通过侧边栏选择城市(2-4个) → 切换对比模式(曲线叠加/柱状图) → 切换气象要素 → 拖动月份滑块探查 → 点击数据节点查看详情卡片

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰蓝(#1e293b)渐变到墨蓝(#0f172a)，强调色蓝(#3b82f6)
- **8色调色盘**：红#ef4444、橙#f97316、黄#eab308、绿#22c55e、青#06b6d4、蓝#3b82f6、紫#8b5cf6、粉#ec4899
- **按钮样式**：深色背景(#334155)，选中时强调色(#3b82f6)，圆角药丸按钮
- **字体**：Inter, system-ui, sans-serif，正文14px，标题16px加粗，白色(#e2e8f0)
- **布局风格**：左侧80%全屏3D场景 + 右侧20%磨砂玻璃侧边面板
- **面板效果**：backdrop-filter: blur(16px)，边框1px rgba(255,255,255,0.15)，圆角16px

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主场景页 | 3D场景区域 | 深色渐变背景，半透明白色坐标轴，发光曲线/柱子，旋转缩放交互 |
| 主场景页 | 侧边面板 | 磨砂玻璃背景，城市多选下拉(复选框)，模式切换按钮，要素药丸按钮组，月份滑块，图例 |
| 主场景页 | 详情卡片 | 毛玻璃背景(blur 10px)，圆角12px，阴影，弹性缩放动画，ESC/外部点击关闭 |
| 主场景页 | 月份滑块 | 底部横向，4px高度半透明轨道，16px圆形按钮，月份刻度圆点 |

### 4.3 响应式适配

- **≥768px**：左侧80% 3D场景 + 右侧20%侧边面板
- **<768px**：3D场景全屏 + 底部80px可展开工具栏（磨砂玻璃）
- **<480px**：控件简化为图标模式

### 4.4 3D场景指导

- **环境**：深色科技感背景，无HDRI，程序化渐变
- **光照**：环境光 + 两个定向光，柔和均匀照明
- **相机**：透视相机，初始正视偏俯视角度，模式切换时平滑移动到正视位置
- **构成**：坐标轴为视觉锚点，曲线/柱子为视觉焦点，地板网格增加空间感
- **交互**：OrbitControls旋转缩放，节点hover放大+发光，点击弹出详情
- **后处理**：节点发光光晕效果(可选bloom)
- **性能预算**：总三角面数≤50000，60FPS响应<50ms
