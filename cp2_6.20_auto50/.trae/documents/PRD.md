## 1. 产品概述

微型团队知识库是一个面向小型技术团队的文档协作平台，用于共享、编辑和管理技术文档与 Wiki 条目。
- 核心目标：解决团队知识沉淀、文档版本追溯、快速信息检索三大痛点
- 目标用户：5-20 人的小型技术团队，开发者、技术负责人、产品经理

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 本地模拟 | 浏览、编辑、创建文档，查看历史版本，使用全文搜索 |

### 2.2 功能模块
1. **首页（文档列表）**：卡片式文档列表、文档分类、快速跳转
2. **文档详情/编辑页**：Markdown 编辑器、版本历史、版本对比、操作面板
3. **全局搜索**：顶部搜索框、关键词高亮、搜索结果下拉面板
4. **文档导航**：桌面端侧边文档树、移动端全屏菜单

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 文档卡片列表 | 卡片宽 280px、圆角 12px、浅蓝渐变背景、左侧 4px 分类色条、悬停上移 4px 动画、点击左→右滑动跳转 |
| 首页 | 文档树导航 | 桌面端宽 260px、可折叠为图标+文字、移动端隐藏 |
| 首页 | 全局搜索框 | 顶部中央、60% 宽度、圆角 20px、半透明白色、输入延迟 200ms 触发搜索、ESC/点击外部关闭、橙色高亮 |
| 文档编辑页 | Markdown 渲染区 | 支持标准 Markdown、代码块深色背景+行号 |
| 文档编辑页 | 浮动操作面板 | 可拖动、圆角 8px、毛玻璃半透明、噪点纹理、保存/历史/分享三按钮 |
| 文档编辑页 | 保存反馈 | 1.5 秒圆形加载动画 → 绿色勾号 |
| 文档编辑页 | 历史版本 | 底部滑入时间线、版本号/作者/修改时间、对比按钮 |
| 文档编辑页 | 版本对比视图 | 左右分栏、差异行浅黄背景、行内删除红/新增绿、右上角关闭 |

## 3. 核心流程

用户进入首页，浏览文档卡片列表 → 通过搜索框或分类树快速定位文档 → 点击卡片（左→右滑动过渡）进入文档详情 → 在编辑区修改 Markdown 内容 → 通过浮动面板点击保存（加载动画→勾号）→ 点击"历史版本"查看时间线 → 选择两个版本点击"对比"查看差异 → 返回首页继续浏览。

## 4. 用户界面设计

### 4.1 设计风格
- 主色：#2563EB（现代蓝），辅色：#10B981（翡翠绿）、#F59E0B（琥珀色）
- 分类色条：5 种预设色（按文档分类随机选取）
- 字体：系统无衬线体 -apple-system, BlinkMacSystemFont；标题 600 字重，正文 400 字重
- 卡片：宽 280px、圆角 12px、浅蓝→白渐变、左侧 4px 彩条、悬停上移 4px + 阴影（过渡 0.2s）
- 操作面板：圆角 8px、backdrop-filter 毛玻璃、噪点纹理、可拖动
- 整体风格：极简主义，最大宽度 1200px 居中

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | 搜索框 | 60% 宽、圆角 20px、半透明白色、橙色高亮匹配 |
| 首页 | 卡片网格 | CSS Grid、桌面 3 列 / 平板 2 列 / 手机 1 列、gap 平滑过渡 |
| 首页 | 侧边导航 | 260px 宽、可折叠、文件树结构 |
| 编辑页 | Markdown 区 | 左栏渲染、代码块深色主题+行号 |
| 编辑页 | 操作面板 | 浮动右栏、3 个按钮、拖动手柄 |
| 编辑页 | 历史时间线 | 底部滑入（0.3s）、垂直时间线样式 |
| 编辑页 | 对比视图 | 双栏并排、高亮差异、右上角关闭 |

### 4.3 响应式
- 桌面端（≥1024px）：左侧 260px 文档树 + 主内容区，Grid 3 列卡片
- 平板端（768px-1023px）：隐藏文档树，Grid 2 列卡片
- 移动端（<768px）：全屏滚动菜单，Grid 1 列卡片，顶部工具栏紧凑布局

### 4.4 性能指标
- 文档列表首次渲染 <200ms
- 全文搜索（100 篇文档）响应 ≤300ms
- 过渡动画均 ≤0.3s，使用 CSS transform/opacity 保证流畅
