## 1. 产品概述

技能工作坊管理平台是一款面向技能爱好者和讲师的在线教学管理工具，解决传统线下技能教学中课程安排混乱、学员作业零散难跟踪、老师反馈不及时的痛点问题。

- 目标用户：技能培训讲师、线上课程学员
- 核心价值：提供课程管理、作业提交批改、实时通知三位一体的完整教学管理闭环
- 产品定位：轻量级、高效率的技能教学工作坊管理工具

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 讲师 | 创建和管理课程、布置作业、批改作业、查看学员进度 |
| 学员 | 浏览课程、查看课程详情、提交作业、查看批改反馈、接收通知 |

### 2.2 功能模块

1. **课程管理模块**：课程列表卡片展示、课程详情查看、课程创建与管理
2. **作业提交与批改模块**：作业查看、作业提交（富文本+文件）、作业批改（反馈+打分）
3. **实时通知系统**：通知气泡弹窗、通知抽屉面板、通知状态管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 课程列表页 | 课程卡片网格 | 以卡片网格形式展示所有课程，支持滚动加载 |
| 课程详情页 | 课程信息+作业列表 | 展示课程详细信息和每堂课的作业要求 |
| 作业提交页 | 提交模态框 | 富文本编辑器+文件上传，支持作业提交 |
| 讲师作业列表 | 作业管理 | 按时间排序的学员作业列表，支持批改操作 |
| 通知抽屉 | 通知面板 | 右侧滑入抽屉，展示所有通知及已读/未读状态 |

## 3. 核心流程

### 3.1 学员学习流程

学员进入课程列表 → 选择课程查看详情 → 查看每堂课的作业要求 → 点击提交作业 → 填写富文本内容并上传文件 → 提交成功 → 收到批改通知 → 查看反馈和分数

```mermaid
flowchart LR
    A["课程列表"] --> B["课程详情"]
    B --> C["查看作业要求"]
    C --> D["提交作业"]
    D --> E["等待批改"]
    E --> F["收到批改通知"]
    F --> G["查看反馈与分数"]
```

### 3.2 讲师批改流程

讲师进入作业列表 → 查看待批改作业 → 点击进入批改 → 输入文字反馈 → 选择星级评分 → 提交批改 → 学员实时收到通知

```mermaid
flowchart LR
    A["作业列表"] --> B["选择待批改作业"]
    B --> C["查看作业内容"]
    C --> D["输入反馈"]
    D --> E["星级评分"]
    E --> F["提交批改"]
    F --> G["学员收到通知"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：紫色 #7C3AED
- **辅助色**：黄色 #FBBF24
- **背景色**：浅灰 #F9FAFB
- **卡片背景**：从浅紫 #E8DEF8 渐变到白色 #FFFFFF
- **进度环颜色**：0-33% 红色 #EF4444，34-66% 琥珀色 #F59E0B，67-100% 绿色 #10B981
- **字体**：采用现代无衬线字体，标题加粗，正文常规
- **按钮风格**：圆角设计，点击时有缩放反馈（缩放至0.95，0.1秒）
- **卡片风格**：圆角 12px，悬停时上移 3px 并增强阴影（0.2秒过渡）
- **图标风格**：使用 emoji 图标（📝 新作业、✅ 批改、🔔 变更）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 课程列表页 | 顶部导航栏 | 应用名、导航链接、用户头像、铃铛图标（未读小红点） |
| 课程列表页 | 课程卡片网格 | 4列网格布局，每卡宽 280px，渐变背景，封面图占 45%，圆形进度环 |
| 课程详情页 | 课程信息区 | 封面大图、标题、讲师、开课时间、课时数、简介 |
| 课程详情页 | 作业列表 | 每堂课作业描述、附件、提交按钮 |
| 作业提交模态框 | 提交表单 | 从底部滑入动画，富文本编辑器，文件拖拽上传区，进度条 |
| 讲师作业列表 | 作业卡片 | 学员头像、提交时间、状态标签（绿色已批改/橙色待批改） |
| 通知气泡 | 弹窗提示 | 右下角弹出，从右侧滑入，320px 宽，3秒后淡出 |
| 通知抽屉 | 侧边面板 | 右侧滑入，380px 宽，按时间倒序，已读/未读状态 |

### 4.3 响应式设计

- **设计原则**：桌面优先，移动端适配
- **断点 768px**：
  - 导航栏变为汉堡菜单，点击展开侧边抽屉
  - 课程卡片网格由 4 列变为 2 列
  - 通知抽屉变为全屏覆盖
- **触摸优化**：按钮和可点击区域最小 40px，确保触摸友好

### 4.4 动效与交互

- **卡片悬停**：0.2 秒上移 3px，阴影增强
- **按钮点击**：0.1 秒缩放至 0.95
- **模态框**：从底部滑入 0.3 秒，背景遮罩 60% 灰黑色
- **通知气泡**：从右侧滑入 0.4 秒，停留 3 秒后淡出
- **通知抽屉**：右侧滑入/滑出 0.3 秒
- **星级评分**：点击时 0.2 秒缩放动画
- **性能要求**：卡片网格滚动加载帧率不低于 55fps
