## 1. 产品概述
连锁书店线上二手书交换平台，会员可发布闲置书籍、申请交换、跟踪物流状态，同时积累积分兑换奖励。通过平台连接读者，让闲置书籍流转起来，实现绿色阅读和社区共建。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 会员 | 模拟登录（选择预设用户） | 发布书籍、浏览交换广场、发起/接受交换请求、查看交换记录、积分查看与排行榜 |

### 2.2 功能模块
1. **交换广场**：书籍卡片瀑布流展示、搜索过滤、排行榜小组件
2. **书籍发布**：上传书籍信息表单
3. **交换请求**：申请交换对话框、收到的请求列表、接受/拒绝操作
4. **交换历史**：交换记录列表、状态跟踪与高亮动画
5. **个人中心**：我的书籍、积分余额、交易明细

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 交换广场 | 瀑布流卡片区 | 展示所有已发布书籍，卡片包含封面、书名、作者、简短描述，支持"申请交换"按钮 |
| 交换广场 | 搜索过滤面板 | 按书名、作者、ISBN搜索，300ms防抖实时刷新，搜索结果带淡入动画 |
| 交换广场 | 排行榜小组件 | 显示积分前十用户排名（头像、昵称、积分），每30秒自动刷新，排名变化带平滑过渡 |
| 书籍发布 | 发布表单 | 输入书名、作者、ISBN、出版年份、封面URL、书籍描述、期望交换类型 |
| 交换请求 | 申请交换对话框 | 选择自己已发布且未交换的书籍作为交换物，确认后发送请求 |
| 交换请求 | 收到的请求列表 | 展示他人发来的交换请求，可接受或拒绝 |
| 交换历史 | 交换记录列表 | 显示双方书籍名称、交换状态（待确认/已接受/已拒绝/已完成）、发起时间和更新时间 |
| 交换历史 | 状态高亮动画 | 状态变更时记录带有高亮闪烁动画（duration 0.3s） |
| 个人中心 | 积分余额卡片 | 显示当前积分，完成交换双方各+10积分 |
| 个人中心 | 交易明细列表 | 时间、对方用户、获得的积分 |
| 个人中心 | 我的书籍列表 | 展示已发布的书籍及交换状态 |

## 3. 核心流程

**书籍发布流程**：用户填写书籍信息 → 提交表单 → 后端存储到JSON文件 → 书籍出现在交换广场

**交换请求流程**：用户A浏览广场 → 点击"申请交换" → 弹出对话框选择自己的书籍 → 确认发送请求 → 用户B在"收到的请求"中查看 → 接受或拒绝 → 若接受则模拟物流完成交换 → 双方各获10积分

```mermaid
    "点击申请交换" --> "选择自己的书籍"
    "选择自己的书籍" --> "确认发送请求"
    "确认发送请求" --> "用户A收到请求通知"
    "用户A收到请求通知" --> "接受请求"
    "用户A收到请求通知" --> "拒绝请求"
    "接受请求" --> "模拟物流配送"
    "模拟物流配送" --> "交换完成"
    "交换完成" --> "双方各获10积分"
    "拒绝请求" --> "交换终止"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色：木色 #D4A574，背景：米白 #FFF8F0，文字：深棕 #5D4037
- 强调色：暖绿 #6B8E23（用于成功状态和积分），柔红 #C75B39（用于拒绝状态）
- 按钮风格：圆角设计（border-radius: 12px），柔和阴影
- 字体：标题使用衬线字体（Playfair Display），正文使用无衬线字体（Lora）
- 布局：卡片式布局，顶部导航
- 图标：使用lucide-react图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 交换广场 | 瀑布流卡片区 | 双栏布局（左70%），卡片圆角12px+柔和阴影，封面图+书名+作者+描述，hover微抬升效果 |
| 交换广场 | 搜索面板 | 右侧30%，搜索输入框+过滤选项，防抖300ms |
| 交换广场 | 排行榜 | 右侧30%，前十排名卡片，头像+昵称+积分，30秒自动刷新 |
| 书籍发布 | 发布表单 | 居中卡片式表单，输入框+下拉选择，提交按钮暖色填充 |
| 交换历史 | 记录列表 | 表格式布局，状态标签颜色区分，状态变更闪烁高亮 |
| 个人中心 | 积分卡片 | 大号积分数字+交易明细时间线 |

### 4.3 响应式设计
- 桌面优先设计，适配768px以上平板
- 768px以下：瀑布流变为单列，右侧面板堆叠到下方
- 所有卡片和按钮保持圆角12px一致性

### 4.4 动画设计
- 搜索结果淡入动画（fadeIn，0.3s）
- 状态变更高亮闪烁（flashHighlight，0.3s）
- Toast通知从上方滑入，2秒后自动滑出（slideIn/slideOut）
- 排行榜排名变化平滑过渡（transform 0.3s ease）
- 卡片hover微抬升（translateY -4px，0.3s）
