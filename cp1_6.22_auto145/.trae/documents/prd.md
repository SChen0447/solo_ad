## 1. 产品概述

塔罗牌神秘学占卜与卡牌收藏网页应用——为塔罗爱好者与玄学好奇者打造的轻量级数字牌馆，集78张完整塔罗牌图鉴浏览、3种经典牌阵占卜、可分享占卜记录卡于一体，解决实体牌携带不便、牌阵记录困难的痛点。

- 目标用户：塔罗爱好者、玄学好奇者、数字收藏爱好者
- 市场价值：将传统塔罗占卜数字化，提供沉浸式星空紫色神秘体验，填补轻量级塔罗网页应用空白

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览图鉴、进行占卜、分享记录卡 |

### 2.2 功能模块
1. **卡牌图鉴页**：22张大阿卡纳卡片展示 + 56张小阿卡纳网格缩略图，点击翻转/弹出详情
2. **占卜页**：3种牌阵选择（三张牌、凯尔特十字、圣三角），自动随机抽牌，拖拽排序，翻转解读
3. **占卜记录卡**：占卜完成后生成可分享的记录卡

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 卡牌图鉴 | 大阿卡纳展示区 | 22张大牌以卡片形式展示（220×340px，圆角12px，深紫背景#2d1b69，边框1px #7c3aed），点击翻转显示正逆位含义、关键词、对应星座 |
| 卡牌图鉴 | 小阿卡纳展示区 | 56张小牌以网格缩略图展示，点击弹出详情浮层 |
| 占卜 | 牌阵选择器 | 三个卡片按钮显示牌阵名称和牌数，选择后触发抽牌请求 |
| 占卜 | 牌桌 | 抽牌结果按牌阵位置水平排列，点击翻转显示解读，支持拖拽重新排序 |
| 占卜 | 记录卡生成 | 占卜完成后生成可分享的占卜记录卡图片 |

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：星空紫色渐变（#0f0c29 → #302b63 → #24243e）
- 辅助色：紫罗兰强调色 #7c3aed，深紫卡片背景 #2d1b69
- 按钮风格：圆角卡片式按钮，悬停发光动画（box-shadow: 0 0 15px #7c3aed）
- 字体：神秘优雅风格，标题使用 Cinzel Decorative，正文使用 Quicksand
- 布局风格：居中卡片布局，水平排列牌阵，网格缩略图
- 图标风格：使用 Lucide 图标库，线性风格配合紫色主题
- 动画：3D CSS transform 翻转效果，鼠标悬停发光，星空粒子背景

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 卡牌图鉴 | 大阿卡纳区 | 卡片220×340px，圆角12px，深紫背景，翻转动画3D perspective，悬停发光 |
| 卡牌图鉴 | 小阿卡纳区 | 网格缩略图，4列布局，点击弹出模态浮层 |
| 占卜 | 牌阵选择 | 三个等宽卡片按钮，图标+名称+牌数，选中高亮 |
| 占卜 | 牌桌 | 水平居中排列，60px间距，拖拽手柄，翻转动画 |
| 占卜 | 记录卡 | 固定尺寸卡片，星空背景，包含牌阵名称+每张牌缩略+解读摘要 |

### 4.3 响应式设计
- 桌面优先设计，移动端自适应
- 移动端卡片宽度缩小为160px
- 触摸优化：拖拽手势支持，点击翻转
- 网格列数自适应：桌面4列 → 平板3列 → 手机2列

### 4.4 性能要求
- 点击抽牌到所有牌出现不超过800ms
- 浏览器缓存已加载的卡牌图片
- 使用 IntersectionObserver 懒加载卡片图片
