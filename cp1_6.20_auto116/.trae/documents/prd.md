## 1. 产品概述

小型社区图书馆自助管理系统，旨在解决管理员手动登记繁琐、读者无法随时查看借阅状态和参加活动的问题。通过数字化手段实现图书自助借阅、逾期智能提醒和读者活动报名管理，提升社区图书馆运营效率和读者体验。

- 目标用户：社区图书馆管理员和社区居民（读者）
- 核心价值：简化借阅流程、实时掌握借阅状态、促进社区文化活动参与

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设账号 | 添加/管理图书、发布活动、查看统计仪表盘 |
| 读者 | 自助注册 | 浏览图书、借阅/归还、查看个人借阅、报名/取消活动 |

### 2.2 功能模块

1. **图书列表页**：图书卡片网格展示、搜索筛选、分页浏览、借阅操作
2. **我的借阅页**：个人借阅列表、逾期倒计时、逾期警告、借阅搜索与分页
3. **活动看板页**：活动列表展示、报名/取消报名、参与头像展示
4. **统计仪表盘**：5项核心统计卡片、近7天借阅趋势折线图（Canvas绘制）

## 3. 核心流程

**图书借阅流程**：读者浏览图书列表 → 搜索/筛选找到目标图书 → 点击"借阅"按钮（需登录）→ 系统创建借阅记录（14天期限）→ 图书可借数减1 → 借阅记录出现在"我的借阅"页面

**逾期提醒流程**：系统实时计算借阅倒计时 → 倒计时≤3天文字变红并闪烁 → 逾期后记录逾期天数 → 读者个人页面顶部显示红色警告条

**活动报名流程**：管理员发布活动 → 读者浏览活动列表 → 点击"报名" → 报名人数+1 → 显示参与者头像 → 活动满员后按钮变灰 → 活动开始前24小时可取消报名

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖白#faf7f2（背景）、原木色#c7b89c（辅助）
- **强调色**：墨绿#2d6a4f、深蓝#4a90d9
- **按钮风格**：圆角8px，统一风格
- **字体**：标题使用 Noto Serif SC（衬线，文化感），正文使用 Noto Sans SC（无衬线，清晰）
- **布局风格**：卡片式布局，顶部导航栏，暖色调营造图书馆温馨氛围
- **图标风格**：线性图标，与暖色调搭配

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 导航栏 | 固定顶部，暖白背景，墨绿文字，移动端折叠为汉堡菜单 |
| 全局 | 页脚 | 原木色背景，白色文字 |
| 图书列表页 | 搜索栏 | 圆角输入框，搜索图标，深蓝强调 |
| 图书列表页 | 图书卡片 | 220×320px，圆角16px，1px #e8e0d4边框，悬停上移8px+加深阴影 |
| 我的借阅页 | 警告条 | 背景#ffccd5，文字#9b2226，尖锐边角 |
| 我的借阅页 | 倒计时 | 三色渐变（绿→橙→红），≤3天闪烁动画 |
| 活动看板页 | 活动卡片 | 圆角12px，背景#f8f7f4，4px彩色竖条（读书会#4a90d9/讲座#e76f51/亲子#2a9d8f） |
| 活动看板页 | 参与者头像 | 圆形40px，1px边框 |
| 统计仪表盘 | 统计卡片 | 180×100px，圆角矩形，左侧5px彩色竖条 |
| 统计仪表盘 | 趋势图 | Canvas折线图，蓝色折线#4a90d9，网格#f0f0f0，坐标轴#ddd |

### 4.3 响应式设计

- **桌面端**：导航栏完整展示，图书卡片多列网格，统计卡片横排
- **移动端**：导航栏折叠为汉堡菜单，卡片单列排列，统计卡片缩至45%宽度两行排列
- 触摸优化：按钮最小点击区域44px，卡片间距适当增大

### 4.4 动画效果

- **卡片悬停上移**：translateY(-8px) + 阴影加深，0.3s缓动
- **统计数字递增动画**：从0过渡到实际值，0.6秒，ease-out缓出
- **倒计时闪烁**：≤3天时文字闪烁动画，1s周期
- **页面切换**：framer-motion淡入淡出
