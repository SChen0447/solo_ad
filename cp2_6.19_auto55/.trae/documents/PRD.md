## 1. 产品概述

像素风模拟经营小岛游戏——玩家扮演岛主，通过放置和升级功能建筑（渔屋、旅馆、餐厅、灯塔、花圃）吸引游客、赚取金币，将荒岛发展为繁荣度假胜地。

- 核心玩法：建筑放置→吸引游客→赚取金币→升级建筑→良性循环
- 目标用户：休闲游戏玩家，喜欢像素风与经营类游戏

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：20×20网格地图、左侧建筑工具栏、右上角金币面板、右下角统计按钮
2. **建筑系统**：5种建筑拖拽放置、点击选中移动、右键升级（1-5级）
3. **游客系统**：码头自动生成游客、沿路径行走、到达建筑消费产生金币
4. **统计面板**：今日游客数、总收入、建筑产出环形图、最近10笔交易记录

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

| 游戏主界面 | 建筑工具栏 | 左侧垂直排列5种建筑卡片，可拖拽放置到网格 |
| 游戏主界面 | 金币面板 | 右上角显示总金币数，实时更新 |
| 游戏主界面 | 统计按钮 | 右下角按钮，点击展开侧边栏 |
| 统计侧边栏 | 游客与收入 | 今日游客数、总收入 |
| 统计侧边栏 | 环形图 | 各建筑产出占比，填充动画0→实际值耗时0.8秒 |

| 升级菜单 | 右键菜单 | 右键点击建筑弹出，显示当前等级和升级选项 |

## 3. 核心流程

1. 玩家从左侧工具栏拖拽建筑到网格上放置（1.2倍→1倍弹入动画+震动）
2. 已放置建筑可点击选中后拖拽调整位置（半透明跟随→松开滑入）
3. 右键建筑打开升级菜单，升级时显示进度条+粒子特效
4. 游客从码头自动生成，沿路径走向旅馆/餐厅
5. 游客到达建筑后头顶冒金币泡泡，金币实时更新到金库
6. 玩家点击统计按钮查看经营数据

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暖色系像素风——天蓝(#87CEEB)到浅黄(#FFFACD)渐变背景
- 建筑卡片：圆角矩形+2px描边，选中时亮白色外发光
- 字体：像素风格字体(Press Start 2P)
- 图标：5种建筑各有独特像素图标和颜色
- 按钮：圆角矩形，像素风格描边

### 4.2 建筑配色

| 建筑 | 颜色 | 图标描述 |
|------|------|----------|
| 渔屋 | 木棕(#8B4513) | 小木屋+渔网 |
| 旅馆 | 砖红(#CD5C5C) | 多层楼房 |
| 餐厅 | 暖橙(#FF8C00) | 餐盘+烟囱 |
| 灯塔 | 石白(#F5F5DC) | 高塔+光芒 |
| 花圃 | 草绿(#32CD32) | 花朵+围栏 |

### 4.3 动画规范

| 动画类型 | 参数 |
|----------|------|
| 建筑放置 | 1.2倍→1倍缩放，0.2秒，轻微震动 |
| 建筑拖拽 | 半透明(0.5)跟随鼠标，松开滑入 |
| 升级进度条 | 0%→100%填充，1秒 |
| 升级粒子 | 从建筑中心散开 |
| 游客行走 | 上下浮动动画 |
| 金币泡泡 | 0.5倍→1.1倍→1倍，0.3秒 |
| 环形图填充 | 0→实际值，0.8秒 |
| 交易记录 | 新记录从底部滑入 |

### 4.4 响应式设计

- 桌面端：网格每格32px，侧边栏固定右侧
- 移动端：网格每格24px，隐藏侧边栏，底部弹出菜单替代

### 4.5 音效

| 交互 | 音效 |
|------|------|
| 放置建筑 | 上升音调 |
| 升级建筑 | 欢快琶音 |
| 金币入账 | 短促叮当声 |

## 5. 性能要求

- 60FPS稳定运行
- 同时最多30个游客、50个建筑
- 帧率波动不超过5%
