## 1. 产品概述

智能家居能耗监测系统是一款面向家庭用户的能耗管理 Web 应用，帮助用户实时查看家中各电器的用电情况，自动计算耗电量并提供个性化节能建议。目标用户为希望降低家庭电费、了解用电习惯的普通家庭用户。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（本地使用） | 添加/编辑/删除设备、查看图表、接收通知、查看节能建议 |

### 2.2 功能模块

1. **首页仪表盘**：圆环图展示各设备耗电占比，折线图展示7天能耗趋势，通知卡片区域
2. **设备管理面板**：添加/编辑/删除家电设备，填写额定功率和日均使用时长
3. **节能贴士面板**：悬浮按钮触发，侧边滑入展示随机节能建议

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页仪表盘 | 圆环图 | 各设备耗电占比可视化，占比最大者高亮，不同色块区分设备 |
| 首页仪表盘 | 折线图 | 过去7天总能耗趋势，X轴日期，悬停显示精确数值，点击设备查看独立曲线 |
| 首页仪表盘 | 通知卡片 | 设备月耗电占比超30%时弹出非侵入式通知，2秒淡出 |
| 首页仪表盘 | 节能贴士按钮 | 悬浮按钮，点击后从右侧滑入贴士面板 |
| 设备管理面板 | 设备列表 | 卡片式展示所有设备，支持添加/编辑/删除 |
| 设备管理面板 | 设备表单 | 填写设备名称、额定功率(W)、日均使用时长(h) |
| 节能贴士面板 | 建议列表 | 5条随机节能建议，每条可标记"已读"后消失 |

## 3. 核心流程

用户打开应用后，首页仪表盘展示所有设备的能耗概览（圆环图+折线图）。用户可以通过设备管理面板添加新设备（输入名称、功率、日均使用时长），系统自动计算日耗电量和月耗电量。当某设备月耗电占比超过30%时，系统自动弹出通知卡片提醒。用户可点击悬浮按钮打开节能贴士面板，查看并标记建议。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：#1a1a2e（深色背景）
- 辅助色：#16213e（卡片/面板背景）
- 强调色：#0f3460（边框/次要元素）
- 文字颜色：#e0e0e0（浅灰色主体文字）
- 按钮/链接颜色：#e94560（醒目的红粉色）
- 布局风格：卡片式布局，每张卡片带微光晕阴影（box-shadow: 0 0 20px rgba(15,52,96,0.5)）和悬停上浮效果（transform: translateY(-4px), transition: transform 0.3s）
- 字体：标题使用 Outfit，正文使用 DM Sans
- 图标风格：简约线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页仪表盘 | 圆环图 | 深色卡片容器，中心显示总耗电量数值，外圈分色弧段，占比最大弧段高亮脉冲效果 |
| 首页仪表盘 | 折线图 | 深色卡片，渐变填充区域图，自定义tooltip，设备筛选按钮组 |
| 首页仪表盘 | 通知卡片 | 顶部固定区域，红色左边框卡片，淡入动画(0.3s)，2秒后淡出(0.5s) |
| 首页仪表盘 | 节能贴士按钮 | 右下角圆形悬浮按钮，脉冲光晕动画，#e94560色 |
| 设备管理面板 | 设备列表 | 响应式网格卡片（PC 3列/平板 2列/手机 1列），每张卡片显示设备名、功率、日/月耗电量，编辑/删除按钮 |
| 设备管理面板 | 设备表单 | 模态弹窗表单，深色半透明遮罩，输入框带聚焦发光效果 |
| 节能贴士面板 | 侧边面板 | 从右侧滑入（缓动动画 cubic-bezier(0.16,1,0.3,1)），5条建议卡片，已读按钮 |

### 4.3 响应式设计

- 桌面优先设计（Desktop-first）
- PC端（≥1024px）：设备列表3列网格，图表并排展示
- 平板端（768px-1023px）：设备列表2列网格，图表上下排列
- 手机端（<768px）：设备列表单列，图表全宽，面板全屏覆盖
- 触摸优化：按钮最小触摸区域44px，表单输入框适配移动端键盘

### 4.4 动画与交互

- 页面加载：卡片依次从底部淡入（staggered animation，每张延迟0.1s）
- 通知卡片：淡入(0.3s ease-out) → 停留2s → 淡出(0.5s ease-in)
- 节能面板：从右侧滑入，缓动曲线 cubic-bezier(0.16,1,0.3,1)，持续0.4s
- 卡片悬停：上浮4px + 光晕增强，transition 0.3s
- 图表交互：悬停显示tooltip，点击设备切换折线图数据
