## 1. 产品概述

幻灵抽卡模拟器是一款游戏化集卡应用，让用户通过消耗虚拟货币抽卡来收集不同稀有度、属性与主题的幻想生物卡片，完成图鉴收藏和进化合成。

- 目标用户：喜欢收集养成类游戏、抽卡模拟体验的休闲玩家
- 核心价值：提供沉浸式暗黑魔法风格的抽卡体验，融合概率机制、保底系统与进化合成策略玩法

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 抽卡、查看图鉴、进化合成 |

### 2.2 功能模块
1. **抽卡页面**：抽卡圆盘、单抽/十连抽、动画展示、保底计数器、货币显示
2. **图鉴页面**：收集进度、卡片网格、稀有度/属性筛选、分页懒加载
3. **详情页面**：卡片大图、属性雷达图、进化链图示、进化合成操作

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 抽卡页面 | 抽卡圆盘 | 中心雕刻风格金色镶边圆盘，点击后卡片飞出辐射排列 |
| 抽卡页面 | 单抽/十连抽按钮 | 单抽消耗10币，十连消耗90币，余额不足时灰显 |
| 抽卡页面 | 抽卡动画 | 持续2秒，卡片旋转翻转+金色闪光，SR蓝色粒子，SSR金色火焰拖尾 |
| 抽卡页面 | 保底计数器 | 显示剩余必出SR/SSR次数，每次抽卡后更新 |
| 抽卡页面 | 货币显示 | 魔法药剂瓶造型，数字变化时液面波动动画 |
| 图鉴页面 | 进度条 | 顶部显示总收集百分比与动态进度条 |
| 图鉴页面 | 卡片网格 | 六宫格展示，已收集亮色+微光动效，未收集灰色剪影 |
| 图鉴页面 | 筛选栏 | 按稀有度(R/SR/SSR)和属性(火水风地)筛选 |
| 图鉴页面 | 分页加载 | 初始30张，按需每次加载15张 |
| 详情页面 | 卡片大图 | 全屏浮层展示，半透明黑色遮罩渐入 |
| 详情页面 | 属性雷达图 | Canvas绘制五维属性雷达图 |
| 详情页面 | 进化链图示 | 显示当前卡的进化路径 |
| 详情页面 | 进化合成 | 三张同属性SR+50币合成一张同属性SSR，魔法阵旋转+粒子爆炸动画 |

## 3. 核心流程

用户进入应用后，初始拥有50幻灵币。通过抽卡页面消耗幻灵币进行单抽或十连抽，抽到的卡片自动存入图鉴。在图鉴页面浏览收集进度，点击卡片查看详情。在详情页可进行进化合成操作，将三张同属性SR卡合成为SSR卡。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深紫(#2D1B4E)与古金(#D4AF37)
- 按钮：悬停描边发光效果，点击下沉动画
- 字体：标题使用Cinzel(魔法雕刻感)，正文使用Noto Sans SC
- 布局：中心化抽卡圆盘，暗黑魔法主题
- 图标/装饰：魔法符文、星辰粒子、金色镶边

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 抽卡页面 | 背景层 | Canvas动态星空粒子，深紫渐变底色 |
| 抽卡页面 | 货币区(右上角) | 魔法药剂瓶造型，黄色液面动画 |
| 抽卡页面 | 抽卡圆盘(中心) | 金色镶边雕刻圆盘，脉冲光效 |
| 抽卡页面 | 抽卡按钮(圆盘下) | 古金色按钮，发光描边+下沉动画 |
| 抽卡页面 | 保底计数器(圆盘下) | 半透明卡片样式，SR/SSR分栏显示 |
| 抽卡页面 | 卡片展示区 | Canvas绘制卡片飞出+辐射排列动画 |
| 图鉴页面 | 进度条(顶部) | 古金色渐变进度条，百分比文字 |
| 图鉴页面 | 筛选栏 | 药丸形按钮组，选中态发光 |
| 图鉴页面 | 卡片网格 | 六宫格，悬停浮起1.1倍放大，微光动效 |
| 图鉴页面 | 未收集卡 | 灰色剪影+名称+概率提示 |
| 详情页面 | 遮罩层 | 半透明黑色渐入动画 |
| 详情页面 | 卡片大图 | 居中大图展示，属性色边框 |
| 详情页面 | 雷达图 | Canvas五维雷达图 |
| 详情页面 | 进化链 | 节点+箭头连线，可点击跳转 |
| 详情页面 | 合成按钮 | 满足条件时发光，不满足灰显 |

### 4.3 响应式设计
- 桌面端：多列网格，标准字号
- 移动端：自动调整为单列卡片流，字体和按钮尺寸增大保证触控
- 抽卡动画在移动端简化粒子数量保证性能

### 4.4 性能要求
- 抽卡动画期间FPS≥30，十连抽3秒内完成
- 图鉴懒加载切换无卡顿
- 首屏渲染时间≤1.5秒
- Canvas粒子数量根据设备性能动态调整
