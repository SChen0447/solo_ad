## 1. 产品概述

交互式环球旅行时间线可视化应用，为旅行博主和爱好者提供沉浸式的旅行足迹展示体验。用户上传包含经纬度、日期、照片和笔记的 CSV 文件，即可在地图上生成动态旅行路径，支持时间轴回放、图钉详情浏览和照片画廊展示。

- 目标用户：旅行博主、摄影爱好者、地理教育工作者
- 核心价值：将静态旅行数据转化为可交互、可回放的动态地理可视化内容

## 2. 核心功能

### 2.1 功能模块

1. **文件上传模块**：CSV 文件拖拽上传，格式校验与错误提示
2. **地图渲染模块**：OpenStreetMap 底图，路径线绘制，彩色图钉标注
3. **时间轴控制模块**：横向滑块时间轴，路径动画回放，播放/暂停控制
4. **详情弹窗模块**：图钉点击弹出照片、天气信息和旅行笔记
5. **照片画廊模块**：侧边栏轮播展示目的地多张照片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主页面 | 文件上传区 | 中央拖拽上传区域，支持 CSV 文件选择与校验，错误提示 |
| 主页面 | 地图容器 | 左侧 70% 宽度，展示旅行路径线和彩色图钉 |
| 主页面 | 右侧控制面板 | 30% 宽度，包含时间轴滑块、播放控制、当前位置信息 |
| 主页面 | 图钉弹窗 | 点击图钉显示照片、天气图标和笔记，支持笔记展开 |
| 主页面 | 照片画廊侧边栏 | 右侧 400px 宽度，轮播展示多张照片 |

## 3. 核心流程

用户上传 CSV 文件 → 系统解析并校验数据 → 地图自动生成按时间顺序的路径和图钉 → 用户拖动时间轴滑块或点击播放 → 地图上动画标记沿路径移动 → 用户点击图钉 → 弹出详情卡片并可打开照片画廊 → 浏览照片和旅行笔记

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：蓝紫渐变（#3f51b5 → #7c4dff）
- **强调色**：暖橙路径线（#ff7043），红色动画标记（#ff5252）
- **图钉渐变色**：从绿色（#4caf50）到红色（#f44336）按日期渐变
- **背景**：深色模式（#1e1e2e），毛玻璃效果面板
- **按钮风格**：圆角渐变按钮，点击下沉反馈（transform: scale(0.98)）
- **字体**：系统字体栈，正文 16px，标题 24px 加粗
- **动效**：淡入淡出 + 向上滑动过渡（0.3s），照片淡入加载动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 文件上传区 | 虚线边框，浅灰背景，悬停变蓝，错误红色文字提示 |
| 主页面 | 地图容器 | OpenStreetMap 底图，暖橙路径线，彩色图钉，红色发光动画点 |
| 主页面 | 时间轴 | 横向滑块，蓝紫渐变轨道，日期刻度，播放/暂停按钮 |
| 主页面 | 图钉弹窗 | 圆角卡片，照片缩略图，天气图标，笔记（100字截断+展开） |
| 主页面 | 照片画廊 | 右侧滑入面板，轮播箭头，底部指示器，水平滑动切换 |

### 4.3 响应式

- 桌面端：地图左侧 70%，右侧 30% 控制面板
- 移动端（<768px）：垂直堆叠，地图 60vh 高度，面板在下方
- 触摸优化：滑块和按钮增大点击区域

### 4.4 性能约束

- 地图初始化：4G 网络下 ≤ 2 秒
- 时间轴拖动动画：≥ 30fps
- 1MB CSV 解析+渲染：≤ 1 秒
- 照片懒加载：仅加载当前轮播图片，并发 ≤ 5 个
