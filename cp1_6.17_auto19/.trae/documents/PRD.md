## 1. 产品概述
跨语言代码注释自动生成与预览应用，支持JavaScript、Python、TypeScript、Java四种主流编程语言。用户可粘贴源代码，系统自动解析函数和类结构，生成符合语言规范的注释，并以差异对比形式展示，支持一键应用和撤销。

- 主要目的：提高代码文档化效率，减少手动编写注释的重复劳动
- 目标用户：开发者、技术文档工程师、代码审查人员
- 产品价值：将注释生成时间缩短80%以上，确保注释格式统一规范

## 2. 核心功能

### 2.1 功能模块
1. **代码编辑区**：多标签管理、语法高亮、行号显示、实时解析触发
2. **注释生成引擎**：语言自动识别、函数/类结构提取、参数类型推断、多格式注释生成
3. **差异对比面板**：行级diff算法、颜色高亮标记、点击联动滚动、改动细节气泡
4. **工具栏**：语言选择、注释风格切换（JSDoc/Sphinx/JavaDoc）、应用/撤销操作
5. **状态栏**：已添加注释行数、操作提示

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主页面 | 顶部工具栏 | 语言下拉选择、注释风格下拉、应用按钮、撤销按钮 |
| 主页面 | 标签页栏 | 多标签切换、关闭标签（带确认）、语言图标显示 |
| 主页面 | 左侧编辑区 | 代码粘贴与编辑、语法高亮、行号、当前行高亮 |
| 主页面 | 右侧差异面板 | 并排代码对比、差异行高亮、点击联动、细节气泡 |
| 主页面 | 底部状态栏 | 注释计数、操作反馈信息 |
| 主页面 | 可拖拽分隔条 | 左右/上下分栏切换、宽度/高度调节 |

## 3. 核心流程

用户粘贴代码 → 系统自动识别语言 → 解析函数和类结构 → 生成注释模板 → 计算代码差异 → 高亮显示对比 → 用户确认/勾选注释 → 点击应用嵌入代码 → 状态栏更新计数 → 支持Ctrl+Z撤销

```mermaid
flowchart LR
    A["用户粘贴代码"] --> B["语言自动识别"]
    B --> C["解析函数/类结构"]
    C --> D["提取参数与返回值"]
    D --> E["生成标准格式注释"]
    E --> F["计算行级差异"]
    F --> G["差异高亮展示"]
    G --> H["用户确认应用"]
    H --> I["注释嵌入代码"]
    I --> J["状态栏更新"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色背景 #1e1e2e，柔和蓝色主色 #89b4fa，白色文字
- **配色方案**：
  - 新增行背景：绿色渐变 #1e4a2e → #2d6a4f
  - 修改行背景：黄色渐变 #4a4a1e → #6a6a2d
  - 删除行背景：红色渐变 #4a1e1e → #6a2d2d
- **按钮样式**：圆角8px，悬停时蓝色发光效果（box-shadow: 0 0 12px rgba(137,180,250,0.4)）
- **字体**：Fira Code等宽字体用于代码，系统无衬线字体用于UI
- **布局**：左右分栏（左60%/右40%），窄屏时垂直堆叠
- **图标风格**：简洁线性图标，使用lucide-react库

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主页面 | 工具栏 | 下拉选择器、操作按钮、发光悬停效果 |
| 主页面 | 编辑区 | 等宽字体、行号栏、当前行浅色高亮条 |
| 主页面 | 差异面板 | 双栏并排、渐变差异背景、弹性滚动动画 |
| 主页面 | 分隔条 | col-resize光标、拖拽反馈 |
| 主页面 | 标签页 | 文件名称、语言图标、关闭按钮、激活状态高亮 |

### 4.3 响应式设计
- 桌面端（>800px）：左右分栏布局，可拖拽水平分隔条
- 移动端/窄屏（≤800px）：垂直堆叠布局，可拖拽垂直分隔条
- 触摸优化：增大点击区域，支持滑动操作
