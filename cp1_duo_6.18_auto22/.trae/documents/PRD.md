## 1. 产品概述

音轨工坊是一款基于 Web 的在线数字音频工作站（DAW）应用，旨在为用户提供轻量级、即时可用的多轨道音频编辑与实时音效合成工具。用户无需安装笨重的本地软件，即可在浏览器中完成简单的混音任务。

- **主要用途**：多轨音频创建、混音、实时音效处理与分享
- **目标用户**：音乐爱好者、播客制作者、内容创作者、教育学习者
- **产品价值**：零安装、即开即用、轻量化、跨平台

## 2. 核心功能

### 2.1 功能模块
1. **轨道控制面板**：轨道列表、添加音轨按钮、各轨道独立控件
2. **主时间线视图**：波形显示、播放头、时间轴刻度、选区操作
3. **音频处理引擎**：文件解码、播放控制、3段均衡器、混音导出
4. **主控区域**：总音量、播放/暂停/重置、时间显示
5. **导出功能**：WAV 文件导出、进度显示、自动下载

## 3. 核心流程

### 3.1 主要用户流程

用户打开应用 → 查看默认空轨道 → 点击"添加音轨"创建轨道 → 选择轨道加载音频文件 → 调节各轨道音量与均衡器 → 设置静音/独奏 → 播放预览混音效果 → 调整至满意后导出 WAV 文件

```mermaid

    A["打开应用"] --> B["创建/添加音轨"]
    B --> C["加载音频文件"]
    C --> D["调节音量/EQ"]
    D --> E["播放预览"]
    E --> F{"满意?}
    F -->|否| D
    F -->|是| G["导出WAV文件"]
    G --> H["自动下载"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **设计主题**：深色专业音频工作站风格
- **主背景色**：#121212
- **表面色**：#1e1e1e
- **文本色**：#e0e0e0
- **强调色**：亮青 #00acc1（主交互）、绿色 #4caf50（播放）、红色 #e53935（重置/警告）
- **轨道颜色**：暖橙 #ff7043、草绿 #66bb6a、天蓝 #42a5f5、薰衣草 #ab47bc、樱粉 #ec407a、柠檬黄 #ffee58、青碧 #26a69a、钢灰 #78909c
- **按钮风格**：圆角按钮，悬停过渡动画 0.2s ease-in-out
- **字体**：Poppins 400（标题）、JetBrains Mono（时间显示）
- **布局风格**：左右分栏，左栏轨道控制面板，右栏主时间线
- **动效**：所有按钮和滑块悬停过渡动画，点击反馈

### 4.2 页面设计概览

| 页面/模块 | UI 模块名称 | UI 元素 |
|-----------|------------|--------|
| 顶部工具栏 | 应用标题 | "音轨工坊，Poppins 18px，白色，居中 |
| 左侧轨道面板 | 轨道列表 | 垂直排列，每轨道高120px，选中态淡蓝背景 |

| 左侧轨道面板 | 添加按钮 | 底部添加音轨按钮 |

### 4.3 响应式设计
- **桌面端**（≥768px）：左右分栏布局，左栏宽280px
- **移动端**（<768px）：单列布局，轨道面板折叠为顶部水平条，汉堡菜单展开
- **触摸优化**：增大按钮点击区域，适配触摸手势

### 4.4 性能指标
- 4轨道同时播放延迟 ≤ 100ms
- 波形绘制帧率 ≥ 30fps
- WAV 导出时间 ≤ 音频时长的2倍
