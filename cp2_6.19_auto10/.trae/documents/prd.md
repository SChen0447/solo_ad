## 1. 产品概述

在线音乐播放器 Dashboard，集中管理个人音乐收藏，支持歌曲播放、专辑浏览和播放列表自定义。面向音乐爱好者，提供沉浸式深色主题的本地音乐管理体验。

## 2. 核心功能

### 2.1 功能模块

1. **音乐播放器主界面**：侧边栏导航 + 推荐轮播 + 专辑网格 + 播放控制栏

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主界面 | 侧边栏导航 | 展示「我的音乐」「专辑库」「播放列表」「最近播放」四个菜单项，点击切换右侧主内容区，切换时有淡入淡出动画，选中项高亮为主题色#1DB954 |
| 主界面 | 推荐歌单轮播图 | 3张卡片自动循环播放，每4秒切换，左右箭头+底部圆点指示器，横向平滑滑动效果 |
| 主界面 | 专辑网格 | 1440px以上三列，768px以下单列，卡片悬停上浮5px+外发光效果 |
| 主界面 | 播放列表 | 展示自定义播放列表内容 |
| 主界面 | 底部播放控制栏 | 固定底部，封面缩略图+歌名+艺术家+可拖拽进度条+播放/暂停/下一首按钮，播放按钮旋转光盘动画 |

## 3. 核心流程

用户打开页面 → 左侧导航菜单选择分类 → 主内容区淡入淡出切换展示对应内容 → 推荐轮播自动播放 → 点击歌曲 → 底部播放栏激活 → 拖拽进度条调节播放进度 → 播放按钮旋转动画

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景 #0f0f0f，卡片 #1a1a1a，主题色 #1DB954（Spotify绿）
- **按钮风格**：圆角胶囊按钮，主题色填充，悬停时亮度提升
- **字体**：标题使用粗体无衬线字体，正文使用常规字重
- **布局风格**：左侧固定侧边栏 + 右侧弹性主内容区 + 底部固定播放栏
- **图标风格**：线性图标，与Spotify风格一致

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主界面 | 侧边栏 | 深色背景#0f0f0f，菜单项hover高亮，选中项#1DB954左边框+文字变色，宽度240-280px响应式 |
| 主界面 | 轮播图 | 3张卡片横向排列，左右箭头按钮，底部圆点指示器，平滑滑动过渡 |
| 主界面 | 专辑网格 | 三列卡片布局，封面图+专辑名+艺术家，hover上浮5px+box-shadow外发光 |
| 主界面 | 播放控制栏 | 固定底部80px高度，左侧封面+歌名+艺术家，中间进度条，右侧控制按钮，播放按钮旋转光盘 |

### 4.3 响应式

- 桌面优先设计
- 1440px+：侧边栏280px，主内容区三列专辑网格
- 1024px-1439px：侧边栏240px，主内容区两列专辑网格
- 768px以下：侧边栏折叠为图标模式，主内容区单列堆叠布局
- 触控优化：进度条拖拽区域增大，按钮点击区域≥44px

### 4.4 动画性能

- 轮播动画使用 CSS transform + will-change 实现60fps
- 进度条更新使用 requestAnimationFrame
- 淡入淡出使用 CSS transition opacity
- 悬停上浮使用 CSS transform translateY + box-shadow transition
