## 1. 产品概述

在线厨艺学习互动平台，让用户浏览专业烹饪课程、提交自己的作品照片，并通过AI驱动的风味评分系统获取六维口味评分与改进建议。目标用户为烹饪爱好者和家庭厨师，核心价值在于将课程学习与实践反馈形成闭环。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览课程、提交作品、查看评分 |

### 2.2 功能模块
1. **课程目录页**：搜索筛选、课程卡片网格、难度标签
2. **课程详情页**：课程步骤说明、作品提交区、AI评分结果展示
3. **作品展示区**：瀑布流排行、作品放大查看

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 课程目录页 | 搜索筛选栏 | 320px搜索框+难度下拉，300ms防抖过滤 |
| 课程目录页 | 课程卡片网格 | 两列自适应网格，封面280x180、标题、星级、难度标签 |
| 课程详情页 | 课程内容区 | 左侧大图480x320+步骤列表，右侧提交区 |
| 课程详情页 | 作品提交区 | 拖拽上传图片+200字符文本输入+提交按钮(0.8s防重复) |
| 课程详情页 | AI评分弹窗 | 六维雷达图+改进建议列表，0.3s缩放动画 |
| 课程详情页 | 作品瀑布流 | 三列瀑布流，评分降序，新作品滑入动画，点击放大 |

## 3. 核心流程

用户进入首页浏览课程 → 搜索/筛选找到目标课程 → 点击进入详情页 → 查看步骤说明 → 拖拽上传作品照片+描述文字 → 提交至评分API → 接收AI六维评分与建议 → 查看雷达图与改进建议 → 浏览他人作品排行

    "提交评分请求" --> "AI返回六维评分"

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：天蓝色 #3b82f6，辅助色：#10b981（初级）、#f59e0b（中级）、#ef4444（高级）
- 背景：#f8fafc 浅灰白，卡片白色，文字 #1e293b 深色 + #64748b 次要
- 按钮风格：圆角胶囊形（border-radius: 20px），品牌色填充
- 字体：标题18px/600，正文14px/400，用户名12px灰色
- 布局风格：顶部固定毛玻璃导航栏 + 居中1200px内容区 + 卡片网格

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 课程目录页 | 导航栏 | 64px高，毛玻璃效果，应用图标+首页按钮 |
| 课程目录页 | 搜索筛选栏 | 320px宽搜索框圆角20px + 难度下拉 |
| 课程目录页 | 课程卡片 | 280x180封面圆角16px，悬停上浮8px，难度标签右上角 |
| 课程详情页 | 课程大图 | 480x320圆角16px，步骤说明列表 |
| 课程详情页 | 提交区域 | 虚线拖拽区2px圆角12px + 文本框100%宽80px高 + 提交按钮120x40 |
| 课程详情页 | 评分弹窗 | 420px宽圆角24px，雷达图0.8透明度渐变填充，建议列表6px灰点 |
| 课程详情页 | 作品卡片 | 等比裁剪100%x160px圆角8px，评分滚动动画0.6s |

### 4.3 响应式设计
- 桌面优先，移动端(<768px)单列布局，卡片100%宽
- 导航栏移动端缩至56px高
- 课程详情页左右布局变为上下堆叠
- 作品瀑布流变为单列

### 4.4 动效规范
- 卡片悬停：上浮8px，阴影0 2px 8px→0 8px 24px，0.2s ease
- 按钮按压：缩放0.95
- 弹窗打开：0.3s ease-out缩放动画
- 新作品：从底部滑入0.4s ease-out
- 评分数字：0滚动至实际评分0.6s ease-out
- 拖拽激活：边框变#3b82f6 + 进度条
- 输入框focus：0.2s box-shadow过渡
