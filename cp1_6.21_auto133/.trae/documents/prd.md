## 1. 产品概述

独立音乐人作品管理与粉丝反馈平台，为个人独立运营的音乐人提供轻量级的在线展示、粉丝互动与播放数据追踪工具，解决现有平台过于复杂不适合个人运营的痛点。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 音乐人 | 预设账号登录（无需注册） | 创建/编辑/删除专辑和歌曲、查看播放统计 |
| 粉丝（访客） | 无需登录 | 浏览作品、播放音乐、留言评价、点赞喜爱 |

### 2.2 功能模块

1. **登录页**：固定用户名登录入口
2. **仪表盘页**：专辑管理、播放统计可视化、热门排行
3. **播放页**：音乐播放、歌曲详情、粉丝留言、点赞互动
4. **底部播放栏**：全局固定播放控制

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 登录表单 | 固定用户名密码登录，登录后跳转仪表盘 |
| 仪表盘 | 音乐人信息栏 | 显示头像（圆形裁剪）、名称、总播放数 |
| 仪表盘 | 专辑列表 | 左栏展示专辑卡片（封面缩略图+名称+歌曲数），悬停上浮3px+阴影 |
| 仪表盘 | 创建专辑模态框 | 输入专辑名、发行日期、上传封面（2MB限制，jpg/png），带缩略图预览 |
| 仪表盘 | 歌曲列表 | 展开专辑后显示歌曲名、时长、播放数、喜爱数，支持编辑和删除 |
| 仪表盘 | 播放趋势图 | 过去7天每日播放数折线图，蓝紫渐变线条，点状虚线网格，tooltip |
| 仪表盘 | 热门排行 | Top5横向条形图，紫粉渐变条形 |
| 播放页 | 歌曲详情 | 封面、标题、艺术家、播放按钮 |
| 播放页 | 留言区 | 昵称+留言内容输入，时间倒序排列，首字母头像，300px固定高度滚动 |
| 播放页 | 爱心按钮 | 空心↔实心粉红切换，数字+1/-1，0.3秒弹跳动画 |
| 底部播放栏 | 播放控制 | 封面小图、曲名、进度条（可拖动）、音量滑块（紫蓝渐变轨道），毛玻璃背景 |

## 3. 核心流程

1. 音乐人使用预设账号登录，进入仪表盘管理专辑和歌曲
2. 音乐人创建专辑（上传封面），添加歌曲（上传音频）
3. 粉丝访问播放页，浏览歌曲列表，点击播放
4. 粉丝在播放页留言和点赞
5. 音乐人在仪表盘查看播放统计和粉丝互动数据

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暗蓝（#0f172a ~ #1e293b），强调色：粉色（#ec4899）、紫色（#8b5cf6）
- 按钮风格：圆角8px，半透明背景，悬停发光效果
- 字体：标题使用 Outfit，正文使用 DM Sans
- 布局风格：侧边栏+主内容区，卡片式布局
- 图标风格：Lucide线性图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 居中卡片，深色半透明背景，粉色焦点边框光晕 |
| 仪表盘 | 信息栏 | 圆形头像，名称，总播放数统计 |
| 仪表盘 | 专辑列表 | 两栏布局，封面缩略图+文字，悬停上浮动画 |
| 仪表盘 | 创建模态框 | 半透明遮罩，缩放动画(0.8→1)，封面预览 |
| 仪表盘 | 折线图 | 深色半透明卡片，蓝紫渐变线，点状虚线网格，tooltip |
| 仪表盘 | 条形图 | 紫粉渐变条形，宽度按比例 |
| 播放页 | 播放区 | 大封面，播放按钮，歌曲信息 |
| 播放页 | 留言区 | 浅灰半透明卡片，300px固定高度，滚动条，首字母头像 |
| 播放页 | 爱心按钮 | 空心→实心粉红，弹跳动画 |
| 底部播放栏 | 控制栏 | 固定底部，毛玻璃背景，粉紫渐变进度条，紫蓝渐变音量滑块 |

### 4.3 响应式设计

- 桌面端：内容区最大宽度1200px，两侧留白
- 平板端：内容区缩至90%宽度
- 手机端：全宽布局，仪表盘单栏显示

### 4.4 动画与过渡

- 页面切换：淡入淡出0.3秒
- 模态框：背景遮罩+缩放动画(0.8→1，缓出)
- 专辑卡片：悬停上浮3px+阴影
- 爱心按钮：0.3秒弹跳动画
- 输入框焦点：粉色光晕边框
