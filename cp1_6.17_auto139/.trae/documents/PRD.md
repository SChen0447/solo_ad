## 1. 产品概述

响应式布局原型工具是一款专为前端开发者设计的可视化布局搭建工具，帮助开发者快速搭建和预览响应式页面布局原型，并支持导出完整页面快照。

- **核心价值**：解决设计阶段反复手动调整CSS布局、频繁截图和比对效率低下的问题
- **目标用户**：前端开发者、UI设计师、产品经理
- **核心能力**：拖拽式布局搭建、多断点实时预览、属性可视化编辑、一键导出HTML和截图

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 前端开发者 | 无需注册 | 使用所有布局搭建、编辑和导出功能 |

### 2.2 功能模块

1. **组件面板**：左侧可拖拽布局组件库，包含导航栏、轮播图、卡片网格、两栏/三栏布局、页脚等常用布局块
2. **画布区域**：中间可视化画布，支持拖拽放置、断点切换、网格对齐
3. **属性面板**：右侧属性编辑面板，支持组件样式的可视化调整
4. **工具栏**：顶部操作栏，提供撤销、重做、清空、导出HTML、导出截图功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作台 | 组件面板 | 展示可拖拽的布局组件列表，支持拖拽到画布，hover时显示组件名称 |
| 主工作台 | 画布区域 | 模拟网页视口，支持断点切换（375px/768px/1024px/1440px），显示网格背景，接收拖拽组件 |
| 主工作台 | 属性面板 | 展示选中组件的所有可编辑属性，修改实时同步到画布 |
| 主工作台 | 工具栏 | 撤销(Ctrl+Z)、重做(Ctrl+Shift+Z)、清空画布、导出HTML、导出截图 |

## 3. 核心流程

### 用户操作流程

1. 用户从左侧组件面板拖拽布局组件到中间画布区域
2. 拖拽过程中显示蓝色虚线位置指示线，组件边缘自动吸附
3. 松手后组件放置到画布，播放0.3秒淡入动画
4. 用户点击画布中的组件，右侧属性面板显示该组件的可编辑属性
5. 用户调整属性面板中的参数，画布组件实时更新
6. 用户可通过顶部断点按钮切换不同视口尺寸，画布平滑过渡
7. 完成布局后，用户可导出HTML文件或PNG截图

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色主题，主背景色 `#1e1e2e`，组件面板背景 `#2a2a3e`，画布背景 `#f5f5f5`
- **色彩方案**：
  - 主色：深蓝 `#4a9eff`（用于选中状态、指示线）
  - 背景：深灰 `#1e1e2e`、中灰 `#2a2a3e`、浅灰 `#f5f5f5`
  - 文字：白色 `#ffffff`、次文字 `#a0a0b0`
- **按钮风格**：圆角8px，阴影 `0 4px 12px rgba(0,0,0,0.15)`，hover时亮度提升20%，点击时弹性缩放 `scale 0.95 -> 1.0`（0.1秒）
- **字体**：Noto Sans SC，无衬线字体
- **布局风格**：左右三栏结构，左侧240px固定，右侧320px固定，中间自适应
- **图标**：使用 lucide-react 图标库，统一线性风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主工作台 | 组件面板 | 深色背景卡片，图标+名称网格布局，拖拽时半透明效果 |
| 主工作台 | 画布区域 | 浅灰色网格背景（5px间隔#e0e0e0细线），宽度平滑过渡动画0.4秒 |
| 主工作台 | 属性面板 | 深色卡片，分组表单，滑块控件，颜色选择器 |
| 主工作台 | 工具栏 | 水平排列按钮组，快捷键提示，二次确认弹窗 |

### 4.3 响应式设计

- **桌面优先**，针对不同屏幕尺寸优化
- 当浏览器宽度小于1024px时，左侧面板自动折叠为图标侧边栏（仅显示图标，hover展开文字标签）
- 触控设备优化：增大点击区域，支持触控拖拽

### 4.4 动画与交互

- 组件放置：0.3秒淡入动画 `opacity 0 -> 1`
- 画布缩放：0.4秒平滑过渡 `width: ease-in-out`
- 按钮交互：hover亮度+20%，点击弹性缩放0.1秒
- 拖拽指示：蓝色虚线 `#4a9eff dashed 2px`
- 属性修改：实时响应，重绘时间≤30ms

### 4.5 性能指标

- 拖拽组件到放置完成总耗时 ≤ 80ms
- 属性修改后画布重绘响应 ≤ 30ms
- 导出截图处理时间 ≤ 1秒（≤20个组件）
