## 1. 产品概述

像素冒险世界是一款2D像素风格沙盒游戏，玩家可以在随机生成的像素地图上自由移动、采集资源、合成物品和战斗。解决传统沙盒游戏中建造与战斗割裂、缺乏深度互动的问题，让采集-合成-战斗形成一个完整的游戏循环。

- 目标用户：喜欢像素风格沙盒游戏的休闲玩家
- 核心价值：在轻量级2D像素世界中体验采集→合成→战斗的完整循环

## 2. 核心功能

### 2.2 功能模块

1. **游戏主界面**: 像素地图渲染、角色控制、UI覆盖层
2. **背包系统**: 物品管理、工具切换、物品展示
3. **采集系统**: 资源采集动画与粒子效果
4. **战斗系统**: 敌人AI、攻击动画、掉落机制
5. **合成系统**: 资源合成、配方管理、合成动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | 地图渲染 | 16x16网格地图，草地/树木/石头/水域四种地形，独特像素纹理和碰撞逻辑 |
| 游戏主界面 | 角色控制 | WASD 8方向移动，4帧行走动画，水域减速50% |
| 游戏主界面 | 资源条 | 右上角显示木材/石头/铁锭/金币数量，数值变化缩放动画0.1秒 |
| 游戏主界面 | 状态栏 | 左下角血量条(红色)和等级条(蓝色)，平滑过渡动画0.3秒 |
| 背包面板 | 物品列表 | 从左侧滑入0.3秒，背景模糊，物品像素图标+数量，选中放大光晕 |
| 背包面板 | 工具切换 | 数字键1-4快速切换，角色手持物品图标更新 |
| 采集交互 | 资源采集 | 感叹号图标0.5秒，粒子飞散0.3秒，+1弹出动画 |
| 战斗交互 | 敌人战斗 | 史莱姆弹性动画1Hz，剑挥弧线0.2秒，闪烁红色缩小消失0.3秒 |
| 战斗交互 | 掉落系统 | 金币随机散落在敌人死亡位置周围 |
| 合成台 | 资源合成 | 旋转动画+光效1秒，"合成成功"底部弹入2秒淡出 |

## 3. 核心流程

玩家进入游戏后，在随机生成的16x16像素地图上自由移动。通过WASD控制角色移动，遇到树木和石头可以切换到锤子工具进行采集，采集获得的资源进入背包。采集到足够资源后，前往地图右上角的合成台，可以合成木栅栏(3木头)或铁剑(2石头+1铁锭)。地图上会随机出现史莱姆敌人，玩家切换到剑工具后点击敌人进行攻击，击败敌人掉落金币。战斗和采集获得的经验累积后角色升级。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：大地色系 - 棕色(#8B6914)、绿色(#4A7C2E)、蓝色(#2E5E8B)
- 辅助色：深棕(#2C1810)背景，浅米色(#E8D5B0)文字
- 按钮风格：圆角矩形(圆角8px)，像素风格边框
- 字体：像素字体(Press Start 2P / 像素宋体)
- 布局风格：游戏画布全屏，UI元素覆盖在画布上方
- 图标风格：16x16像素图标，手工像素绘制

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 游戏主界面 | 地图区域 | 全屏Canvas，像素纹理地形块，暗角效果 |
| 游戏主界面 | 资源条(右上) | 4个资源图标+数值，半透明暗色背景，缩放动画 |
| 游戏主界面 | 血量条(左下) | 红色条状，宽度随血量变化，平滑过渡0.3秒 |
| 游戏主界面 | 等级条(左下) | 蓝色条状，经验累积填充，升级闪白光0.2秒 |
| 游戏主界面 | 状态栏(底部) | 半透明0.6，暗色渐变背景 |
| 背包面板 | 物品列表 | 左侧滑入面板，像素图标网格，选中光晕效果 |
| 合成台面板 | 合成界面 | 弹性缩放动画(0.8→1.0, 0.3秒)，旋转光效，底部弹入提示 |
| 移动端 | 虚拟控制 | 虚拟摇杆+操作按钮，UI放大1.5倍 |

### 4.3 响应式适配

- 桌面优先设计，键盘WASD+鼠标点击操作
- 手机屏幕(<768px)时UI元素放大1.5倍
- 移动端操作改为虚拟摇杆和按钮
- 性能自适应：帧率<25FPS时关闭粒子效果和行走动画细节
