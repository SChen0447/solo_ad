## 1. 产品概述

极光回响是一个3D交互式音乐可视化Web应用，用户上传音频文件后，系统实时解析频率和振幅数据，驱动5000+粒子在3D空间中生成动态视觉场景——低频鼓点触发地面脉冲波纹，中频旋律驱动粒子色彩与轨迹变幻，高频人声点亮光带闪烁。目标用户为音乐爱好者、VJ表演者和数字艺术创作者。

## 2. 核心功能

### 2.1 功能模块

1. **音频上传与预览页**：上传区域、波形Canvas预览、播放控制
2. **3D可视化场景**：粒子系统、渲染模式切换、音频驱动的动态效果
3. **参数控制面板**：实时滑块调整、渐变模式切换、渲染模式切换

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 音频上传区域 | 拖拽或点击上传MP3/WAV文件，上传后显示波形预览 |
| 主页面 | 波形预览 | Canvas绘制音频波形，播放时波形随进度滚动并高亮已播放部分 |
| 主页面 | 3D渲染容器 | 全屏Three.js渲染画布，粒子系统实时响应音频数据 |
| 主页面 | 播放控制 | 播放/暂停按钮，点击后3D场景开始响应音频 |
| 主页面 | 参数控制面板 | 右侧半透明毛玻璃面板，包含所有可视化参数调整 |

## 3. 核心流程

用户上传音频 → 系统解码并显示波形预览 → 用户点击播放 → Web Audio API实时分析频谱 → 频率数据映射到粒子参数 → Three.js渲染动态3D场景 → 用户通过控制面板实时调整参数 → 视觉效果即时响应

```mermaid
flowchart TD
    "A[用户上传音频]" --> "B[解码音频 & 绘制波形预览]"
    "B" --> "C[用户点击播放]"
    "C" --> "D[Web Audio API 分析频谱]"
    "D" --> "E[频率分段映射]"
    "E --> F[低频 → 地面脉冲波纹]"
    "E --> G[中频 → 粒子色彩与轨迹]"
    "E --> H[高频 → 光带闪烁]"
    "F & G & H" --> "I[Three.js 渲染 3D 场景]"
    "I" --> "J[用户调整控制面板参数]"
    "J" --> "I"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色空间主题，径向渐变背景 #0A0A1A → #000000
- 强调色：亮紫 #7B2FFF（极光核心色）、橙红 #FF6B35（高频热色）
- 按钮风格：圆角胶囊按钮，半透明玻璃质感，悬停时放大1.05倍+浅色光晕
- 字体：标题使用 Orbitron（科技感），正文使用 Inter
- 布局：全屏3D场景 + 右侧浮动控制面板，移动端面板折叠为底部抽屉
- 图标：简洁线性图标，与极光主题呼应

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 音频上传区域 | 虚线边框拖拽区，居中显示，上传后替换为波形Canvas |
| 主页面 | 波形预览 | Canvas全宽，深色底色，已播放部分用紫色渐变高亮 |
| 主页面 | 3D渲染容器 | 全屏Canvas，径向渐变背景，粒子系统覆盖 |
| 主页面 | 控制面板 | 右侧固定，毛玻璃背景(模糊10px)，边框微光#FFFFFF22 |
| 主页面 | 播放控制 | 底部居中浮动栏，播放/暂停按钮+进度条+音量 |

### 4.3 响应式设计

- 桌面端：全屏3D场景 + 右侧固定控制面板
- 平板端：3D场景 + 可折叠侧边面板
- 移动端：3D场景占满 + 底部抽屉式控制面板（上滑展开）

### 4.4 3D场景指引

- 环境：深空黑背景，径向渐变从中心微亮向四周暗沉
- 灯光：中心点光源(紫色) + 环绕点光源(橙红)，随音频脉冲变化强度
- 相机：透视相机，默认45度俯角观察中心球体，支持鼠标拖拽旋转
- 构图：粒子围绕中心球体分布，低频粒子靠近中心，高频粒子扩散至外围
- 交互：鼠标拖拽旋转场景，滚轮缩放，粒子系统持续动画
- 后处理：可选辉光(Bloom)效果增强极光视觉
- 性能预算：5000+粒子@60fps，使用BufferGeometry+Points渲染

## 5. 性能要求

- 渲染帧率：稳定60fps，无卡顿掉帧
- 粒子数量：不少于5000个，支持用户实时调节
- 音频分析延迟：< 16ms（单帧内完成频谱计算到视觉映射）
- 参数调整响应：即时生效，视觉过渡平滑无跳帧
- 缓动过渡：颜色渐变0.8秒缓动，模式切换1.2秒淡入淡出
