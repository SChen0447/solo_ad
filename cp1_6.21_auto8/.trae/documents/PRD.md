## 1. 产品概述
FlowForge 是一个基于订阅的创意工作流自动化平台，帮助用户通过可视化管道编辑器将重复的日常任务（文件格式转换、图片批量处理、定时通知等）组合成自动化流程，大幅提升工作效率。

- **核心价值**：零代码可视化编排、实时执行监控、多触发方式、高性能实时通信
- **目标用户**：创意工作者、运营人员、开发者、需要自动化重复性工作的团队

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 创建和管理管道、执行管道、查看日志、配置触发器 |

### 2.2 功能模块
1. **管道编辑器页面**：可视化画布、节点组件面板、节点配置抽屉、执行控制栏
2. **执行日志页面**：运行记录列表、筛选器、日志详情、画布路径高亮
3. **管道管理页面**：管道列表、创建/编辑/删除管道、触发器配置

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 管道编辑器 | 节点组件面板 | 拖拽节点到画布、分类展示节点类型、可折叠面板 |
| 管道编辑器 | 画布区域 | 无限滚动缩放、节点拖拽、连线编辑、状态可视化 |
| 管道编辑器 | 节点配置抽屉 | 参数表单配置、实时生效、验证提示 |
| 管道编辑器 | 执行控制栏 | 运行/停止按钮、执行进度显示 |
| 执行日志 | 日志列表 | 运行记录展示、多维度筛选 |
| 执行日志 | 日志详情 | 节点输入输出、耗时统计、错误信息 |
| 管道管理 | 管道列表 | 管道卡片展示、状态指示、快速操作 |
| 管道管理 | 触发器配置 | Cron表达式配置、文件监听触发 |

## 3. 核心流程

用户从左侧组件面板拖拽节点到画布 → 连接节点形成数据流管道 → 点击节点打开配置面板设置参数 → 配置定时触发器或手动执行 → 管道执行时数据按拓扑顺序流动 → 实时通过WebSocket推送执行状态 → 节点边框颜色变化表示运行状态 → 执行完成后生成日志记录 → 用户可在日志页面查看详情并高亮画布路径。

```mermaid

    A["拖拽节点到画布"] --> B["连接节点形成管道"]
    B --> C["配置节点参数"]
    C --> D["设置触发方式"]
    D --> E["执行管道"]
    E --> F["WebSocket推送状态"]
    F --> G["画布节点状态更新"]
    G --> H["生成执行日志"]
    H --> I["日志查看与路径高亮"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：蓝灰色系 (#64748B) 搭配薄荷绿 (#34D399)
- **背景**：清爽浅色主题，近白色背景 (#F8FAFC)
- **按钮风格**：圆角矩形 (8px)、薄荷绿主按钮、蓝灰次要按钮
- **字体**：标题使用 Inter SemiBold，正文使用 Inter Regular
- **布局**：三栏布局（左组件面板 + 中间画布 + 右配置抽屉）
- **图标**：使用 lucide-react 线性图标，风格统一简洁

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 管道编辑器 | 画布区域 | 圆角节点卡片、贝塞尔曲线连线、发光选中效果、无限网格背景 |
| 管道编辑器 | 组件面板 | 分类折叠、节点图标预览、拖拽反馈动画 |
| 管道编辑器 | 配置抽屉 | 平滑滑入动画、表单分组、参数说明提示 |
| 执行日志 | 日志列表 | 状态标签（成功绿色/失败红色）、时间轴样式、筛选标签栏 |
| 执行日志 | 日志详情 | 节点路径时间线、输入输出代码块、耗时统计卡片 |

### 4.3 响应式
- **桌面端 (≥1024px)**：三栏布局，左面板固定280px，右抽屉固定360px
- **平板端 (768px-1023px)**：组件面板自动变为底部标签栏，配置抽屉变为底部弹出层
- **触摸优化**：节点拖拽区域≥44x44px，按钮最小尺寸40x40px，支持双指缩放画布

### 4.4 交互动效
- 节点拖拽：平滑跟随效果 (150ms cubic-bezier(0.4, 0, 0.2, 1))
- 画布缩放：带过渡动画的视口变换
- 节点选中：发光外发光效果 (box-shadow: 0 0 0 3px rgba(52, 211, 153, 0.3))
- 抽屉滑出：300ms ease-out 平移动画
- 连线：贝塞尔曲线，带箭头方向指示
