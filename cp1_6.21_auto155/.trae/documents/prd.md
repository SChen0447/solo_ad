## 1. 产品概述

宠物托管服务预约与评价系统，为城市社区宠物主人提供集中化的临时看护和遛狗服务对接平台。解决微信群零散信息无法追踪、口头约定缺少评价机制的问题，帮助宠物主人快速找到经过认证和评价的可靠看护者。

- 目标用户：有临时托管/遛狗需求的宠物主人，及愿意提供看护服务的社区看护者
- 市场价值：构建宠物服务信用体系，通过评价机制保障服务质量，降低宠物主人寻找看护者的信任成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 宠物主人 | 注册账号 | 管理宠物档案、发布任务、评价看护者 |
| 看护者 | 注册账号后申请认证 | 浏览任务、接单、查看个人评价 |

### 2.2 功能模块

1. **首页**：看护者排行榜TOP5、本周热门任务列表、快速导航
2. **任务大厅**：任务列表展示、任务发布、看护者接单
3. **我的档案**：宠物档案卡片管理（添加/编辑）、个人信息
4. **评价系统**：星级评分弹窗、文字评价提交、看护者评分统计
5. **看护者详情页**：历史评价列表、评分统计、接单记录

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 看护者排行榜 | 展示评分前5名看护者卡片，排名数字带奖牌色，显示评分/接单数/最近评价摘要 |
| 首页 | 热门任务列表 | 展示最近发布的待接单任务，显示宠物信息/时间/地点/报酬 |
| 任务大厅 | 任务列表 | 所有待接单任务以卡片展示，报酬渐变绿色高亮，悬停上移3px加深阴影 |
| 任务大厅 | 发布任务 | 选择宠物、填写起止时间/地点/时长/报酬，发布后显示在列表 |
| 任务大厅 | 接单操作 | 看护者点击接单按钮，任务状态变"已接单"，按钮灰色不可用，显示看护者信息 |
| 我的档案 | 宠物档案卡片 | 卡片展示宠物信息，底色随品种类别变化（狗暖黄/猫浅粉/其他浅紫） |
| 我的档案 | 添加/编辑宠物 | 表单：名称、品种（12种预设）、年龄、性格多选、疫苗单选 |
| 评价弹窗 | 星级评分 | 1-5星点击评分，悬停金色脉冲放大，填充动画0.2秒 |
| 评价弹窗 | 文字评价 | 200字以内文字输入，提交后看护者统计实时更新 |
| 看护者详情 | 评价列表 | 历史评价按时间倒序，显示星级/文字/宠物名/服务时间 |

## 3. 核心流程

用户注册登录 → 创建宠物档案 → 发布托管/遛狗任务 → 看护者浏览任务并接单 → 服务完成 → 主人为看护者评分评价 → 系统更新看护者统计数据 → 排行榜实时刷新

```mermaid
    "看护者浏览任务大厅" --> "看护者接单"
    "看护者接单" --> "服务进行中"
    "服务进行中" --> "主人提交评价"
    "主人提交评价" --> "系统更新评分统计"
    "系统更新评分统计" --> "排行榜刷新"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：温暖米色(#FFF8F0)到淡粉(#FFE8E8)渐变背景
- 辅助色：柔和绿色(#5CB85C)作为报酬高亮、按钮主色
- 卡片：白色背景 + 2px圆角边框(#E8DDD0) + 轻微阴影
- 按钮：悬停0.2秒背景色渐变过渡，点击缩放0.95复原0.15秒
- 字体：标题使用 Playfair Display，正文使用 Noto Sans SC
- 图标：lucide-react图标库，狗类骨头图标、猫类鱼骨图标
- 宠物卡片品种底色：狗类暖黄(#FFF3CD)、猫类浅粉(#FFE0E6)、其他浅紫(#E8D5F5)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 看护者排行榜 | 排名圆形数字(金/银/铜/蓝色背景)、姓名、评分(颜色渐变)、接单数、评价摘要 |
| 首页 | 热门任务 | 任务卡片列表、宠物缩略信息、时间范围、地点、报酬(渐变绿色) |
| 任务大厅 | 任务列表 | 悬停上移3px卡片、接单按钮、灰色不可用状态、看护者缩略信息 |
| 我的档案 | 宠物卡片 | 品种类别底色图标区域、编辑模式切换、性格标签彩色胶囊 |
| 评价弹窗 | 星级评分 | 金色星星、悬停脉冲放大、0.2秒填充动画、文字输入框 |

### 4.3 响应式设计

- 桌面优先设计，移动端（<768px）自动切换单列布局
- 任务卡片和排行榜卡片宽度撑满容器
- 导航栏变为汉堡菜单
- 宠物档案卡片单列堆叠

### 4.4 骨架屏

- 所有列表页面数据加载时展示灰色闪烁骨架屏
- 持续到API请求返回数据
- 使用 `animate-pulse` Tailwind类实现
