## 1. 产品概述

在线代码片段版本对比与高亮预览工具——一个纯前端的代码差异可视化对比应用，帮助开发者快速查阅和对比不同版本的代码片段，通过逐行差异高亮和直观的交互反馈，大幅提升代码审查和版本追踪的效率。

- 目标用户：日常开发中需要快速对比代码版本差异的前端/全栈开发者
- 核心价值：零配置、纯浏览器端运行，提供专业级的差异可视化和一键复制能力

## 2. 核心功能

### 2.1 功能模块

1. **对比页面**：双栏代码输入编辑器、并排差异高亮视图、工具栏、帮助弹窗

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 对比页面 | 工具栏 | 语言选择下拉器、复制左版/右版/合并版按钮 |
| 对比页面 | 左侧代码编辑区 | 等宽字体代码文本框，支持粘贴/输入，monospace背景#1e1e1e |
| 对比页面 | 右侧代码编辑区 | 与左侧对称，实时同步滚动（延迟≤100ms） |
| 对比页面 | 可拖拽分隔线 | 4px垂直分隔线，支持拖拽调整左右宽度 |
| 对比页面 | 差异高亮面板 | 并排显示对比结果，新增行绿色、删除行红色、修改行黄色，行号旁显示增减标记 |
| 对比页面 | 浮动气泡框 | 点击高亮行弹出差异说明，2秒后自动消失 |
| 对比页面 | 复制提示条 | 固定底部中央，复制成功后显示绿色提示，1.5秒后消失 |
| 对比页面 | 帮助弹窗 | 右下角圆形按钮触发，居中弹窗，半透明遮罩 |

## 3. 核心流程

用户在左侧粘贴/输入旧版代码，右侧粘贴/输入新版代码 → 系统自动防抖300ms后触发diff对比 → 使用diff库逐行计算差异 → 生成结构化DiffLine数组 → SideBySideView组件渲染左右并排高亮视图 → 用户可点击高亮行查看差异说明、通过工具栏一键复制各版本代码。

```mermaid

    "A[用户输入左侧代码]" --> "C[防抖300ms]"
    "B[用户输入右侧代码]" --> "C"
    "C" --> "D[diffEngine.compare计算差异]"
    "D" --> "E[生成DiffLine数组]"
    "E" --> "F[SideBySideView渲染高亮视图]"
    "F" --> "G[用户交互：点击高亮行/复制/切换语言]"
    "G" --> "H[气泡框/提示条/语法高亮更新]"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主题色：#007acc（VS Code风格蓝）
- 整体风格：深色程序员主题，背景 #1e1e1e，JetBrains Mono等宽字体
- 按钮：主题色渐变背景（#007acc → #005c99），圆角4px，白色文字
- 布局：顶部固定工具栏 + 左右并排代码面板（可拖拽分隔线调整比例）
- 动画：悬停缩放、按钮点击缩小、淡入淡出过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 对比页面 | 工具栏 | 高48px，背景#252526，圆角8px，底部1px #333边框，轻微阴影 |
| 对比页面 | 代码编辑区 | monospace字体，14px，行高1.6，背景#1e1e1e，文字#d4d4d4 |
| 对比页面 | 行号区域 | 宽40px，背景#333，文字#aaa |
| 对比页面 | 高亮行 | 新增#d4fcbc+绿色+，删除#fbb+红色-，修改#ffeaa7+橙色~ |
| 对比页面 | 分隔线 | 4px宽，悬停变#007acc+col-resize光标，拖拽过渡0.2s |

### 4.3 响应式适配

- 桌面端（≥768px）：左右并排布局，工具栏按钮单行排列
- 平板端（<768px）：上下堆叠布局，每区域100%宽度，工具栏按钮两行排列
- 手机端（<480px）：按钮仅显示图标（←→↔），增加底部安全边距16px

### 4.4 语法高亮

- 支持语言：JavaScript、TypeScript、HTML、CSS
- 关键字：蓝色 #569cd6
- 字符串：橙色 #ce9178
- 注释：绿色 #6a9955
- 数字：浅紫色 #b5cea8
