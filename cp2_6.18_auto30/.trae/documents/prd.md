## 1. 产品概述

CSS渐变动画调色板工作台——一款面向设计师和前端开发者的交互式渐变色方案创作工具，解决在浏览器中无法直观预览多种颜色组合动态效果、难以在预设方案间快速对比和保存偏好的问题。

- 目标用户：UI设计师、前端开发者、视觉创意人员
- 核心价值：实时预览渐变动画效果，快速对比多种配色方案，一键导出生产级CSS代码

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 直接使用，无需注册 | 创建/编辑/删除渐变方案，保存收藏，导出CSS代码 |

### 2.2 功能模块

1. **调色板编辑区**：颜色停止点管理（添加/删除/拖拽排序）、颜色选择器、渐变方向旋转控件、线性/径向渐变模式切换
2. **渐变预览区**：实时渲染CSS渐变背景、requestAnimationFrame驱动动画循环、播放/暂停/停止控制
3. **动画控制面板**：动画时长滑块、延迟滑块、缓动函数下拉选择
4. **预设模板选择**：6个内置渐变方案模板卡片
5. **收藏夹管理**：localStorage持久化、网格卡片展示、编辑/删除操作
6. **CSS代码导出**：highlight.js代码高亮、一键复制

### 2.3 页面详情

| 页面/区域 | 模块名称 | 功能描述 |
|-----------|----------|----------|
| 主编辑区 | 调色板编辑器 | 支持至少4个颜色停止点的添加/删除/拖拽排序，原生颜色选择器，实时预览 |
| 主编辑区 | 渐变方向控件 | 可旋转圆形控件（半径60px），中心箭头指示方向，0-360度角度调整 |
| 预览区 | 渐变预览 | 600×400px预览区域，线性/径向渐变切换，实时渲染 |
| 预览区 | 动画控制 | 播放/暂停按钮（图标切换0.3s过渡），停止按钮 |
| 右侧面板 | 动画参数 | 时长1-10s滑块、延迟0-5s滑块、缓动函数下拉 |
| 模板区 | 预设模板 | 6个模板卡片（160px宽，圆角12px，悬停放大1.05倍0.15s） |
| 收藏夹区 | 方案收藏 | 网格布局（每行4个，200×120px卡片，圆角16px），编辑/删除 |
| 模态窗口 | 代码导出 | 600×400px白色模态框，highlight.js monokai主题，一键复制 |

## 3. 核心流程

1. **创建渐变方案**：用户在调色板添加颜色停止点 → 选择颜色 → 调整角度 → 选择渐变类型 → 实时预览
2. **添加动画**：设置时长/延迟/缓动函数 → 点击播放 → 预览区循环动画
3. **使用模板**：点击预设模板卡片 → 自动加载颜色和默认动画参数
4. **保存方案**：点击保存 → 方案存入localStorage收藏夹 → 卡片展示
5. **导出代码**：点击导出按钮 → 模态窗口显示CSS代码 → 一键复制

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，主背景#1e1e2e，卡片背景#2a2a3e，文字主色#e0e0f0
- **强调色**：蓝色#3b82f6（按钮、聚焦边框），悬停#2563eb
- **输入控件**：统一圆角8px、背景#3a3a4e、边框#4a4a5e，聚焦边框#3b82f6（0.2s transition）
- **字体**：现代无衬线字体，白色方案名16px带1px文字阴影
- **布局**：flex水平布局，间隙20px，固定1280px居中，最小900px等比缩放
- **过渡效果**：所有按钮和交互元素hover 0.2s-0.3s ease-out，不可拖拽时cursor: not-allowed

### 4.2 页面设计概览

| 页面/区域 | 模块名称 | UI元素 |
|-----------|----------|--------|
| 主编辑区 | 调色板编辑器 | 颜色停止点列表（可拖拽排序），input type color，角度旋转圆盘（半径60px） |
| 主编辑区 | 渐变预览 | 600×400px预览框，线性/径向切换按钮，播放/暂停/停止按钮 |
| 右侧面板 | 动画参数 | range滑块（时长/延迟），select下拉（缓动函数） |
| 模板区 | 预设模板 | 6张卡片（160px宽，圆角12px，渐变背景），hover放大1.05倍0.15s |
| 收藏夹区 | 方案收藏 | 网格卡片（200×120px，圆角16px），hover显示编辑/删除按钮 |
| 模态窗口 | 代码导出 | 600×400px白色框，圆角20px，0.3s fadeIn，highlight.js monokai |

### 4.3 响应式设计

- 桌面优先设计，固定1280px宽度居中
- 最小支持900px宽度，元素等比例缩放
- 所有交互元素保持可点击和可操作
- 颜色选择器和滑块在小屏幕上保持可用
