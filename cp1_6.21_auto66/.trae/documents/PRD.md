## 1. 产品概述

为小型社区和爱好者团体（陶艺、木工、烘焙等手工工作坊）设计的线上报名与材料包配送管理平台，解决报名渠道分散、材料包库存难以预估、课后反馈收集困难三大核心痛点。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 主办方 | 直接访问管理后台 | 创建/管理工作坊、发送反馈问卷、查看报名数据 |
| 参与者 | 浏览即可报名 | 浏览工作坊、报名购买、查看历史记录、提交反馈 |

### 2.2 功能模块

1. **首页（工作坊列表）**：卡片网格展示所有工作坊，含剩余名额进度环、悬停动画、点击进入详情
2. **工作坊详情页**：展示工作坊完整信息、材料包选项、报名入口
3. **报名流程页**：三步分步表单（参加人数 → 材料包选择 → 参与者信息）
4. **主办方管理页**：创建/编辑工作坊、上传封面图、一键发送反馈问卷
5. **参与者历史页**：时间线展示已参加工作坊记录，含作品照片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 工作坊卡片网格 | 展示所有工作坊卡片，卡片含封面图、名称、费用、进度环；鼠标悬停旋转3度并显示日期时间覆盖层 |
| 首页 | 进度环组件 | 左上角显示剩余名额：绿色（充足>50%）、黄色（紧张10%-50%）、红色（已满0%） |
| 工作坊详情 | 基本信息 | 名称、封面图、简介、日期时间、最大人数、费用 |
| 工作坊详情 | 材料包列表 | 材料包名称、价格、库存数量、是否必选 |
| 工作坊详情 | 报名按钮 | 跳转到报名流程页 |
| 报名流程 | 步骤1-参加人数 | 选择1-5人参加 |
| 报名流程 | 步骤2-材料包选择 | 选择材料包及数量，实时更新库存 |
| 报名流程 | 步骤3-参与者信息 | 填写每位参与者姓名和联系方式 |
| 报名流程 | 确认码 | 提交成功后生成报名确认码 |
| 主办方管理 | 工作坊表单 | 创建/编辑工作坊，含封面上传 |
| 主办方管理 | 反馈发送 | 一键发送反馈问卷链接给所有参与者 |
| 参与者历史 | 时间线 | 按时间展示已参加工作坊，含名称、日期、材料包、作品照片 |

## 3. 核心流程

### 报名流程
用户浏览工作坊列表 → 点击进入详情页 → 点击报名 → 选择参加人数(1-5人) → 选择材料包及数量(实时库存校验) → 填写参与者信息 → 提交 → 系统扣减库存并生成确认码

```mermaid
flowchart TD
    "浏览工作坊列表" --> "点击工作坊卡片"
    "点击工作坊卡片" --> "查看详情页"
    "查看详情页" --> "点击报名"
    "点击报名" --> "步骤1:选择参加人数"
    "步骤1:选择参加人数" --> "步骤2:选择材料包"
    "步骤2:选择材料包" --> "步骤3:填写参与者信息"
    "步骤3:填写参与者信息" --> "提交报名"
    "提交报名" --> "校验库存"
    "校验库存" -->|"库存充足"| "扣减库存+生成确认码"
    "校验库存" -->|"库存不足"| "提示库存不足"
    "扣减库存+生成确认码" --> "报名成功"
```

### 主办方管理流程
主办方登录 → 创建工作坊(含材料包) → 参与者报名 → 工作坊结束 → 一键发送反馈问卷 → 收集反馈

```mermaid
flowchart TD
    "主办方创建工作坊" --> "设置基本信息+材料包"
    "设置基本信息+材料包" --> "发布工作坊"
    "发布工作坊" --> "参与者浏览报名"
    "参与者浏览报名" --> "工作坊结束"
    "工作坊结束" --> "一键发送反馈问卷"
    "一键发送反馈问卷" --> "参与者提交反馈"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：陶土色 #A0522D，米白色 #FFF8F0 背景
- **按钮样式**：渐变琥珀色按钮，圆角12px
- **字体**：正文使用衬线体增添手工质感，标题使用圆润的无衬线体
- **布局风格**：卡片网格布局，响应式自适应
- **背景**：细微纸纹理背景，整体温暖手工风
- **图标风格**：手绘线条风格图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 工作坊卡片 | 陶土色边框、圆角12px、封面图、进度环、悬停旋转3度+日期覆盖层、底部向上滑入动画 |
| 首页 | 进度环 | SVG环形进度条、绿色/黄色/红色状态、左上角定位 |
| 报名流程 | 步骤指示器 | 三步进度条、当前步骤高亮、琥珀渐变色 |
| 报名流程 | 输入框 | 聚焦时底部发光线条动画、米白色背景 |
| 详情页 | 信息区 | 封面大图、陶土色标题、简介文本、材料包列表 |
| 管理页 | 表单 | 工作坊创建表单、封面上传区、材料包动态添加 |

### 4.3 响应式设计

- 桌面端：卡片3列网格，详情页双栏布局
- 平板端：卡片2列网格，详情页单栏布局
- 移动端：卡片1列，报名步骤全屏展示，触摸优化按钮尺寸≥44px

### 4.4 动画规范

- 卡片进入：从底部向上滑动 + 透明度0到1，缓入缓出
- 卡片悬停：顺时针旋转3度 + 日期时间覆盖层淡入
- 步骤切换：200ms内完成，左滑出右滑入
- 输入框聚焦：底部2px发光线条从中心向两侧扩展
- 进度环：加载时从0动画到目标值
