## 1. 产品概述

在线课程智能笔记生成与复习文档导出应用——帮助学生在录播课程学习中，通过语音转文字和关键词提取自动生成结构化笔记，支持手动标记重点、添加注释，并导出为PDF或Markdown复习文档，显著提升课后复习效率。

- 目标用户：在线教育平台的学生用户，需要高效整理课程内容并进行结构化复习
- 核心价值：将被动观看视频转变为主动结构化学习，减少手动记笔记的时间，提升知识留存率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 上传课程文件、生成笔记、编辑笔记、导出文档、使用复习模式 |

### 2.2 功能模块

1. **笔记列表页(/)**：瀑布流卡片展示所有笔记，支持上传新课程文件，显示笔记标题/日期/概要
2. **笔记编辑页(/notes/:id)**：Markdown编辑器+实时预览，高亮重点段落，添加自定义标签，进入复习模式
3. **导出页面(/notes/:id 导出弹窗)**：选择导出格式(PDF/Markdown)，生成并下载复习文档

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 笔记列表页 | 上传区域 | 支持上传MP4/MP3文件(最大50MB)，拖拽或点击上传，上传后自动调用后端转录接口 |
| 笔记列表页 | 瀑布流笔记卡片 | 每张卡片显示标题、创建日期、概要，点击进入编辑页，悬停上移动效 |
| 笔记列表页 | 转录进度条 | 实时显示转录进度(高度8px，圆角4px，填充色#e94560) |
| 笔记编辑页 | Markdown编辑器 | 左侧编辑区(宽45%)，支持Markdown语法，行号显示，文字#e0e0e0 |
| 笔记编辑页 | 实时预览区 | 右侧预览(宽45%)，标题蓝色#4da6ff，滚动同步 |
| 笔记编辑页 | 重点高亮标注 | 自动提取3-5个重点段落，高亮色#e94560，用户可手动调整范围 |
| 笔记编辑页 | 自定义标签 | 添加自定义标签，标签气泡直径24px，背景#0f3460，文字白色 |
| 笔记编辑页 | 时间戳文本块 | 转录完成后显示带时间戳的文本块，点击可跳转播放 |
| 笔记编辑页 | 复习模式 | 全屏暗色调(背景#0a0a15)，只显示高亮段落和标签，字号20px，行高1.8 |
| 导出弹窗 | 格式选择 | 选择PDF或Markdown格式，点击确认调用exportApi |
| 导出弹窗 | 下载按钮 | 导出完成后显示下载按钮(渐变背景，圆角22px) |

## 3. 核心流程

用户上传课程视频/音频 → 后端语音转录为带时间戳文本 → 关键词提取与重点标注 → 自动生成结构化笔记 → 用户编辑/标记重点/添加标签 → 选择导出格式 → 生成并下载复习文档

```mermaid
flowchart TD
    "A[用户上传MP4/MP3文件]" --> "B[后端语音转录]"
    "B" --> "C[关键词提取与重点标注]"
    "C" --> "D[生成结构化笔记]"
    "D" --> "E[用户编辑笔记]"
    "E" --> "F[手动调整高亮/添加标签]"
    "F" --> "G{选择操作}"
    "G" --> "H[导出PDF/Markdown]"
    "G" --> "I[进入复习模式]"
    "I" --> "J[浏览高亮重点段落]"
    "J" --> "K[点击时间戳播放音视频]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主背景色**：#1a1a2e（深蓝黑色）
- **辅助背景色**：#16213e（深蓝色）
- **强调色**：#e94560（玫红色）
- **高亮色**：#0f3460（中蓝色）
- **文字主色**：#e0e0e0（浅灰白）
- **辅助文字**：#888888（灰色）
- **字体**：Noto Sans SC（全局字体）
- **按钮风格**：圆角渐变按钮，悬停亮度提升10%，0.2-0.3秒缓动过渡
- **布局风格**：左侧导航栏(64px) + 右侧内容区域
- **图标风格**：白色线性图标(lucide-react)，悬停缩放1.1倍+发光阴影

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|

| 笔记列表页 | 上传区域 | 虚线边框拖拽区，强调色提示文字 |

### 4.3 响应式适配

- 桌面优先设计，768px以下屏幕时：
  - 导航栏折叠为底部标签栏
  - 笔记编辑页编辑器和预览区改为上下排列
  - 卡片宽度自适应屏幕
