## 1. 产品概述

在线协作白板应用——支持多用户实时绘图、便签添加与画布缩放平移的协作工具。目标用户为需要远程协作的团队和个人，核心价值在于低延迟的实时同步体验和流畅的交互动画。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接访问 | 绘图、便签操作、查看在线用户 |

### 2.2 功能模块

1. **白板页面**：画布绘图、便签管理、实时协作、撤销重做

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 白板页面 | 画布区域 | SVG绘图层+便签DOM层，支持鼠标绘制二次贝塞尔曲线，缩放平移，网格背景 |
| 白板页面 | 工具栏 | 画笔颜色/粗细选择，便签颜色选择与添加，撤销重做，清空画布 |
| 白板页面 | 顶部栏 | 应用名称，在线用户数量，连接状态指示 |
| 白板页面 | 便签组件 | 显示/编辑文本，拖拽移动，删除动画，颜色切换动画 |

## 3. 核心流程

用户打开页面 → WebSocket自动连接 → 收到画布状态快照 → 在画布上绘图或添加便签 → 操作通过WebSocket实时同步 → 新用户加入时获取完整状态快照

```mermaid
flowchart TD
    "A[用户打开页面]" --> "B[WebSocket连接]"
    "B" --> "C[接收状态快照]"
    "C" --> "D[渲染画布+便签]"
    "D" --> "E{选择操作}"
    "E" --> "F[绘图：鼠标绘制贝塞尔曲线]"
    "E" --> "G[便签：添加/编辑/拖拽/删除]"
    "F" --> "H[发送操作到服务器]"
    "G" --> "H"
    "H" --> "I[服务器广播给其他客户端]"
    "I" --> "J[其他客户端实时更新画布]"
    "D" --> "K[撤销/重做操作]"
    "K" --> "D"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：蓝白灰（蓝色#3B82F6为强调色，白色#FFFFFF为基底，灰色#f0f0f0为画布背景）
- 按钮风格：Material Design圆形图标按钮，选中时蓝色背景高亮
- 字体：系统默认无衬线字体，标题14px，正文12px
- 布局：左侧固定60px工具栏 + 顶部窄条 + 主体画布区域
- 图标风格：Material Design风格，圆形图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 白板页面 | 画布区域 | 浅灰#f0f0f0背景，5px间距浅蓝网格线，SVG绘图层，DOM便签层 |
| 白板页面 | 工具栏 | 白色背景60px宽，浅阴影，圆形图标按钮，选中蓝色高亮，悬停缩放1.05倍 |
| 白板页面 | 顶部栏 | 蓝色窄条，白色文字，在线用户数，绿色/红色连接状态圆点 |
| 白板页面 | 便签 | 120x80px圆角矩形，信封图标，可拖拽，半透明阴影+上浮动画 |

### 4.3 响应式适配

- 桌面端：左侧60px工具栏 + 主体画布
- 移动端（<768px）：工具栏变为底部横条，画布区域自适应
- 画布区域随窗口大小自动缩放

### 4.4 动画规范

- 按钮悬停：缩放1.05倍，0.15s过渡
- 按钮点击：轻微按压效果
- 便签拖拽：半透明阴影+轻微上浮
- 便签释放：平滑落下动画
- 便签删除：0.3秒缩小消失
- 便签颜色切换：0.3秒渐变动画
- 撤销重做：0.2秒淡入淡出
- 连接断线：红色圆点每1秒闪烁
