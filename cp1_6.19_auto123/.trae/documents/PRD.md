# 城市通勤碳排放追踪器 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
随着全球气候变化问题日益严重，个人碳足迹追踪成为提升环保意识的重要手段。本项目旨在帮助用户记录日常通勤的碳排放，通过可视化图表和社区排行榜激励用户选择低碳出行方式。

### 1.2 目标用户
- 关注环保的城市通勤者
- 希望了解个人碳足迹的用户
- 愿意与朋友比较减排成果的社交型用户

### 1.3 核心价值
- 便捷记录每次通勤的出行方式和里程
- 自动计算并可视化展示碳排放数据
- 通过排行榜激励用户选择低碳出行
- 帮助用户建立环保出行习惯

## 2. 功能需求

### 2.1 通勤记录表单
- **交通工具选择**：支持步行、自行车、电动车、公交车、地铁、私家车、拼车
- **里程输入**：公里数输入，支持小数
- **实时计算**：根据预设排放因子自动计算本次碳排放量
- **动态效果**：提交后卡片从底部弹性飞入记录列表，数值滚动动画

### 2.2 月度碳排圆环图
- **中心数据**：当月总碳排放 vs 目标值（默认200kg CO2）
- **分色展示**：各交通工具碳排放占比
- **交互功能**：点击分段高亮并显示具体数值
- **状态动画**：超标红色脉冲，达标绿色呼吸动画

### 2.3 7天趋势折线图
- **X轴**：最近7天日期
- **Y轴**：每日碳排量
- **可视化**：柱状点 + 平滑曲线连接，带缓动动画
- **悬停提示**：当日各交通方式细分数据 + 平均线

### 2.4 好友排行榜
- **数据展示**：好友当月碳排、环比减排百分比
- **视觉标识**：绿色上箭头（减排）/红色下箭头（增排）
- **详情面板**：点击卡片展开周度碳排条形图 + 减排小贴士

### 2.5 性能要求
- 页面滚动帧率 ≥ 50fps
- 图表渲染响应时间 ≤ 100ms

## 3. 设计规范

### 3.1 色彩系统
- **主色调**：深绿色 (#1B5E20) - 代表环保、自然
- **强调色**：暖橙色 (#FF8A65) - 用于交互元素和数据高亮
- **中性色**：白色、浅灰、深灰 - 用于背景和文字
- **状态色**：绿色（达标）、红色（超标）

### 3.2 字体系统
- **主字体**：无衬线字体（系统无衬线栈）
- **层级**：清晰的标题、副标题、正文、辅助文字层级

### 3.3 组件规范
- **卡片**：圆角12px，微阴影，悬停上移3px + 阴影加深
- **输入框**：聚焦时墨绿色渐变边框发光效果
- **动画**：弹性缓动函数，平滑过渡效果

### 3.4 响应式布局
- **桌面端**：左侧图表区域 + 右侧表单区域
- **移动端**：垂直排列，图表在上，表单位置下方

## 4. 技术栈
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **图表库**：Chart.js + react-chartjs-2
- **ID生成**：uuid
- **样式**：CSS3 + CSS变量
