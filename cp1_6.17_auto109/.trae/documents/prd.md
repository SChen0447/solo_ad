## 1. 产品概述
在线作业批改与反馈系统，旨在解决传统纸质作业批改流程繁琐、反馈不及时、缺乏在线协作批阅能力的问题，为教师和学生提供高效的课程作业管理平台。
- 目标用户：高校/中学教师与学生
- 核心价值：实现作业全流程数字化——从创建、提交、批阅到反馈，提升教学效率与互动体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 教师 | 账号密码登录 | 创建课程与作业、批阅作业、发布成绩与评语 |
| 学生 | 扫码/链接直接进入 | 查看作业、提交作答、查看批阅反馈 |

### 2.2 功能模块
1. **教师工作台**：课程导航、作业列表管理、作业创建（多题型+截止日期+总分）、二维码/短链接生成
2. **学生提交页**：倒计时、文本作答（字数统计）、文件上传（拖拽+缩略图）、选择题、提交确认动画
3. **批阅界面**：卡片式待批列表、左右分栏批阅、Canvas矩形区域高亮标注、三色标记（红/黄/绿）、批注拖拽调整
4. **反馈详情页**：批注标注查看、分数显示、雷达图（各题型分数占比）、实时通知

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

教师创建课程和作业后，系统生成二维码与短链接。学生通过扫码或点击链接进入提交页面，在倒计时内完成作答并提交。教师进入批阅页面查看待批改作业，选中内容区域进行高亮标注和批注，完成后输入分数与评语。学生端收到实时通知，查看反馈详情和雷达图。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：浅蓝灰(#E8ECF1) + 白色(#FFFFFF)
- 强调色：柔和珊瑚红(#E8655A)，用于按钮、高亮标记、未读通知图标
- 辅助色：浅灰(#F5F6F8)用于背景，深灰(#3A3F47)用于文字
- 按钮风格：圆角8px，珊瑚红填充，点击涟漪波纹效果
- 字体：标题用 Noto Sans SC Bold，正文用 Noto Sans SC Regular
- 布局：教师端三栏式，学生端顶部导航+中部内容
- 卡片：白色背景，圆角12px，悬停上移4px+阴影加深
- 图标：使用 lucide-react 图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 教师工作台 | 三栏布局 | 左栏240px课程导航(浅蓝灰背景)、中栏flex作业卡片列表、右栏360px详情面板 |
| 教师工作台 | 作业卡片 | 白色圆角12px卡片，悬停translateY(-4px) 0.25s，阴影4px→8px |
| 作业创建 | 创建表单 | 输入框聚焦边框珊瑚红+底部展开下划线动画0.2s，提交按钮涟漪效果+圆环旋转加载 |
| 学生提交页 | 倒计时 | 顶部居中大号数字，珊瑚红强调 |
| 学生提交页 | 文本输入 | 输入框底部字数统计，低于最低字数时珊瑚红提示 |
| 学生提交页 | 文件上传 | 虚线边框拖拽区，缩略图列表展示，显示文件大小和格式 |
| 学生提交页 | 提交确认 | 全屏对勾动画，0.5s后跳转 |
| 批阅界面 | 左右分栏 | 左60%内容区+右40%批注面板，移动端上下分栏 |
| 批阅界面 | 高亮标记 | 红色(#E8655A)/黄色(#F5C842)/绿色(#4CAF50)三色半透明高亮 |
| 批阅界面 | 批注框 | 白色圆角卡片，可拖拽调整位置，阴影效果 |
| 学生反馈页 | 通知提示 | 顶部固定提示条，淡入动画0.3s |
| 学生反馈页 | 雷达图 | Recharts雷达图，珊瑚红填充，展示各题型分数占比 |

### 4.3 响应式设计
- 桌面优先设计，移动端自适应
- 屏幕宽度<768px时：
  - 教师端三栏变为单栏垂直堆叠
  - 批阅界面左右分栏变为上下分栏
  - 作业卡片在手机上以轮播形式展示
- 触摸优化：按钮最小点击区域44px，拖拽操作支持触摸

### 4.4 性能要求
- 批注文本实时同步延迟≤200ms
- 雷达图渲染时间≤100ms
