## 1. 产品概述

代码依赖关系可视化工具是一款面向前端开发者的分析工具，通过上传项目ZIP包自动解析代码文件间的导入依赖关系，并以交互式力导向图直观展示。帮助开发者快速理解项目架构、识别循环依赖、优化代码组织。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|----------|
| 开发者 | 无需注册 | 上传ZIP、解析依赖、查看力导向图、筛选节点 |

### 2.2 功能模块

1. **上传页面**: ZIP文件拖拽上传、文件列表展示、解析触发
2. **力导向图页面**: SVG力导向图渲染、节点拖拽、缩放平移、筛选面板、搜索功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 上传页面 | 上传区域 | 拖拽或点击上传.zip文件，实时显示文件列表（文件名、大小、修改时间），列表行淡入动画 |
| 上传页面 | 解析触发 | 点击解析按钮读取ZIP内容，解析.js/.jsx/.ts/.tsx文件的import语句 |
| 力导向图页面 | 力导向图渲染 | D3力导向布局，节点半径随文件行数20-60px动态调整，按文件类型着色 |
| 力导向图页面 | 节点交互 | 悬停显示信息卡片（文件名、行数、导入数、被导入数），点击高亮一级邻居 |
| 力导向图页面 | 视图控制 | 滚轮缩放(0.1-3倍)、平移、拖拽节点、重置布局按钮 |
| 力导向图页面 | 筛选面板 | 按文件类型筛选、模糊搜索匹配（发光动画）、移动端折叠为工具栏 |

## 3. 核心流程

用户上传ZIP包 → 系统解析所有代码文件的import语句 → 生成节点与边数据 → 切换到力导向图页面 → D3力模拟渲染SVG图表 → 用户交互（筛选、搜索、拖拽、缩放）

```mermaid
    D --> E["生成节点和边数据"]
    E --> F["力导向图渲染"]
    F --> G["用户交互操作"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深灰蓝 #1e1e2e（背景）、#2a2a3e（卡片）、#e0e0e0（文字）
- **节点配色**: 组件蓝色 #4a9eff、工具函数绿色 #4ade80、配置文件橙色 #f97316、样式文件紫色 #a855f7
- **边渐变**: 浅灰 #b0b0b0 到深灰 #505050
- **交互色**: 拖拽高亮 #6c6cf0、上传边框 #4a4a5e
- **字体**: 使用 JetBrains Mono 等宽字体作为代码展示，Inter 作为界面字体
- **按钮风格**: 圆角8px，悬停背景微亮，0.3s过渡动画
- **布局**: 左侧筛选面板 + 右侧主图区域，卡片式设计，微弱网格背景增强深度感
- **图标**: 使用 lucide-react 线性图标风格

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 上传页面 | 上传区域 | 虚线边框上传框，拖拽时边框变色 + 半透明蓝色覆盖层 |
| 上传页面 | 文件列表 | 卡片式列表，每行0.2s淡入动画，显示文件名/大小/修改时间 |
| 力导向图页面 | SVG画布 | 暗网格背景，节点拖拽抓取手势，0.3s CSS过渡 |
| 力导向图页面 | 筛选面板 | 类型复选框、搜索输入框、重置按钮，<768px折叠为顶部工具栏 |
| 力导向图页面 | 信息卡片 | 悬停节点时显示，包含文件详情，指针跟随 |

### 4.3 响应式

- 桌面端（≥768px）: 左侧固定宽度筛选面板 + 右侧自适应图表面板
- 移动端（<768px）: 筛选面板折叠为顶部工具栏图标按钮，点击展开
- 触控优化: 增大节点触摸区域，支持双指缩放

### 4.4 动画效果

- 页面加载: 元素错落淡入，animation-delay 阶梯式
- 节点悬停: 缩放1.1倍 + 阴影增强
- 搜索匹配: 外圈呼吸发光动画（box-shadow + opacity 循环）
- 拖拽节点: cursor: grab/grabbing，释放后1秒内力导向重新收敛
- 列表项: 0.2s opacity 淡入动画
