## 1. 产品概述

社区手工艺工作坊课程报名与材料包管理系统——为社区和手工艺社团提供一站式课程发布、在线报名、材料包自动生成与统计看板功能，解决传统接龙填表导致的名额错乱、材料统计不准的问题。

- 目标用户：社区管理员（发布课程、管理报名、查看统计）和学员（浏览课程、在线报名）
- 核心价值：集中化管理课程与报名流程，自动生成材料需求清单，实时掌握报名人数和材料准备进度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设账号 | 创建/管理课程、管理报名、查看统计看板、导出材料清单 |
| 学员 | 无需注册 | 浏览课程、在线报名 |

### 2.2 功能模块

1. **首页（课程列表）**：课程卡片网格展示、按日期排序、报名入口
2. **课程详情页**：课程信息展示、在线报名表单、已报名学员列表、材料包清单
3. **管理员看板页**：统计卡片、材料需求总览、CSV导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 课程卡片网格 | 展示所有课程卡片，按日期从近到远排列；卡片顶部封面图占位圆角矩形，中部课程名称（24px粗体）和日期时间（14px灰色），底部剩余名额（绿色数字）和报名按钮；悬停上浮3px加深阴影；点击报名按钮弹出圆形进度动画（0.6秒）后跳转报名表单 |
| 课程详情页 | 课程信息展示 | 展示课程名称、日期时间、时长、费用、简介、封面图、名额信息 |
| 课程详情页 | 在线报名表单 | 姓名输入、手机号输入、邮箱输入、技能等级选择（初级/中级/高级圆形按钮，选中填充对应颜色），提交后自动增加报名记录 |
| 课程详情页 | 已报名学员列表 | 列表形式展示已报名学员（姓名首字母圆形头像+姓名+等级标签）；管理员视图每行右侧显示取消报名按钮，点击弹出确认弹窗（模糊半透明背景、红色渐变警告按钮），取消时列表项左滑消失动画（0.3秒），名额加回 |
| 课程详情页 | 材料包清单 | 预设材料内容（名称、预估用量、单位），一键生成总材料需求清单（用量×报名人数），表格展示（深色表头、交替浅灰白行、悬停加深），CSV导出按钮 |
| 管理员看板页 | 统计卡片 | 三个并排卡片：总课程数（淡蓝色渐变背景）、总报名人次（淡绿色渐变背景）、待办材料包数量（淡橙色渐变背景），左侧圆角图标占位，数字计数动画从0递增（1.5秒缓出曲线），卡片间距24px居中 |
| 管理员看板页 | 课程管理快捷入口 | 创建新课程表单（课程名称、日期时间、时长、名额上限、费用、简介、封面图链接） |

## 3. 核心流程

### 学员报名流程
1. 学员访问首页，浏览课程卡片列表
2. 点击课程卡片上的"报名"按钮，触发圆形进度动画
3. 动画完成后跳转到课程详情页的报名表单
4. 填写姓名、手机号、邮箱，选择技能等级
5. 提交报名，系统自动创建报名记录，更新剩余名额
6. 页面底部学员列表实时刷新显示新报名信息

### 管理员管理流程
1. 管理员访问看板页查看统计数据
2. 创建新课程，填写课程信息和材料包预设
3. 查看各课程报名情况，必要时取消学员报名
4. 一键生成材料需求清单，导出CSV用于采购

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖橙色（#FF8C42）+ 深灰（#333333）
- **背景色**：浅米色（#FFF5E6）
- **按钮风格**：圆角8px，暖橙色主按钮，统一交互反馈
- **字体**：标题使用 Noto Sans SC（粗体），正文使用 Noto Sans SC（常规）
- **布局**：卡片式设计，圆角+柔和阴影
- **图标**：线性圆角图标风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 课程卡片网格 | 三列响应式网格，每张卡片圆角12px、阴影0 2px 12px rgba(0,0,0,0.08)，封面图圆角矩形占位，名称24px粗体深灰，日期14px灰色，剩余名额绿色数字，报名按钮暖橙色圆角8px；悬停上浮3px阴影加深；报名按钮点击后圆形进度动画覆盖 |
| 课程详情页 | 报名表单 | 居中表单卡片，输入框圆角8px浅灰边框，聚焦时1px橙色边框过渡；技能等级三个并列圆形按钮（初级绿色#4CAF50、中级橙色#FF8C42、高级红色#E53935），选中时填充对应颜色 |
| 课程详情页 | 学员列表 | 列表项左侧圆形头像（姓名首字母+随机色）、姓名、等级标签；管理员取消按钮右侧，确认弹窗背景模糊半透明，确认按钮红色渐变；取消时列表项左滑0.3秒消失 |
| 课程详情页 | 材料清单表格 | 表头深色背景#333白色文字，行交替#F9F9F9和#FFFFFF，悬停行背景加深；导出按钮暖橙色 |
| 管理员看板页 | 统计卡片 | 三个并排卡片，宽280px高160px，圆角16px柔和阴影；总课程数淡蓝渐变(#E3F2FD→#BBDEFB)、总报名人次淡绿渐变(#E8F5E9→#C8E6C9)、待办材料包淡橙渐变(#FFF3E0→#FFE0B2)；左侧圆角图标占位，数字计数动画1.5秒缓出；间距24px居中 |

### 4.3 响应式设计

- 桌面端（≥1024px）：课程卡片三列，统计卡片三列并排
- 平板端（768px-1023px）：课程卡片两列，统计卡片两列+一列
- 手机端（<768px）：课程卡片单列，统计卡片单列纵向堆叠
