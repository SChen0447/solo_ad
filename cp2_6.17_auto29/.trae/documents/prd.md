## 1. 产品概述

云端档案室是一款个人数字资产管理与展示应用，以档案馆视觉风格让用户在浏览器中上传、分类、加标签和搜索数字文件（设计稿、电子书、代码片段、思维导图截图等），并自动生成可视化档案统计看板，帮助用户直观了解数字资产全貌和变化趋势。

- 目标用户：个人知识工作者、设计师、开发者、阅读爱好者
- 核心价值：将散乱的数字文件以档案馆式美学统一管理，通过可视化看板洞察资产分布

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册 | 上传、浏览、编辑、搜索、统计查看 |

### 2.2 功能模块

1. **档案馆首页**：统计看板 + 文件卡片网格 + 拖拽上传区
2. **档案详情页**：文件预览 + 标签/备注/星级编辑

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 档案馆首页 | 统计看板 | 横向柱状图展示各类型文件数量分布，总文件数，今日上传数 |
| 档案馆首页 | 文件卡片网格 | 三列自适应网格展示文件卡片，支持拖拽上传 |
| 档案馆首页 | 侧边栏筛选 | 按类型、标签、日期范围、重要程度多条件组合搜索 |
| 档案馆首页 | 加载动画 | 筛选结果加载时显示环绕渐变色圆形加载动画 |
| 档案详情页 | 文件预览 | 图片直接展示，PDF iframe嵌入前3页，TXT等宽字体展示 |
| 档案详情页 | 标签编辑 | 最多5个标签，圆角矩形样式，悬停交互 |
| 档案详情页 | 备注编辑 | 不超过200字文本编辑 |
| 档案详情页 | 星级评分 | 1-5星重要程度，点击时星星放大旋转动画 |

## 3. 核心流程

1. 用户拖拽文件到档案馆首页上传区
2. 系统自动提取文件元信息（文件名、大小、类型、上传时间）
3. 文件以卡片形式出现在网格中，统计看板同步更新
4. 用户点击卡片进入详情页，可预览、添加标签、备注、设置星级
5. 用户通过侧边栏进行多条件筛选搜索
6. 筛选结果以列表展示，点击跳转对应详情页

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白 #f5f0e1，辅色 #e8d9c8，边框 #d4c9b3，文字 #5a4a3a
- **文件类型图标色**：PDF红色、SVG橙色、PNG蓝色、TXT绿色、JPG紫色
- **按钮风格**：圆角矩形，档案馆风格温暖色调
- **字体**：标题使用衬线体营造古典档案馆氛围，正文使用清晰的无衬线体
- **布局风格**：卡片网格 + 固定侧边栏，顶部统计看板
- **图标风格**：线性图标，配合档案馆温暖色调

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 档案馆首页 | 统计看板 | 高200px，渐变背景#f5f0e1到#e8d9c8，圆角12px，横向柱状图+数字摘要 |
| 档案馆首页 | 文件卡片 | 三列网格，卡片宽280-320px，背景#f5f0e1，圆角10px，1px边框#d4c9b3，8px阴影，悬停阴影加深+上浮2px |
| 档案馆首页 | 侧边栏 | 宽280px，固定右侧，半透明背景rgba(245,240,225,0.9)，顶部偏移60px，圆角12px |
| 档案馆首页 | 加载动画 | 环绕渐变色圆形，直径60px，#8b7a6a到#c4a89b循环 |
| 档案馆首页 | 筛选结果列表 | 每条48px高，含图标+文件名+上传时间 |
| 档案详情页 | 文件预览 | 图片直接展示，PDF iframe嵌入，TXT等宽字体 |
| 档案详情页 | 标签 | 圆角矩形，高26px，背景#e8d9c8，文字#5a4a3a，悬停#d4c0a8 |
| 档案详情页 | 星级 | 1-5星，点击时星星放大+旋转动画0.3s |

### 4.3 响应式设计

- **桌面端（≥768px）**：三列网格 + 完整侧边栏固定右侧
- **移动端（<768px）**：两列网格 + 侧边栏折叠为底部浮层，点击按钮从底部滑出（动画0.3s，背景纯白）
- 所有控件支持触控操作

### 4.4 性能要求

- 筛选请求响应 ≤ 200ms
- 上传后卡片出现动画 ≤ 0.5s
- 统计看板数据保存后1s内同步刷新
