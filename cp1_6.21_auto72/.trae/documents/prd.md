## 1. 产品概述

CSS动画工作台是一款面向前端开发者的在线可视化动画编辑工具，旨在解决日常开发中反复手动编写Keyframes、调试缓动函数和同步多动画时间轴的痛点。通过可视化时间轴编辑、实时属性调整与预览、一键代码导出，让CSS动画的创建从"手写代码→浏览器刷新"的循环，变为"拖拽调整→即时预览→导出代码"的高效工作流。

- **目标用户**：前端开发者、UI动效设计师
- **核心价值**：将CSS动画从手动编码转化为可视化操作，提升动画开发效率3-5倍

## 2. 核心功能

### 2.1 功能模块

1. **主编辑页面**：时间轴面板 + 属性面板 + 预览画布

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑页面 | 时间轴面板 | 可视化时间轴，添加/删除/拖拽动画片段，片段以彩色条形图显示，按动画类型自动着色（位移=蓝色系、旋转=绿色系、缩放=橙色系、透明度=紫色系），支持时间轴缩放 |
| 主编辑页面 | 属性面板 | 选中片段后展示属性编辑器（起点/终点数值滑块、缓动函数选择、循环次数、方向），顶部实时CSS代码预览带语法高亮，一键复制，面板展开带弹性动画 |
| 主编辑页面 | 预览画布 | 实时渲染动画效果，中央展示带阴影的目标卡片元素，右下角显示FPS帧率和运行进度条，背景为浅灰网格辅助线 |
| 主编辑页面 | 导出功能 | 导出为完整CssAnimation类代码文件，包含Keyframes和Class定义，格式整洁带注释，支持一键下载为.js文件 |

## 3. 核心流程

用户打开工具 → 在时间轴面板添加动画片段 → 拖拽调整片段位置和时长 → 点击片段在属性面板调整参数 → 画布实时预览效果 → 满意后点击导出 → 下载生成的代码文件

```mermaid

    "选择动画类型" --> "拖拽调整时间轴位置与时长"
    "拖拽调整时间轴位置与时长" --> "在属性面板微调参数"
    "在属性面板微调参数" --> "预览画布实时查看效果"
    "预览画布实时查看效果" --> "效果满意?"

    "效果满意?" --> "是: 导出代码"
    "导出代码" --> "复制CSS代码或下载.js文件"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题 + 霓虹蓝紫渐变（科技感编码氛围）
- **强调色**：霓虹蓝(#00d4ff)、霓虹紫(#a855f7)
- **动画类型色彩**：位移=#3b82f6蓝色系、旋转=#22c55e绿色系、缩放=#f97316橙色系、透明度=#a855f7紫色系
- **按钮风格**：圆角8px，悬停时发光效果
- **字体**：代码区使用等宽字体(JetBrains Mono)，UI使用无衬线字体
- **布局**：三栏布局（左侧时间轴 | 中央画布 | 右侧属性面板）
- **图标风格**：线性图标，2px描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主编辑页面 | 时间轴面板 | 深色背景、彩色条形片段、时间刻度尺、添加按钮、拖拽手柄、缩放滑块 |
| 主编辑页面 | 属性面板 | 深色卡片容器、滑块控件、下拉选择器、CSS代码高亮区、复制按钮、弹性展开动画 |
| 主编辑页面 | 预览画布 | 浅灰网格背景、居中阴影卡片、FPS指示器、进度条 |
| 主编辑页面 | 顶部工具栏 | Logo、导出按钮、添加片段按钮 |

### 4.3 响应式

桌面优先设计，最小支持1280px宽度，三栏布局在小屏幕下可折叠属性面板

## 5. 数据模型

### 动画片段数据结构

```typescript
interface AnimationClip {
  id: string;
  selector: string;
  type: 'translate' | 'rotate' | 'scale' | 'opacity';
  startValue: number;
  endValue: number;
  startTime: number;
  duration: number;
  easing: string;
  iterationCount: number | 'infinite';
  direction: 'normal' | 'reverse' | 'alternate' | 'alternate-reverse';
}
```
