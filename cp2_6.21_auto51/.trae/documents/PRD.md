## 1. 产品概述

手工皮具作品集与材料包推荐应用，为小型文创工作室提供在线作品展示和材料包预订服务，解决现有电商平台无法满足课程作品展示和材料包搭配推荐的痛点。

- 目标用户：手工皮具爱好者、文创课程学员
- 核心价值：精美的作品展示 + 智能材料包推荐 + 便捷预订流程

## 2. 核心功能

### 2.1 功能模块

1. **作品展示页**：响应式网格展示作品卡片、作品详情页、标签筛选
2. **材料包推荐**：基于作品标签的智能推荐引擎、横向滚动推荐卡片、匹配度展示
3. **预订与订单跟踪**：材料包预订表单、订单列表、订单详情模态框、提交成功通知

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页（作品展示） | 作品卡片网格 | 三列响应式布局，卡片含图片、标题、描述、标签胶囊 |
| 首页（作品展示） | 侧边栏 | 标签筛选器、最近浏览记录 |
| 作品详情页 | 详情展示 | 大图、完整描述、作品标签 |
| 作品详情页 | 推荐材料包 | 横向滚动卡片，展示名称、价格、匹配度 |
| 订单列表页 | 订单卡片 | 堆叠式卡片，状态标签、查看详情按钮 |
| 订单列表页 | 订单详情模态框 | 完整订单信息弹窗 |
| 全局 | 顶部通知条 | 订单提交成功提示，5秒后自动淡出 |

## 3. 核心流程

用户浏览作品 → 点击作品卡片查看详情 → 点击"查看推荐材料包"按钮 → 浏览推荐材料包 → 选择材料包并填写预订数量 → 提交订单 → 查看订单列表与状态

```mermaid
    D --> E["填写预订数量"]
    E --> F["提交订单"]
    F --> G["订单成功通知"]

    H --> I["订单详情模态框"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：大地色系
  - 主色：#8B5CF6（紫罗兰）
  - 辅色：#D97706（琥珀色）
  - 背景：#FDF6E3（浅米色）
  - 文字：#1F2937（深灰）
- **按钮风格**：圆角设计，悬停上移2px+阴影加深，点击涟漪动画
- **卡片风格**：圆角12px，加载时浅灰#E5E7EB占位背景，交错弹出动画
- **布局**：双栏结构（左65%内容区 / 右35%侧边栏），移动端自适应单栏

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 作品列表页 | 作品卡片 | 图片55%高度、标题、描述、标签胶囊（动态配色：植鞣革#D97706/铬鞣革#7C3AED/编织#059669） |
| 作品详情页 | 背景 | 浅米色#FDF6E3渐变到白色 |
| 作品详情页 | 推荐卡片 | 宽220px、圆角10px、浅灰#F9FAFB背景、绿色#10B981价格、圆形渐变匹配度徽章 |
| 订单列表页 | 订单卡片 | 宽320px、状态标签（已提交#6B7280/已发货#3B82F6/已完成#10B981） |
| 全局 | 通知条 | 背景#F59E0B、白色文字、0.5s过渡、5秒淡出 |

### 4.3 响应式适配

- **桌面端（≥768px）**：双栏布局，3列作品网格
- **平板（480px-768px）**：单栏布局（内容在上、侧边栏在下），2列作品网格
- **移动端（<480px）**：单栏布局，1列作品网格，卡片占满屏幕宽度

### 4.4 动画与交互

- **卡片入场**：交错逐项弹出（延迟0.1s、从下方20px上移、过渡0.3s cubic-bezier）
- **按钮交互**：悬停上移2px+阴影（0.2s ease）、点击涟漪扩散（0.4s）
- **推荐滚动**：惯性滑动（惯性系数0.6、阻力0.3）
- **通知条**：淡出过渡（0.5s ease）
