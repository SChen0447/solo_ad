## 1. 产品概述

在线独立音乐人作品展示与虚拟演播厅互动应用，为独立音乐人提供曲目发布与试听平台，为乐迷提供实时弹幕互动和虚拟演播厅沉浸式体验，打造集音乐展示、社交互动和虚拟演出于一体的轻量级在线音乐社区。

- 目标用户：独立音乐人（发布曲目、发起虚拟演出）、乐迷（试听曲目、发送弹幕、参与虚拟演播厅）
- 核心价值：填补独立音乐生态中曲目展示+实时互动+虚拟演出一体化的市场空白

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 乐迷（普通用户） | 用户名+邮箱+密码注册 | 浏览曲目、播放试听、发送弹幕、进入虚拟演播厅 |
| 音乐人 | 注册后升级为音乐人 | 上传/编辑/删除曲目、发起虚拟演播厅活动 |

### 2.2 功能模块

1. **首页**：曲目列表展示、搜索过滤、演播厅状态标签
2. **播放页**：曲目播放器、弹幕互动、歌词与热门弹幕
3. **演播厅页**：2D虚拟演播厅Canvas场景、实时聊天、观众互动
4. **认证页**：用户注册、登录
5. **曲目管理页**：音乐人上传/编辑曲目

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索栏 | 按曲名和艺术家实时搜索，防抖300ms |
| 首页 | 曲目卡片网格 | 两列网格展示曲目封面、标题、艺术家、播放按钮、演播厅状态标签 |
| 播放页 | 曲目播放器 | 圆形封面、播放/暂停、进度条拖拽跳转、音频播放 |
| 播放页 | 弹幕互动层 | 弹幕从右向左滚动，颜色随机，字体14-18px随机，上限50条 |
| 播放页 | 弹幕输入框 | 底部输入框，最多30字符，实时发送 |
| 播放页 | 右侧详情面板 | 歌词展示、热门弹幕列表（内容+昵称+时间） |
| 演播厅页 | Canvas场景 | 2D俯瞰演播厅，舞台圆形+观众圆点+话泡动画+感谢动画 |
| 演播厅页 | 实时聊天 | 观众发送聊天消息，圆点闪烁+话泡弹出 |
| 认证页 | 注册/登录表单 | 用户名、邮箱、密码注册；邮箱+密码登录JWT认证 |
| 曲目管理页 | 上传表单 | 曲名(50字符)、艺术家(30字符)、封面URL、音频URL、歌词(2000字符) |
| 曲目管理页 | 编辑/删除 | 音乐人专属操作按钮 |

## 3. 核心流程

**乐迷浏览与互动流程**：首页浏览曲目列表 → 点击卡片进入播放页 → 播放曲目并发送弹幕 → 查看歌词和热门弹幕 → 发现演播厅活动并进入 → 在虚拟演播厅中与其他乐迷互动

**音乐人发布与演出流程**：注册登录 → 升级为音乐人 → 上传曲目详情和音频 → 发起虚拟演播厅活动 → 乐迷实时参与互动

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#0f0f23（深蓝黑主背景）、#1e1e2e（卡片/组件背景）、#6366f1（靛蓝紫主色）、#a78bfa（浅紫罗兰强调色）
- **按钮风格**：圆角24px，渐变背景(#6366f1→#a78bfa)，悬停放大1.05倍+阴影，点击缩小0.95倍
- **字体**：Inter字体系统，标题16px font-medium，正文14px，二级文本14px #9ca3af
- **布局风格**：顶部固定导航栏64px，底部固定导航栏，卡片毛玻璃效果，圆角16px
- **图标风格**：使用lucide-react图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 首页 | 搜索栏 | 居中60%宽，最大500px，圆角24px，背景#1e1e2e，焦点边框#6366f1 |
| 首页 | 曲目卡片 | 280×340px，毛玻璃背景，圆角16px，悬停上移10px+阴影 |
| 首页 | 播放按钮 | 悬停显示，60×60px圆形，背景rgba(99,102,241,0.8) |
| 播放页 | 封面图 | 圆形半径80px，2px #6366f1边框 |
| 播放页 | 进度条 | 高6px，背景#2a2a3e，已播#6366f1→#a78bfa渐变 |
| 播放页 | 弹幕层 | 右→左滚动，12秒时长，字体14-18px随机，5种颜色随机 |
| 播放页 | 弹幕输入框 | 背景#1e1e2e，圆角24px，高44px，边框焦点#6366f1 |
| 演播厅页 | Canvas | 全屏渲染，径向渐变#1a1a2e→#16213e，舞台#2a2a3e圆形 |
| 演播厅页 | 观众圆点 | 直径6px，颜色由用户ID哈希决定，消息时放大12px闪烁 |
| 演播厅页 | 话泡 | 背景#1e1e2e，14px文字，圆角8px，最大宽200px，2秒淡出 |
| 演播厅页 | 感谢动画 | 底部上移+放大至24px，1.5秒后消失 |
| 认证页 | 表单 | 毛玻璃卡片，输入框圆角12px，提交按钮渐变 |

### 4.3 响应式设计

- 桌面优先设计，默认两列网格布局
- 视口宽度<768px：曲目卡片切换为单列，宽度95%视口
- 视口宽度<1024px：播放页改为上下布局
- Canvas尺寸根据容器动态调整，监听resize事件
- 底部固定导航栏在移动端始终可见

### 4.4 3D场景指引

不适用（本项目使用2D Canvas渲染虚拟演播厅场景）
