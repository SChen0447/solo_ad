## 1. 产品概述

在线音乐歌词与乐谱同步编辑器，专为音乐爱好者和乐队成员设计。用户可输入带时间戳的歌词和简谱数字序列，自动生成跟随播放进度高亮显示的滚动歌词页面和联动的简谱旋律卡片，方便练唱和记旋律。

- 核心价值：降低歌曲学习门槛，实现歌词与旋律的同步可视化练习
- 目标用户：音乐爱好者、乐队成员、声乐学习者

## 2. 核心功能

### 2.1 功能模块

1. **主页编辑器**：歌词文本编辑区、简谱数字编辑区、生成预览按钮
2. **播放器控制**：播放/暂停按钮、可拖动进度条、时间显示
3. **歌词滚动展示**：按时间轴逐行高亮、已唱行灰色、未唱行暗色、点击跳转
4. **简谱卡片联动**：垂直滚动卡片展示旋律、与歌词同步高亮、点击跳转

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|---------|
| 主页 | 编辑器区域 | 左栏45%宽度，深色背景#1f2937，包含歌词textarea和简谱textarea，中间1px分割线#374151，生成预览按钮蓝色#3b82f6 |
| 主页 | 预览区域 | 右栏55%宽度，浅色背景#f9fafb，上半部分播放器控制，下半部分歌词与简谱flex-row排列 |

## 3. 核心流程

用户在左侧编辑器输入带时间戳的歌词和简谱数字序列，点击"生成预览"按钮后，系统解析歌词时间轴和简谱数据，在右侧预览区展示播放器、歌词和简谱卡片。播放时歌词逐行高亮，简谱卡片同步联动，用户可点击任意歌词行跳转到对应时间点。

```mermaid

    D --> E["渲染播放器与歌词区域"]
    E --> F["播放/暂停控制"]
    F --> G["歌词逐行高亮 + 简谱联动"]
    G --> H["点击歌词行跳转时间轴"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深灰#1f2937（编辑区）、浅灰#f9fafb（预览区）
- 强调色：蓝色#3b82f6（按钮）、紫色系#6366f1/#8b5cf6（进度条）、浅紫#ede9fe（简谱背景）
- 圆角：统一6-8px
- 阴影：0 2px 8px rgba(0,0,0,0.06)
- 过渡动画：0.2-0.3s ease-out
- 字体：现代无衬线字体，标题粗体，正文适中

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 主页 | 编辑器区域 | 深色背景、双textarea布局、细分割线、蓝色按钮悬停效果 |
| 主页 | 播放器 | 渐变进度条、圆形播放按钮、时间显示 |
| 主页 | 歌词区域 | 行高亮动画、文字缩放效果、背景微亮、垂直滚动 |
| 主页 | 简谱卡片 | 圆形数字按钮、高亮缩放效果、垂直滚动、与歌词同步 |

### 4.3 响应式

- 桌面端（≥900px）：左右两栏布局，左45%右55%，简谱卡片宽140px
- 移动端（<900px）：上下堆叠布局，简谱卡片缩至宽度100px

### 4.4 性能要求

- 歌词超过50句时，高亮切换帧率不低于50fps
- 使用 requestAnimationFrame 优化播放进度更新
- 避免频繁重排，使用 CSS transform 和 opacity 实现动画
