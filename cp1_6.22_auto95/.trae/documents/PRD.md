## 1. 产品概述

在线简历与作品集生成工具，帮助程序员和设计师快速生成个性化的个人主页和简历 PDF。用户只需填写基本信息和工作经历，选择模板风格，即可自动生成带交互作品展示的响应式个人主页，并支持一键导出 PDF 简历。

- 核心目标：解决求职/接私活时反复手动排版简历和作品集的痛点
- 目标用户：程序员、设计师等需要展示个人作品和履历的创意工作者
- 市场价值：提高简历制作效率，提供多样化的专业模板，降低设计门槛

## 2. 核心功能

### 2.1 功能模块
1. **模板选择页**：卡片网格展示三套预设模板，点击进入编辑页
2. **编辑页面**：左侧表单输入，右侧实时预览
3. **实时预览**：根据模板类型渲染不同风格，响应式适配
4. **作品展示**：瀑布流布局，hover 交互，点击弹窗展示详情
5. **PDF 导出**：一键导出简历 PDF，保留样式，显示进度动画

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 模板选择页 | 模板卡片网格 | 展示极简风、科技风、创意风三套模板，卡片 260x180px，圆角 16px，hover 上浮 10px 阴影渐变 0.3s |
| 模板选择页 | 模板缩略图 | 卡片中央展示模板效果小型缩略图 |
| 编辑页面 | 顶部工具栏 | 固定高度 60px，深色背景 #1a202c，包含项目名称、导出按钮、模板切换下拉菜单 |
| 编辑页面 | 左侧表单区 | 宽度 360px，背景 #edf2f7，内边距 24px，包含 6 个字段：姓名、头像链接、个人简介、工作经历（动态增删）、技能标签、作品链接 |
| 编辑页面 | 右侧预览区 | 背景白色，4:3 宽高比，防抖 0.5s 自动更新，背景色跟随模板主题 |
| 实时预览 | 技能标签 | 可点击彩色胶囊，圆角 20px，基于技能类型生成固定哈希色 |
| 作品展示 | 作品卡片 | 宽 180px，圆角 12px，hover 放大 1.05 倍显示半透明遮罩（作品名称 + 查看详情按钮） |
| 作品展示 | 作品弹窗 | 淡入动画 0.2s，全尺寸展示作品图像，带关闭按钮 |
| PDF 导出 | 导出按钮 | 调用后端接口发送 HTML，接收 PDF 触发下载，显示进度条动画 |

## 4. 用户界面设计

### 4.1 设计风格
- **极简风主题**：主色灰白 #f7fafc，辅色蓝灰 #2d3748，简洁干净
- **科技风主题**：主色深蓝 #1a202c，辅色青绿 #38b2ac，专业现代
- **创意风主题**：主色暖粉 #fed7d7，辅色亮橙 #dd6b20，活泼个性
- **按钮样式**：圆角按钮，hover 有平滑过渡动画
- **布局风格**：桌面端左右分栏，移动端上下堆叠
- **字体**：现代无衬线字体，标题加粗，正文常规

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 模板选择页 | 模板卡片网格 | 卡片布局、hover 上浮阴影、圆角边框、主题色预览 |
| 编辑页面 | 顶部工具栏 | 深色导航、下拉菜单、主要操作按钮 |
| 编辑页面 | 左侧表单 | 分组表单、动态添加删除按钮、输入验证提示 |
| 编辑页面 | 右侧预览区 | 4:3 容器、居中布局、主题色背景、响应式内容 |
| 作品展示 | 瀑布流卡片 | 缩略图、hover 遮罩、放大动画、详情按钮 |
| 作品展示 | 弹窗 | 半透明背景、淡入动画、大图展示、关闭按钮 |
| PDF 导出 | 进度条 | 动画进度条、加载状态提示 |

### 4.3 响应式设计
- 桌面端：左右分栏布局，左侧表单 360px 固定宽度
- 平板端：表单区与预览区自适应比例
- 移动端：上下堆叠布局，表单区全宽，预览区可滚动
- 触摸优化：增大按钮点击区域，优化触摸反馈
