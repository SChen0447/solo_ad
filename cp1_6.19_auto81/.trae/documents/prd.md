## 1. 产品概述

社区团购订单聚合与分拣辅助系统，面向社区团长，解决每日手动统计订单和分拣商品耗时易错的问题，提供数字化工具实现订单管理、智能分拣、商品管理和数据统计的全流程覆盖。

- 目标用户：社区团购团长
- 核心价值：减少80%手动操作时间，降低分拣差错率，提升团长运营效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团长 | 系统预设账号 | 订单管理、分拣操作、商品管理、数据查看 |
| 团员 | 通过小程序下单 | 下单、查看订单状态（不在本系统范围） |

### 2.2 功能模块

1. **团长大盘页面**：实时统计卡片、订单完成进度环形图
2. **订单列表页面**：订单搜索筛选、状态管理、详情展开
3. **分拣辅助页面**：商品分类分拣清单、勾选标记、批量完成
4. **商品管理页面**：商品目录CRUD、表单验证、库存预警
5. **数据统计页面**：订单趋势折线图、热销商品柱状图、分类占比饼图
6. **通知提醒功能**：侧边栏角标、铃铛下拉通知

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

### 3.1 团长日常操作流程

团长登录系统 → 查看大盘了解今日概况 → 处理新订单（确认支付状态） → 进入分拣辅助按分类拣货 → 勾选已完成条目 → 标记全部分拣完成 → 查看数据统计优化运营

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：蓝紫渐变（#667eea → #764ba2）
- 辅助色：深蓝灰导航（#2c3e50）、亮蓝选中（#3498db）
- 状态色：待支付#ff6b6b、已支付#feca57、已分拣#48dbfb、已完成#1dd1a1
- 字体：Noto Sans SC，确保中文渲染清晰
- 布局：左右两栏，左240px导航+右自适应内容
- 卡片风格：白色背景，16px圆角，阴影0 4px 12px rgba(0,0,0,0.06)
- 按钮风格：圆角按钮，hover放大1.05倍0.2s过渡，点击缩放0.95回弹

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 团长大盘 | 统计卡片 | 蓝紫渐变背景，白色字体，左上角小图标，悬停上浮3px+阴影加深 |
| 团长大盘 | 环形进度图 | 渐变环形，中心百分比数字翻转动画 |
| 订单列表 | 表格 | 行悬停#f8f9fa，4色状态标签 |
| 订单列表 | 搜索栏 | 搜索框+下拉筛选，300ms防抖 |
| 分拣辅助 | 分类侧栏 | 左侧分类列表，选中高亮 |
| 分拣辅助 | 商品卡片 | 色块占位图+商品名+红色加粗数量+用户列表勾选 |
| 分拣辅助 | 完成按钮 | 底部按钮，确认弹窗+绿色涟漪动画0.5s |
| 商品管理 | 网格卡片 | 每行4个，12px圆角，阴影0 2px 8px rgba(0,0,0,0.06) |
| 商品管理 | 删除弹窗 | 半透明模糊背景，二次确认 |
| 数据统计 | 图表区域 | 白色背景，12px圆角，1px solid #e0e0e0边框 |
| 通知提醒 | 铃铛下拉 | 5条通知，相对时间，悬停#ebf4ff |

### 4.3 响应式设计

- 桌面优先设计，最小宽度1280px
- 移动端（<768px）左侧导航收起为汉堡菜单
- 商品管理网格在窄屏自动调整为每行2个或1个
- 图表区域自适应宽度

### 4.4 动效规范

- 卡片悬停：上浮3px + 阴影加深，0.3s ease
- 按钮交互：hover放大1.05倍0.2s，点击缩放0.95回弹
- 数字翻转：占比变化时百分比数字翻转动画
- 闪烁动画：0.3s透明度1→0.5→1
- 涟漪动画：从中心向外扩散绿色涟漪0.5s
- 页面切换：无闪烁，动画帧率≥50fps
