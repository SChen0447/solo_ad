## 1. 产品概述

虚拟画廊是一个面向艺术家和观众的沉浸式在线画展平台，提供画作浏览、评论互动、收藏管理和热度排行功能，让用户在深色主题的优雅界面中获得沉浸式的艺术欣赏体验。

- 目标用户：艺术爱好者、画廊策展人、普通观众
- 核心价值：为在线画展提供专业的虚拟画廊体验，打破地理限制，让用户随时随地欣赏和互动

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览画作、查看排行、评论互动、收藏画作 |

### 2.2 功能模块

1. **画廊主页**：画作网格展示、热度排行栏、收藏面板
2. **画作详情面板**：画作大图、评论互动、收藏操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画廊主页 | 热度排行栏 | 顶部固定横向柱状图，根据点赞数生成柱子，点击柱子跳转画作详情 |
| 画廊主页 | 画作网格 | 3列网格展示画作卡片，含缩略图、标题、作者、点赞数、评论数 |
| 画廊主页 | 收藏面板 | 侧边展示用户收藏列表，支持取消收藏，本地存储持久化 |
| 画作详情面板 | 详情视图 | 右侧滑入面板(400px)，展示画作大图和信息 |
| 画作详情面板 | 评论区域 | 展示评论列表，支持输入新评论，新评论淡入动画添加 |

## 3. 核心流程

用户打开画廊主页 → 浏览热度排行和画作网格 → 点击画作卡片触发详情面板滑入 → 在详情面板中查看画作信息、评论和收藏 → 点击心形按钮收藏/取消收藏 → 点击排行柱子跳转对应画作详情 → 在评论区域输入并提交评论

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色主题（背景 #1a1a2e，卡片 #16213e，文字 #e0e0e0）
- 强调色：收藏按钮红色 #e94560，排行柱渐变 #e94560 → #0f3460
- 按钮风格：圆角卡片，心形收藏按钮带颜色渐变过渡
- 字体：Playfair Display（标题展示字体）+ Noto Sans SC（正文UI字体）
- 布局风格：3列卡片网格 + 顶部排行栏 + 右侧滑入详情面板
- 图标风格：Lucide React线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 画廊主页 | 热度排行栏 | 固定顶部，渐变色柱状图，底部圆角，弹入动画0.5s依次延迟0.1s |
| 画廊主页 | 画作网格 | 3列CSS Grid，卡片背景#16213e，圆角12px，悬停上浮5px+加深阴影0.3s |
| 画作主页 | 收藏面板 | 侧边浮动面板，收藏列表，心形图标切换 |
| 画作详情面板 | 滑入面板 | 400px宽，右侧0.3s缓动滑入，遮罩层0→0.5透明度过渡 |
| 画作详情面板 | 评论区域 | 圆形头像+用户名+时间+内容，输入框+发送按钮，新评论淡入动画 |

### 4.3 响应式设计

- 桌面优先设计，手机端（≤768px）切换为单列布局
- 卡片宽度自适应屏幕
- 详情面板在移动端全屏覆盖
- 排行栏在移动端自适应宽度
