## 1. 产品概述

二手家居同城交换平台——一个专注于家居用品的轻量级发布与同城交换应用。用户可快速发布闲置家居物品（含图片、描述和价格），通过地图视图浏览附近在售物品，买卖双方通过内置即时消息沟通交易。

- 目标用户：装修/搬家后有闲置家居物品的家庭，以及希望低价购买二手家居用品的小区邻居和同城用户
- 核心价值：解决家居闲置物品的流转需求，提供地理位置感知的同城交易体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 手机号/邮箱注册 | 浏览物品、发布物品、即时消息、个人中心 |

### 2.2 功能模块

1. **首页/浏览页**：地图视图 + 物品列表、搜索与筛选
2. **物品详情页**：物品信息展示、联系卖家、图片轮播
3. **发布页**：图片上传压缩、物品信息填写
4. **即时消息页**：聊天列表、对话气泡、WebSocket实时消息
5. **个人中心**：我的发布、收到的留言、个人信息修改

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 浏览页 | 地图视图 | Leaflet地图，圆形标记按价格区间着色，点击弹出信息窗口，标记聚合，价格梯度连线 |
| 浏览页 | 物品列表 | CSS Grid网格卡片，虚拟列表滚动，拖动地图自动刷新视口范围内物品 |
| 浏览页 | 搜索筛选 | 0.5秒防抖搜索、类别/价格/距离筛选面板 |
| 发布页 | 图片上传 | 最多6张，自动压缩至500KB/800px，预览与删除 |
| 发布页 | 信息填写 | 标题50字、描述500字、原价/售价、城市/小区、联系电话(选填) |
| 详情页 | 物品展示 | 图片轮播、完整信息、联系卖家按钮 |
| 消息页 | 联系人列表 | 头像(首字母圆形)、昵称、最新消息摘要、未读角标 |

| 个人中心 | 个人信息 | 修改头像、昵称等 |

## 3. 核心流程

### 3.1 物品发布流程
用户点击发布按钮 → 上传图片(自动压缩) → 填写物品信息 → 提交 → 状态为待审核 → 24小时自动上架

### 3.2 浏览与联系流程
用户打开浏览页 → 地图/列表查看附近物品 → 点击卡片/标记查看详情 → 点击联系卖家 → 进入聊天窗口 → 实时消息沟通

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：暖白 #faf3e0
- 主色调：森林绿 #2ecc71（按钮、价格标签、活跃导航项）
- 文字主色：深灰 #2d3436
- 价格文字：绿色 #2ecc71，18px加粗
- 时间文字：灰色 #95a5a6，12px
- 按钮风格：圆角实心按钮，hover变暗，点击缩放0.95倍
- 布局风格：顶部固定导航+卡片网格+地图视图
- 图标风格：简洁线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 浏览页 | 导航栏 | 白色背景56px高，2px底部边框#e0e0e0，左侧沙发Logo+名称，右侧发布按钮+头像 |
| 浏览页 | 地图区域 | Leaflet地图，圆形标记(绿/蓝/紫按价格)，弹出窗口带缩放动画 |
| 浏览页 | 物品列表 | CSS Grid，列宽280px-1fr，间距20px，虚拟列表 |
| 浏览页 | 物品卡片 | 缩略图150×150圆角12px，标题、绿色价格、灰色时间、区域标签，hover上移6px+深阴影 |
| 发布页 | 表单 | 图片上传区(6格)、输入框(标题/描述/价格/城市/小区/电话)、提交按钮 |
| 消息页 | 联系人列表 | 左侧：首字母头像40px(随机柔和色)、昵称、最新消息摘要(15字截断)、未读角标 |
| 消息页 | 聊天窗口 | 右侧：气泡(对方左#f0f0f0圆角18px，自己右#6c5ce7白色字圆角18px)、输入框+回车发送 |
| 个人中心 | Tab切换 | 已上架/审核中Tab，物品卡片列表，留言分组+未读角标 |
| 全局 | 页脚 | 高60px，背景#2d3436，白色文字12px，版权信息+联系邮箱 |

### 4.3 响应式设计

- 桌面优先设计，屏幕宽度 < 768px 时：
  - 地图模式隐藏，切换为全屏物品卡片列表
  - 导航栏变为汉堡菜单（三条横线，24px宽×2px高，间距5px，颜色#2d3436）
- 页面切换动画：framer-motion AnimatePresence，淡入淡出0.3秒，translateY(-10px)→0
- 地图标记弹出：缩放0.5→1倍，0.3秒
- 按钮点击：缩放0.95倍，0.1秒，cubic-bezier

### 4.4 性能设计

- 物品列表虚拟列表：仅渲染可视区域卡片，FPS ≥ 55
- 地图标记聚合：缩放级别 < 14时合并密集标记
- 图片占位：加载前显示灰色方块#e0e0e0
